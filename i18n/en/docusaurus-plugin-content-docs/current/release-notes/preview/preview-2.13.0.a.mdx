  For the changed items, see the following.

  **Changing the column name**

  - <Cmdname sid="TTL06959" className="b500" /> → <Cmdname sid="TTL08965" className="uitext" />: Total HTTP call time / Total number of HTTP calls in transactions

  - <Cmdname sid="TTL06956" className="b500" /> → <Cmdname sid="TTL08964" className="uitext" />: Total SQL time / Total number of SQLs in transactions

  - <Cmdname sid="TTL06953" className="b500" /> → <Cmdname sid="TTL08966" className="uitext" />: Total SQL patch time / Total number of SQL patches in transactions

  **Newly added average value columns**

  - <Cmdname sid="TTL06959" className="uitext" />: Total HTTP call time / Number of transactions

  - <Cmdname sid="TTL06956" className="uitext" />: 총 SQL 시간 / 트랜잭션 발생 건수

  - <Cmdname sid="TTL06953" className="uitext" />: Total SQL patch time / Number of transactions

  :::
