February 18, 2025

<Status priority='1'>New</Status> Added monitoring for GraphQL modules (@apollo/server).

- **profile_graphql_enabled** <span class='type'>Boolean</span>

  Default `true`

  Set whether or not to enable the GraphQL monitoring.

- **profile_graphql_variable_enabled** <span class='type'>Boolean</span> 

  Default `false` 

  Set whether to include variables from GraphQL requests in the transaction profile.

- **ignore_graphql_operation** <span class='type'>String</span> 

  It sets monitoring of the specified operation name to be ignored. For multiple values, use comma (`,`) as the delimiter. 

  ```ini title='whatap.conf'
  ignore_graphql_operation=operationNameA,operationNameB
  ```
