August 8, 2024

## MySQL

<Status priority='0'>Fixed</Status> Fixed the bug where data in the `db_mysql_sqlstat` category is accumulated.

:::note

Affected range: DBX agent v1.6.8 - v1.9.3

:::

## Oracle

<Status priority='0'>Fixed</Status> Fixed the bug where data in the `db_oracle_sqlstat` category is accumulated.

:::note

- Affected range: DBX agent v1.6.8 - v1.9.0

- It is recommended to use the latest agent because the Oracle <code className="my">V2</code> operating environment can be affected. The Oracle <code className="my">V1</code> operating environment is not affected.

:::

## Tibero

<Status priority='0'>Fixed</Status> Modified not to collect the Cpu_cores data.
