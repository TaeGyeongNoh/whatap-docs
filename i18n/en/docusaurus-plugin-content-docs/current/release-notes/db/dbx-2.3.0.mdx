December 19, 2024

## PostgreSQL

- <Status priority='1'>Feature</Status> Added the Amazon Aurora metric.

  - `cpu`
  - `active_txns`
  - `log_stream_speed_in_kib_per_second`
  - `pending_read_ios`
  - `read_ios`

- <Status priority='0'>Fixed</Status> Fixed the `vacuum` and `statement` errors that occur in the versions less than PostgreSQL 17.
