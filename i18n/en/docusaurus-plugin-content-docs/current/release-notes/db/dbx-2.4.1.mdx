February 24, 2025

## MySQL

<Status priority='0'>Fixed</Status> Modified the lock collection query in MySQL 8 or later.

- Changed the existing `data_lock_waits` count collection to count only the number of waiting sessions.

- Existing collection queries can be used when the `mysql8_lock_status=false` option is added.

## PostgreSQL

- <Status priority='1'>New</Status> Added the checkpoint metrics collection in PostgreSQL 17 or later.

- <Status priority='0'>Fixed</Status> Fixed the error where the `aws_region` in *whatap.conf* is not recognized by the autoscale agent.
