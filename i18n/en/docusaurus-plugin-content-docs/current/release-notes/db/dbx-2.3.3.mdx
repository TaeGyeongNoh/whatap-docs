January 7, 2025

## SQL Server

- <Status priority='1'>Changed</Status> Added the `mssql_query_hash` field (active session, xview, query stat, and sql stat).

- <Status priority='1'>Changed</Status> Added the `growth` field in `filegroup`.

- <Status priority='0'>Fixed</Status> Fixed the bugs in `query_stat` 2008 or earlier.

- <Status priority='0'>Fixed</Status> Modified the `buffer cache hit ratio` and `cache hit ratio` metrics not to exceed 100%.
