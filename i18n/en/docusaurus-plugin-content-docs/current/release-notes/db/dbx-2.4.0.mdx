## PostgreSQL, MySQL, MS SQL Server, Altibase, Cubrid, Mongo, Redis, IBM DB2

- <Status priority='1'>Changed</Status> Changed not to generate 5-minute statistics for the following categories:

  > `active session`, `xview`, `lockinfo`, `pq_tree`, `segments`, `tablespace`

- <Status priority='1'>Changed</Status> Changed not to generate 5-minute statistics for the following categories and added the `!rectype` and `@id` fields.

  > `active_session`, `xos_disk_usage`
