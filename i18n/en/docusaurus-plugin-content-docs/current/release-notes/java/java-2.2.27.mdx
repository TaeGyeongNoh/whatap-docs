February 6, 2024

- <Status>Feature</Status> Expanded the mule-3.9.5 tracing range.

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=mule-3.9.5
  ```

- <Status>Feature</Status> camel-cxf-3.15 tracing

  ```ini title="whatap.conf"
  # requires agent restart
  weaving=camel-cxf-3.15
  ```

- <Status>Feature</Status> Added the <code>OracleCallableStatement</code> tracing.

- <Status>Deprecated</Status> Redis key collection is stopped when using the Lettuce driver.

- <Status>Fixed</Status> Separated querystring from the request URL when using netty in spring-boot.

- <Status>Fixed</Status> Fixed the issue where the request HTTP parameters cannot be collected when using netty in spring-boot.

- <Status>Fixed</Status> Fixed the display error in the transaction count and error count.

- <Status>Deprecate</Status> Redis value collection is stopped when using the Lettuce driver.
