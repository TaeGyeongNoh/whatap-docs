February 27, 2025

- <Status priority='1'>Feature</Status> Added the traceparent HTTP header issuance feature when using `okhttp3` and `okhttp3-4.4` settings.

- <Status priority='1'>Changed</Status> Changed Jakarta tracing to be possible without setting `weaving=tomcat10`.

- <Status priority='0'>Fixed</Status> Fixed the issue where `{port}` data set in `whatap.name` is not collected when using embedded-tomcat in the Spring Boot environment.

- <Status priority='0'>Fixed</Status> Fixed the issue where linkage tracking is broken in the multi-transaction tab while using the `okhttp` library when tracing with the OpenTelemetry agent.

- <Status priority='0'>Fixed</Status> Fixed the issue where Pod names without ‘-’ are collected as `null`.
