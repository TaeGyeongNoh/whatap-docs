August 27, 2024

- <Status priority='1'>New</Status> Added the module tracing feature to support various versions of **graphql-core**.

  In **graphql-core 3.x+** versions, instrumentation has been added to the `graphql.execution.execute` module, while in **graphql-core 2.x** versions, the `graphql.execution.executor` module has been supported.

- <Status priority='1'>Changed</Status> Modified the *trace_module_definition.py* file to properly instrument modules based on the **graphql-core** version (supporting both the latest and older versions).
