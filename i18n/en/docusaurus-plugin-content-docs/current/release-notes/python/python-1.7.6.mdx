March 4, 2025

- <Status priority='1'>New</Status> Added the tracing of mtid values ​​in the logging module (logging, loguru) monitoring.

  - **trace_logging_mtid_enabled** <span class='type'>Boolean</span>

    Default `false`

    It is enabled if the `trace_logging_enabled` value is `true`, while it is output in the log if the mtid value exists. 

  - **trace_loguru_mtid_enabled** <span class='type'>Boolean</span>

    Default `false`

    It is enabled if the `trace_loguru_enabled` value is `true`, while it is output in the log if the mtid value exists. 

- <Status priority='0'>Fixed</Status> Fixed the issue where it is not reraised when an exception occurs in Celery Task.
