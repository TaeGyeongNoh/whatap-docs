June 19, 2024

- <Status>New</Status> Additionally supported .NET 6, 7, and 8.

  - Supported the ASP.NET core.

  - Supported the HTTP call, Redis, WCF client, user log, Active Stack, and SQL.

  - Download it by using a separate installation link. [whatap_dotnet_core.exe](https://repo.whatap.io/windows/whatap_dotnet_core.exe)

- <Status>Changed</Status> Changed the default value of the `trace_http_client_ip_header_key` option to `X-Forwarded-For`.

:::note

- An integrated installer is being prepared.

- Supporting Cloud and containerized applications is under development.

:::
