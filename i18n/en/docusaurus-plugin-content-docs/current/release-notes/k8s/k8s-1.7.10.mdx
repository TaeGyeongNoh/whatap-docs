August 29, 2024

- <Status priority='1'>Feature</Status> Added the **capacity** node and **allocatable** related metrics to the tag count `kube_node` category.

  - `allocatable_ephemeral_storage`: It represents ephemeral storage available to the node.

  - `capacityEphemeralStorage`: Total capacity of the node's ephemeral storage.

  - `capacityPods`: Maximum number of Pods that can be scheduled on the node.

- <Status priority='0'>Fixed</Status> Fixed the issue where millicore values are missing ​​in the `allocatable_cpu` metric to provide more accurate CPU allocation data.
