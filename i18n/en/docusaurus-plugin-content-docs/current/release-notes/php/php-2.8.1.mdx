July 24, 2024

- <Status>New</Status> Added the agent configuration options to collect transaction statistics by IP address.

  > **whatap.stat_ipurl_enabled** <span class='type'>Boolean</span>, Default `false`

- <Status>Feature</Status> Added the agent configuration option for setting the statistics compression transmission.

  > **whatap.stat_zip_enabled** <span class='type'>Boolean</span>, Default `true`

:::info
