December 11, 2024

- <Status priority='1'>New</Status> PHP 8.4 has been supported.

- <Status priority='0'>Fixed</Status> Fixed the location of the `session_start` profile.

  When Redis query is performed inside the `session_start` function, the `session_start` function is modified to be displayed before the query.

- <Status priority='0'>Fixed</Status> Modified the multi-transaction sampling.

  When multi-transaction sampling is disabled, distributed transaction tracing stops even when there are multiple external HTTP calls.

:::info
