Restart the whatap-telegraf service if it is not running or encounters an error.

:::note

Telegraf plug-ins and agents must have been installed and configured directly on the service.

:::

- **Red Hat/CentOS**

  ```bash title=SH
  $ service whatap-telegraf restart
  ```

- **Debian/Ubuntu**

  ```bash title=SH
  $ sudo service whatap-telegraf restart
  ```

- **FreeBSD**

  ```bash title=SH
  $ service whatap_telegraf restart
  ```
