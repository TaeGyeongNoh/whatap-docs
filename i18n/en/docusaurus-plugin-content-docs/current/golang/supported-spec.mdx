## Go agent support environment

- **Supported version**: 1.17 or later

- **OS**

  - CentOS/Red Hat 6.x (64bit) or later 
  - Debian/Ubuntu 12.04 (64 bit) or later
  - Amazon linux (64bit)
  - Alpine Linux (64 bit)
  - Linux-aarch64 (ARM64)

- **Library**

  - net/http, gin, chi, fiber, gorilla, echo, fasthttp
  - database/sql, gorm, redigo
  - grpc, sarama, k8s.io/client-go

:::note

See the [GitHub site](https://github.com/whatap/go-api-example) on how to apply the framework, libraries, and examples.

:::
