Syslog is a standard protocol for collecting log messages generated by devices. Each device must be configured to send Syslog messages via the NMS server's IP address and specified port. You can enable the port for log collection in the <Cmdname sid="side_syslogSetting" className="uitext" /> menu. 

<ImgLang img="nms-conf-syslog-st.png" desc="Sylog Setting" />

- ***Port***: It enables the specified port on the NMS to collect Syslog messages generated from devices. By default, UDP port 514 is used to collect Syslog messages.

- ***Activate***: ![Toggle icon](/img/ic-toggle-on.svg) Select whether to enable or disable the setting with the toggle button.
