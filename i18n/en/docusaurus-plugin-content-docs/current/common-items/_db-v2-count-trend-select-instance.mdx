### Selecting a comparison target

Select ![number 1](/img/number-01.png) <Cmdname sid="TTL06202" className="uitext" /> and <Cmdname sid="instance" className="uitext" />, and then select ![number 2](/img/number-02.png) <Cmdname sid="TTL06203" className="uitext" /> and <Cmdname sid="instance" className="uitext" />. If you hover your mouse over the chart, you can compare the data through the tooltip that appears.
