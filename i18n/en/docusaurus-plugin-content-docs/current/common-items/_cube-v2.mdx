## What is Cube?

We call the performance statistics generated every 5 minutes as **Cube**. Cube analysis uses each 5-minute performance data stored in cubes.

<ImgLang img='cube-apm.png' desc='Cube' />

## ![](/img/number-01.png) Cube selection panel

Select a cube based on the specific time in the cube selection panel in the left ![number1](/img/number-01.png) area. Because the cubes are stored every 5 minutes, select a specific time. 

- **Panel interest metric**

  - <Cmdname sid="tps" className="uitext" />

## ![](/img/number-03.png) Cube data

In the area of ![number3](/img/number-03.png), the cube data can be checked through the charts for <Cmdname sid="hitmap" className="uitext" />, <Cmdname sid="country" className="uitext" />, <Cmdname sid="tps" className="uitext" />, <Cmdname sid="instance_resp_time" className="uitext" />, <Cmdname sid="cpu" className="uitext" />, <Cmdname sid="realtime_user" className="uitext" />, <Cmdname sid="top_tx" className="uitext" />, and <Cmdname sid="heap_use" className="uitext" />.

- <Cmdname sid="hitmap" className="uitext" />

  It displays the distribution of transactions over the cube time. Detailed trace analysis is possible.

- <Cmdname sid="country" className="uitext" />

  By mapping countries based on the client IP, it displays from which country the transactions are coming. The colors on the map display the relative amount of transactions.

  :::note

  WhaTap basically stores the client-related information. For more information about the agent settings for user data collection, see [the following](agent-number-of-user).

  :::

- <Cmdname sid="tps" className="uitext" />

  Transaction Per Second (**_TPS_**) means the number of transactions processed per second.

- <Cmdname sid="instance_resp_time" className="uitext" />

  Average response time for transactions

- <Cmdname sid="cpu" className="uitext" />

  After sorting the project applications (agents) by the IP, it displays CPUs of the busy server with a bar chart.

- <Cmdname sid="realtime_user" className="uitext" />

  It displays the number of real-time browser users. 

- <Cmdname sid="top_tx" className="uitext" />

  It displays 5 transactions of high invocation. Transaction statistics provide more detailed data.

- <Cmdname sid="heap_use" className="uitext" />

  Among the applications (instances), the instances are displayed in order of heap memory usage.

:::note

- If you select the ![Individual icon](/img/ic-line.svg) icon on the upper right of the chart, merged line charts are provided.
- If you select the ![Individual icon](/img/ic-multiline-chart.svg) icon on the upper right of the chart, individual line charts are provided.
- If you select the ![Bar chart icon](/img/ic-dashboard-bar-chart.svg) icon on the upper right of the chart, the bar-type chart appears. 

:::

WhaTap uses the IP2Location LITE database for [IP geolocation](https://lite.ip2location.com).
