### Setting the columns in the trace list

You can additionally see the details about the search result by setting the columns in the <Cmdname sid="trace_list" className="uitext" /> section. Select ![Column icon](/img/ico-column.svg) <Cmdname sid="column_select" className="uitext" />.

<ImgLang img="analysis-hitmap-tx-list-select-column.png" desc="Select column" />

Select or unselect the desired columns. The settings are immediately applied in the table. Once you've completed all the settings, select ![Close icon](/img/ico-close.svg) on the upper right.

- <Cmdname sid="select_all" className="uitext" />: It selects all columns.

- <Cmdname sid="deselect_all" className="uitext" />: All columns are unselected.

- <Cmdname sid="initialize_default" className="uitext" />: The column is set as the default value.
