## Viewing and visualizing the metrics data

WhaTap provides a list of original data collected according to the conditions specified by user and various charts visualized for convenience as follows. See the menus such as <Cmdname sid="tag_count" className="uitext" /> that can look up the original data of metrics, <Cmdname sid="tag_count_chart" className="uitext" /> that can look up metrics data through visualized charts, and <Cmdname sid="tag_count_anomaly_detection" className="uitext" /> that can detect out-of-pattern anomalies by comparing with the patterns of metrics learned by AI.
