### Adding to the SMS alert reception

If SMS notification is required to be received, set the user phone number in <Cmdname sid="MNU07360" className="uitext" />. 

<ImgLang img='set-event-sms-notification.png' desc='User phone' />

1. Select the profile icon on the upper right of the screen. 

2. If a pop-up menu appears, select <Cmdname sid="MNU07360" className="uitext" />. 

3. In the <Cmdname sid="account_user_phone" className="uitext" /> section, select <Cmdname sid="ITM06575" className="uitext" />.

4. Enter the phone number to receive authentication numbers in <Cmdname sid="phone" className="uitext" />.

5. Select <Cmdname sid="BTN06578" className="uitext" />.

6. Enter a 6-digit authentication number on your mobile phone.

7. Select <Cmdname sid="BTN06501" className="uitext" />.

- Only **Korean mobile phone numbers** can be registered because they can receive SMS notifications.

:::
