- **whatap.okind** <span class='type'>String</span>

  You can configure monitoring units by grouping multiple agents into "type-based" groups. This option allows you to monitor and manage multiple agents in groups, analyze performance metrics, and set the alerts.

  :::note

  You can check it as a group option classified as <Cmdname sid="ITM06006" className="uitext" /> in the <Cmdname sid="side_dashboard" className="uitext" /> and <Cmdname sid="side_analysis" className="uitext" />, <Cmdname sid="side_statistics" className="uitext" /> menus.

  :::

- **whatap.onode** <span class='type'>String</span>

  You can configure monitoring units by grouping multiple agents into "node-based" groups. This option allows you to monitor and manage multiple agents in groups, analyze performance metrics, and set the alerts.

  :::note

  You can check it as a group option classified for nodes in the <Cmdname sid="side_dashboard" className="uitext" /> and <Cmdname sid="side_analysis" className="uitext" />, <Cmdname sid="side_statistics" className="uitext" /> menus.

  :::
