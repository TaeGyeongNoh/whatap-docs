It provides a customization function to allow to modify the alert messages that are delivered to project members via email or third-party plugins. On the upper right of the screen, select <Cmdname sid="goto_alertTemplate" className="uitext" />. You can hide unnecessary items from the received messages or add any desired content.

<ImgLang img='notification-set-message.png' desc='알림 메시지 설정' />

:::note

- This function can be applied only to the <Cmdname sid="metrics" className="b600" /> and <Cmdname sid="composite_metrics" className="b600" /> alerts.

- This function can also be accessed through Select Project &gt; <Cmdname sid="side_lab" className="uitext" /> &gt; <Cmdname sid="side_alertTemplate" className="uitext" />.

- This function can be set by the members with the <Cmdname sid="sms_alert_settings" className="b600" /> role. For more information, see [the following](../project/project-structure#project-auth).

:::

## Basic screen guide

<ImgLang img='notification-set-message-main.png' desc='알림 메시지 설정' />

- The notification messages can be classified into <Cmdname sid="OCCURRED" className="occur set-noti" /> and <Cmdname sid="DSC08323" className="resolve set-noti" />. <Cmdname sid="OCCURRED" className="occur set-noti" /> is a notification that is delivered when an event occurs. <Cmdname sid="DSC08323" className="resolve set-noti" /> is a notification that is delivered when an event has been resolved.

- To hide the details of a received message, select the ![number 1](/img/number-01.png) toggle button to disable it.

- To modify the content of the received message, select ![Number 2](/img/number-02.png) ![Edit icon](/img/ico-edit.svg) <Cmdname sid="edit" className="uitext" />.

- The ![Number 3](/img/number-03.png) area displays the item for details of a received message. The ![Number 4](/img/number-04.png) area displays the details of a message to receive. If you select the ![Number 5](/img/number-05.png) ![Preview icon](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" />, you can preview the content of a received message.

## Hiding the incoming message details

You can hide unnecessary items of the incoming messages. You can hide or display the item in incoming messages by selecting the toggle button at the utmost right of each detailed item. By selecting the ![Preview icon](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" />, you can preview the content of a received message.

- Example of activation of detailed items

  <ImgLang img='notification-set-message-preview.png' desc='활성화 미리보기' />

- Example of deactivating details

  <ImgLang img='notification-set-message-preview-disabled.png' desc='비활성화 미리보기' />

Select ![Save icon](/img/ic-s-check.svg) <Cmdname sid="save" className="uitext" /> to reflect the changes. Select <Cmdname sid="save_all" className="uitext" /> on the upper right of the screen to reflect all changes in different categories at a time.

:::note

- To add hidden item details to incoming messages, activate it by selecting the toggle button of the item.

- Select ![Activation icon](/img/ic-power.svg) <Cmdname sid="active_all" className="uitext" /> to add all detailed items to incoming messages.

:::

## Modifying the details

You can add any special content to the detailed items. 

1. Select ![Edit icon](/img/ico-edit.svg) <Cmdname sid="edit" className="uitext" /> at the utmost right of each detailed item.

2. When the <Cmdname sid="edit_message" className="uitext" /> window appears, enter any content to add.

   <ImgLang img='notification-set-message-edit.png' desc='메시지 편집' />

3. Select <Cmdname sid="confirm" className="uitext" />.

4. By selecting the ![Preview icon](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" />, check the content of a message to receive.

   <ImgLang img='notification-set-message-edit-preview.png' desc='메시지 편집' />

Select ![Save icon](/img/ic-s-check.svg) <Cmdname sid="save" className="uitext" /> to reflect the changes. Select <Cmdname sid="save_all" className="uitext" /> on the upper right of the screen to reflect all changes in different categories at a time.

:::note

In the <Cmdname sid="edit_message" className="uitext" /> window, the text block is a variable that is output depending on the occurred notification. You can reposition the text block by dragging it. If you have deleted the text block, you can add it again by selecting the text block button at the bottom of the window.

<ImgLang img='notification-set-message-edit-block.png' desc='메시지 편집' />

:::

:::caution

Before the content is modified, the message is translated and sent depending on the user's language. After the content is modified, notifications are sent only in the custom format regardless of the language setting.

:::

## Copying the settings to other projects

User settings can be copied to other projects. 

1. On the upper right of the screen, select <Cmdname sid="apply_to_other_project" className="uitext" />.

2. When the <Cmdname sid="apply_project" className="uitext" /> window appears, select a project to copy the settings to. To select all projects, select <Cmdname sid="select_all" className="uitext" />.

3. Select <Cmdname sid="apply" className="uitext" />.

:::info

Only the same product type as the current project can be selected.

:::

## Initialization

Select ![Reset icon](/img/ico-reset.svg) <Cmdname sid="initialized" className="uitext" /> to reset all changes made by users. Select <Cmdname sid="alerttemplate_reset_all" className="uitext" /> on the upper right of the screen to reflect all changes in different categories at a time. If the confirmation message appears, select <Cmdname sid="confirm" className="uitext" />.

:::caution

The initialized configuration cannot be restored.

:::
