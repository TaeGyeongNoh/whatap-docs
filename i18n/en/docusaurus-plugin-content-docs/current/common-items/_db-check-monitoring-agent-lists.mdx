## Checking the monitoring

<Xclude product="oracle-pro">

To check that the log has been normally uploaded from the server and the console has been registered normally, go to the initial screen of [Whatap Monitoring Service](https://service.whatap.io). Select the project to check and then select <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_dbAgents" className="uitext" /> on the left. From <Cmdname sid="side_dbAgents" className="uitext" />, you can check the name of the database server. To set the names of database servers separately, see [the following](agent-dbx-settings#naming).

</Xclude>
