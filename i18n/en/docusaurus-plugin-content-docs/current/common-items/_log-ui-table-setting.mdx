  1. <Cmdname sid="column_settings" className="uitext" />

  - <Cmdname sid="add_a_column" className="uitext" />: You can add a column to the table by selecting the tag.

  - <Cmdname sid="column_order_setting" className="uitext" />: If a column is added, the column is added to the column sequence setting. Drag a desired column to change the column order.

    :::note 

  When the ***log*** column is deselected upon <Cmdname sid="add_a_column" className="uitext" />, <Cmdname sid="ITM08349" className="uitext" /> cannot be seen.

    :::

  2. <Cmdname sid="ITM08349" className="uitext" />

  <ImgLang img='log-table-contents.png' desc='Log Table Content Setting'/>

  - **Whether to display logs**

    - Unchecked items are not displayed in the table. By default, both **_content_** and <Cmdname sid="tag" className="uitext" /> are checked, and both items are displayed. 

    - If you deselect the <Cmdname sid="tag" className="uitext" /> as follows, the log's <Cmdname sid="tag" className="uitext" /> does not appear in the table.
      <ImgLang img='log-table-contents-remove-tag.png' desc='Log table tag invisible'/>

  - **Tag Management**

    - When you add tags in the <Cmdname sid="tag_management" className="uitext" /> list, the tags in the log are listed in the order you added. You can change the order of the tags by dragging them.

    - If you disable an added tag, the disabled tag does not appear. 

:::note
