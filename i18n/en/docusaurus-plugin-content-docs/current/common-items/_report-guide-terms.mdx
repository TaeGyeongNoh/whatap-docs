## Guide to key terms and metrics {#metricdesc}

The following lists the key metrics displayed for each report. 

<div className="java-name">

| Item                                                                                                                                     | Description                                                                                                               |
| ---------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| <Cmdname sid="visitor_dau" className="b600" report/> (**DAU**, Daily Active User) / <Cmdname sid="total_users" className="b600" report/> | It is the number of daily active users or number of users who used the application for a day.                             |
| <Cmdname sid="weekly_user" className="b600" report/>                                                                                     | Total number of visitors who used the application for a week                                                              |
| <Cmdname sid="monthly_active_users" className="b600" report/> (**MAU**, Monthly Active User)                                             | Number of unique users who used the application at least once in a month                                                  |
| <Cmdname sid="_realtime_user" className="b600" report/>                                                                                  | Number of concurrent users with duplicates removed                                                                        |
| <Cmdname sid="users" className="b600" report/> / <Cmdname sid="realtime_visitor" className="b600" report/>                               | Number of users who accessed the application at a specific time zone                                                      |
| <Cmdname sid="tx_count" className="b600" report/>                                                                                        | Number of all requested transactions                                                                                      |
| <Cmdname sid="total_tx" className="b600" report/> / <Cmdname sid="daily_txcount" className="b600" report/>                               | Total number of transactions processed for a day                                                                          |
| <Cmdname sid="weekly_txcount" className="b600" report/>                                                                                  | Number of transactions requested for a week                                                                               |
| <Cmdname sid="daily_hitmap" className="b600" report/>                                                                                    | This distribution chart allows you to see the distribution of transactions and errors by time zone for a day at a glance. |
| <Cmdname sid="error_ratio" className="b600" report/>                                                                                     | Error rate out of the total number of transactions                                                                        |
| <Cmdname sid="event_count" className="b600" report/>                                                                                     | Number of metrics events in <span class="vslow">Critical</span> level                                                     |
| <Cmdname sid="error_cnt" className="b600" report/>                                                                                       | Number of failed transactions due to errors encountered while processing transactions                                     |
| <Cmdname sid="application_count" className="b600" report/>                                                                               | Number of applications (number of agents) monitored during the search period                                              |
| <Cmdname sid="total_core" className="b600" report/>                                                                                      | Number of CPU cores used during the search period                                                                         |
| <b>APDEX\*100**</b> (Application Performance Index)                                                                                      | It indicates the customer performance satisfaction with the application for the query period.                             |
| <Cmdname sid="tps" className="b600" report/> (**TPS**, Transaction Per Second)                                                           | Number of transactions processed per second                                                                               |
| <Cmdname sid="response_avg" className="b600" report/> / <Cmdname sid="response_time" className="b600" report/>                           | Average response time for transactions                                                                                    |
| <Cmdname sid="cpu_core" className="b600" report/>                                                                                        | Number of used CPU cores                                                                                                  |
| <Cmdname sid="visitor_per_hour" className="b600" report/>                                                                                | Number of visitors per hour during the search period                                                                      |
| <Cmdname sid="cpu_trend" className="b600" report/>                                                                                       | CPU utilization of the server                                                                                             |
| <Cmdname sid="mem_sys" className="b600" report/>                                                                                         | Memory utilization of the server                                                                                          |
| <Cmdname sid="active_cnt" className="b600" report/> / <Cmdname sid="active_tx_cnt" className="b600" report/>                             | Number of transactions in progress                                                                                        |
| <Cmdname sid="_cpu" className="b600" report/>                                                                                            | CPU utilization                                                                                                           |
| <Cmdname sid="heap_min" className="b600" report/>                                                                                        | Minimum heap memory usage                                                                                                 |
| <Cmdname sid="operating_rate" className="b600" report/>                                                                                  | This is the percentage of the normal operation time for the application during a given period of time.                    |
