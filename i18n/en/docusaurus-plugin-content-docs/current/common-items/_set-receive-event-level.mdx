## Selecting the reception level

You can select whether to receive notifications based on the alert notification level. In the <Cmdname sid="alert_receive_settings_by_member" className="uitext" /> and <Cmdname sid="3rd_party_plugin" className="uitext" /> sections, for <Cmdname sid="receive_level" className="uitext" /> select <Cmdname sid="all" className="uitext" />, <Cmdname sid="warn" className="uitext" />, or <Cmdname sid="fatal" className="uitext" />.

|   <Cmdname sid="alert_receive_settings_by_member" className="uitext" />  |          <Cmdname sid="3rd_party_plugin" className="uitext" />          |
| :----------------------------------------------------------------------: | :---------------------------------------------------------------------: |
| <ImgLang img="set-event-user-select-level.png" desc="Reception Level" /> | <ImgLang img="set-event-3rd-select-level.png" desc="Reception Level" /> |

- <Cmdname sid="all" className="uitext" />: All alerts can be received.

- <Cmdname sid="warn" className="uitext" />: Only the notifications in warning level can be received.

- <Cmdname sid="fatal" className="uitext" />: Alerts in critical level can be received.
