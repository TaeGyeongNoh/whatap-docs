### HTTP calls {#anlaysis-trace-httpcall-summary}

<ImgLang img="analysis-tx-http-call.png" desc="HTTP Calls" />

To see the count, time sum, average time, and such for HTTP calls, select the <Cmdname sid="TTL06131" className="uitext" /> tab. If unnecessary external calls are repeatedly executed, it can be used as analytical information to determine whether the transaction delays are external or internal by finding the cases where the execution time takes longer compared to the low count.

- ![Statistics icon](/img/ic-static.svg): You can see the <Cmdname sid="transaction_statistics" className="uitext" /> information. If a graph on the chart is selected, the <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" /> menu appears. You can see the list of transactions for the selected time zone.

- ![Download icon](/img/ico-download.svg) <Cmdname sid="http" className="uitext" />: You can download the HTTP call-related details as a TXT file.
