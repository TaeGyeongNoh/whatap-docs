### Guide to transaction information window{#transaction-info}

<ImgLang img='transaction-info-window.png' desc='transaction information' />

- In the ![Number 1](/img/number-01.png) area, the step data can be seen in various formats. Select a desired type of tab.

- In the ![number 2](/img/number-02.png) area, you can see the basic information for transactions.

- ![Number 3](/img/number-03.png) ![Statistics icon](/img/ic-static.svg): The statistical data window for the corresponding URL or each execution section appears. Click a desired time on the graph chart in the statistical data window to go to <Cmdname sid="side_statistics" className="uitext" /> or <Cmdname sid="side_txSearch" className="uitext" />. You can view the statistical data based on the selected time.

  :::note

  - For more information about <Cmdname sid="side_statistics" className="uitext" />, see [the following](analysis-apm-trs).

  - For more information about the <Cmdname sid="side_txSearch" className="uitext" /> menu, see [the following](tx-profile).

  :::

- If you select ![Number 4](/img/number-04.png) SQL step, the SQL variables and HTTP queries can be viewed. For more information, see [the following](#transaction-parameter).
