## Cluster configuration {#cluster-set}

- **cluster_name** <span class='type'>String</span>

  Databases with the same `cluster_name` option as an option to define the cluster set are displayed in a tree structure in <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_instanceList" className="uitext" />. In the agent selection option of <Cmdname sid="side_multiInstance" className="uitext" />, select the <Cmdname sid="ITM07760" className="uitext" /> item to see the instances that make up a cluster on the dashboard at a glance. Some widgets provide cluster-level information. For more information, see [the following](multi-instance-monitoring#select-agent-category).
