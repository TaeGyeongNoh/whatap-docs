You can check the application status for the day. Set the search conditions (<Cmdname sid="select_date2" className="uitext" />, <Cmdname sid="option_type" className="uitext" />, <Cmdname sid="application" className="uitext" />) and then select ![Search icon](/img/ico-btn-search.png). The conditions of the <Cmdname sid="option_type" className="uitext" /> option are as follows:

| Classification                                  | Description                                                           |
| ----------------------------------------------- | --------------------------------------------------------------------- |
| <Cmdname sid="type_oid" className="uitext" />   | Application agent name included in the project                        |
| <Cmdname sid="type_okind" className="uitext" /> | Group unit categorized as the `whatap.okind` option in agent settings |
| <Cmdname sid="type_onode" className="uitext" /> | Group unit categorized as the `whatap.onode` option in agent settings |

The provided charts are as follows: The numbers on the right of the chart display the accumulated value or average, and maximum value for the day.
