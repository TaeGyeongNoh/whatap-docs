### Deleting on the service screen

To delete the agent from [Whatap Monitoring Service](https://service.whatap.io), go to <Cmdname sid="side_dbAgents" className="uitext" />. Before deleting the agent, terminate the agent process on the server where the agent has been installed. 

1. Select a project to delete the agent from the <Cmdname sid="side_projectList" className="uitext" />.

2. On the left of the screen, select <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_dbAgents" className="uitext" />.

3. Select ![Delete icon](/img/ico-trash.svg) on the utmost right of the agent to delete from <Cmdname sid="side_dbAgents" className="uitext" />.

:::note

- Only inactive agents can be deleted.

  <ImgLang img='db-disable-agent.png' desc='' />

- Depending on the agent version, the delete function may not be supported. Update the agent to the latest version.

:::
