### SQLs {#anlaysis-trace-sql-summary}

<ImgLang img="analysis-tx-sql-summary.png" desc="SQLs summary" />

To check the SQL statement information, select the <Cmdname sid="TTL06129" className="uitext" /> tab. In case of unnecessarily repetitive execution, it can be used as analytical data to improve query performance by finding the case where the execution time takes longer compared to the low count.

- ![Statistics icon](/img/ic-static.svg): You can see the <Cmdname sid="transaction_statistics" className="uitext" /> information. If a graph on the chart is selected, the <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" /> menu appears. You can see the list of transactions for the selected time zone.

- ![Download icon](/img/ico-download.svg) <Cmdname sid="sql" className="uitext" />: You can download the SQL-related details as a TXT file.
