It periodically runs the executable user script on the {props.server} server and uploads the results. The user script monitoring feature runs the script at set intervals and uploads the results to the collection server. It collects the results by running the script at 30-second intervals. 

Data is collected based on the script completion time, and there may be some difference between the actual collection time and the data reflection time depending on the content of the running script. Write your own script noting the following example.
