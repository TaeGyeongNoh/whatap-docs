### Downloading JDBC

Copy the *db2jcc4.jar* file from the */`db2_instance_path`/sqllib/java* path of the DB server and then place it in the */`agent_installation_path`/jdbc* folder.

```txt
/unzipped folder/jdbc
```

:::note

For more information about the JDBC driver configuration in IBM DB2, see the [following link](https://www.ibm.com/docs/baw/23.x?topic=installation-configuring-jdbc-drivers).

:::
