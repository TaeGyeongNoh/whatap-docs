2024年12月11日

- <Status priority='1'>New</Status> PHP 8.4バージョンに対応

- <Status priority='0'>Fixed</Status> `session_start`プロファイル情報の位置を修正

  `session_start`関数の内部でRedisクエリーが実行される時、`session_start`関数情報をRedisクエリより先に表示するように修正します。

- <Status priority='0'>Fixed</Status>マルチトランザクションサンプリングを修正

  マルチトランザクションサンプリングが非活性化すると、外部HTTP呼び出しが複数あっても分散トランザクションの追跡を中断します。
