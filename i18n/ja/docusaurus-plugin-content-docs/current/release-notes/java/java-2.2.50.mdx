2025年01月22日

- <Status priority='1'>New</Status> logstash-logback-encoder-7.3追跡

  logbackエンコーダで`net.logstash.logback.encoder.LogstashEncoder`を設定した場合の追跡

  ```ini title="whatap.conf"
  weaving=logback-1.2.8,logstash-logback-encoder-7.3
  ```

- <Status priority='1'>New</Status> logback-ecs-encoder-1.5.0追跡

  logbackエンコーダで`co.elastic.logging.logback.EcsEncoder`を設定した場合の追跡

  ```ini title="whatap.conf"
  weaving=logback-1.2.8,logback-ecs-encoder-1.5.0
  ```

- <Status priority='1'>Feature</Status> logback async appender追跡

  logback appenderでasync appenderを設定した場合の追跡

  ```ini title="whatap.conf"
  weaving=logback-1.2.8
  ```

- <Status priority='1'>Feature</Status> `service_name_mode`オプションにreturn方式の追加

  ```ini title="whatap.conf"
  service_name_mode=return
  ```

  例え、`io.home.Test`クラスのtestメソッドのreturn値をトランザクション名に指定

  ```ini title="whatap.conf"
  hook_service_patterns=io.home.Test.test
  service_name_mode=return
  ```
