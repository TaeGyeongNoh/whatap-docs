2024年09月05日

- <Status priority='1'>New</Status>ランザクションの収集から除外するURL形式(suffix)を設定できるエージェントオプションを追加

  **trace_ignore_url_suffix** <span class='type'>String</span>

  設定した形式と一致するURLは、トランザクションパフォーマンスを収集しません。 複数の値を設定するには、コンマ(,)で区切ります。

- <Status priority='0'>Fixed</Status>Parameterized Queryの追跡機能を追加

  :::note

  この機能に関連するエージェントオプション(`profile_sql_param_enabled`)は、[次の文書](../../dotnet/agent-dbsql)を参照してください。

  :::

- <Status priority='0'>Fixed</Status>REST APIトランザクションのステータスコード(status code) 表示で発生するエラーを修正

- <Status priority='0'>Fixed</Status>`System.Data.Common.DbCommand`の関連メソッドの中でモニタリングに含まれていなかったメソッドを追加

  次の非同期メソッドの中で、オーバーロードされたメソッドの一部が漏れて追加されました。

  - `ExecuteReaderAsync`
  - `ExecuteNonQueryAsync`
  - `ExecuteScalarAsync`
