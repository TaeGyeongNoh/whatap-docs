2024年04月09日

- <Status>Feature</Status>エージェントオプションの追加

  - **prepend_app_pool_name** <span class='type'>Boolean</span>

    既定値`false`

    オプション値を`true`に設定すると、トランザクションのURLの前にアプリケーションのフルネームを表示します。

- <Status>Changed</Status>古いバージョンの互換性を高めるためにインストールファイルを分離

  - .NET 4.6.1バージョン以降のランタイムを使用する場合: [whatap_dotnet.exe](https://repo.whatap.io/windows/whatap_dotnet.exe)

  - .NET 4.6.1バージョン未満のランタイムを使用するか、Windows Server 2012未満の場合: [whatap_dotnet_450.exe](https://repo.whatap.io/windows/whatap_dotnet_450.exe)

    :::note

    Windows Server 2008 R2から対応します。

    :::

- <Status>Fixed</Status> <code>webservice_method_enabled</code>オプション：メソッドが重複して実行される場合にモニタリングされないバグを修正

- <Status>Fixed</Status>32ビットドットネットランタイムで<em>whatap.conf</em>ファイルが見つからない問題を修正
