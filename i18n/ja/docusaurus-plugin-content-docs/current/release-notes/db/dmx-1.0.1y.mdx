2024年12月24日

<Status priority='1'>Changed</Status> prxでdmxエージェントを再実行する周期を調整

- 基本周期**20回**：基本周期は10秒、*prx.conf*ファイルに`interval`オプションで設定できます。

- 以後1分ごとに10回、その以後10分ごとに5回、その以後は1時間ごとに再実行を試みます。

:::note

prxプロセスについての詳細は[次の文書](agent-settings#prxoption)を参照してください。

:::
