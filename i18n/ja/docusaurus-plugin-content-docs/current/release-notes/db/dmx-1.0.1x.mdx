2024年12月20日

- <Status priority='1'>Feature</Status> `tablespace`オプションを追加

  free space照会時にquery timeoutが発生し、オプションによって異なるビューを照会するようにします。

  - `tablespace=1`：**dba_free_space**照会

  - `tablespace=2`：**dba_lmt_free_space**照会

  - `tablespace=3`：**usage_metrics**照会

- <Status priority='1'>Changed</Status> `object_info`にsequence追加

- <Status priority='0'>Fixed</Status>エージェントと同じ経路にファイル名(32文字以上)の長いファイルがある場合、**SIGSEGV**問題修正
