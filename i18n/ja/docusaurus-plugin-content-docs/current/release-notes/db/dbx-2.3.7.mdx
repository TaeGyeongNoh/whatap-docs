2025年01月23日

## Oracle

- <Status priority='1'>Feature</Status> `tablespace`オプションを追加してFree spaceを照会する時、クエリータイムアウト発生問題を解決するためにオプションによって異なるビューを照会

  - `tablespace=1`: `dba_free_space` 照会

  - `tablespace=2`: `dba_lmt_free_space` 照会

  - `tablespace=3`: `usage_metrics` 照会

- <Status priority='1'>Feature</Status> Datafile収集時、`maxsize`情報を追加

## Tibero

- <Status priority='1'>Feature</Status> Datafile収集時、`autoextensible`, `maxsize`情報を追加

- <Status priority='1'>Changed</Status> Tablespace情報収集の時にエージェントにログを残すように修正

## Altibase

- <Status priority='1'>Changed</Status> Datafile収集する時、max_mをmaxsizeに名前を変更

- <Status priority='1'>Changed</Status> Tablespace情報収集の時にエージェントにログを残すように修正
