- <Status priority='1'>New</Status> ロギングモジュール(logging、loguru) モニタリングでmtid値の追跡を追加

  - **trace_logging_mtid_enabled** <span class='type'>Boolean</span>

    既定値`false`

    `trace_logging_enabled`値が`true`の場合は活性化されmtid値が存在する場合はログに出力します。 

  - **trace_loguru_mtid_enabled** <span class='type'>Boolean</span>

    既定値`false`

    `trace_loguru_enabled`値が`true`の場合は活性化されmtid値が存在する場合はログに出力します。
