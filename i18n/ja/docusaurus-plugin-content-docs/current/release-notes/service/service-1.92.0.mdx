## Official Release: 2022-11-02

### ログ

<Status>Change</Status> ログパーサ設定のUIを修正

- ログパーサを追加時にパーサのタイプを選択できるように修正(Grok、 JSONパーサ)
- JSON形式のパーサのUI変更 - [次の文書](../../log/log-setting) を参照

### インテグレーション

<Status>Change</Status> AWS Cloud Watch - Access Key deprecated ガイダンスを修正

- 新規アイテム作成時、Access Key入力欄を削除
- 既存アイテムの修正時、Access Keyの修正ができないように処理し、deprecatedのガイダンスを追加

### Flexボード

- <Status>Fixed</Status> ダッシュボード検索ボタンのUI修正
