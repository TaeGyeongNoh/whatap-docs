2024年05月13日

<Status>Feature</Status> <b>TagCount</b>にデータを追加

- `kube_pod`カテゴリーに`reason`, `message`フィールドを追加

  - `reason`: ポッドstatusのreason
  - `message`: ポッドstatusのmessage

- `container`カテゴリーに`kube_state`(コンテナstate) フィールドを追加

- pendingステータスポッド、コンテナデータ(`kube_pending_pod`, `kube_pending_container`)を収集

  関連データは、<Cmdname sid="side_lab" className="uitext" /> &gt; <Cmdname sid="side_pendingPod" className="uitext" />メニューで確認できます。

- サービス(`kube_service`)に関するデータを収集

- エージェント起動時、<code>#whatapEvent</code>カテゴリでユーザーのインストールRuntime、クラスターのRuntimeを転送
