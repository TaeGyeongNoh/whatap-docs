WhaTapデータベースモニタリングサービス（DBX）は、Queryベースでデータを収集し、性能モニタリングサービスを提供します。 データベースの性能管理は、データベースの性能をリアルタイムでモニタリングし、問題点を診断、分析します。 問題となるSQLをチューニングして適用し、問題が解決されたかを確認します。

## CUBRIDモニタリングの特長

- **リアルタイムDBモニタリング**

  リアルタイムでDB性能指標と実行中のSQLを収集し、問題分析と性能の最適化に必要なデータを共有し、データベースの意思決定を可能にします。 Active Session、Lock Treeなど障害の主な懸念事項のダッシュボードを提供し、通知内容をリアルタイムで検知することができます。 特定のSQLのPlan情報を確認し、SQLチューニングの手がかりを提供します。

- **特定時点の分析**

  指標の短期推移、長期推移、指標比較などの機能を提供し、過去の状況を分析できます。 SQL、Lock Tree、Parallel Query、テーブルスペースの増減などの履歴情報を確認できます。  データベースのパラメータ変更履歴を追跡します。

- **大容量規模でも効率的な構造**

  複数のDBサーバーを管理する場合でも、一元化された性能データ収集および効率的なデータストレージ構造により、最適な分析環境を提供します。

- **クラウドDBモニタリング可能**

  Queryベースのモニタリングデータ収集構造では、DBサーバーの種類に制限はありません。 AWS、GCP、Azureおよび、OCI環境のDBもモニタリングができ、クラウドのリソース指標との統合モニタリングにも対応しています。

- **統合モニタリング**

  Application、Server、Databaseモニタリング製品をすべてインストールすると、連携されたビューを使用して障害の原因を素早く分析し特定できます。

- **WhaTap SaaS**

  多様なDatabase製品を選択でき、サブスクリプションで利用期間を自由に設定できます。 (オンプレミスタイプも提供可能)
