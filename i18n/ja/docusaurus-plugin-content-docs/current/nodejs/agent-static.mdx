エージェントによって収集されたデータから統計を収集する機能に関連するオプションです。

- **stat_domain_enabled** <span class='type'>Boolean</span>

  既定値`false`

  クライアントに接続中のドメインによるトランザクション統計の収集機能を有効にします。

- **stat_mtrace_enabled** <span class='type'>Boolean</span>

  既定値`false`

  MSAアーキテクチャのサービス間の呼び出し統計を有効にします。値が`true`の場合、外部呼び出しを行うときに独自の情報を送信します。

- **stat_ipurl_enabled** <span class='type'>Boolean</span>

  既定値`false`

  IPアドレスごとのトランザクション統計を収集できます。
