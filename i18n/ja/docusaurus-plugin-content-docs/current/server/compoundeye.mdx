ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_compoundEye" className="uitext" />

WhaTapモニタリングサービスの初期画面でプロジェクトを選択した後、<Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_compoundEye" className="uitext" />を選択してください。

<ImgLang img='compoundeye-chart.png' desc='ダッシュボードコンパウンドアイ'/>

コンパウンドアイにより、WhaTapエージェントがインストールされているすべてのサーバーのビューを確認でき、大規模サーバーのリソース状況を一目で確認できます。サンプリングまたは要約された結果でなく、視覚的に可視化されたビューを提供します。

## 提供情報

![コンパウンドアイsc](/img/compound-eye.png)

ひとつの目(Eye)です。全部で5つの情報を提供します。

- <font color='73C2FC'>CPU</font>：CPU使用量

- <font color='C0CA33'>Memory</font>：メモリ使用量

- <font color='7632FF'>Disk</font>：Disk使用量

- <font color='00BEB8'>Rx</font>：ネットワークのRx(受信量)

- <font color='0076AC'>Tx</font>：ネットワークのTx(送信量)

:::note

ネットワーク <font color='00BEB8'>Rx</font>/<font color='0076AC'>Tx</font> 使用量のメトリクスを表示して、DDoSなどの外部攻撃が複数のサーバーで同時に発生しているかを確認できます。

:::

## 異常現象発生

![リソースマップ](/img/compoundeye-chart-ab.png)

サーバーに異常現象が発生した場合、個々のアイ(Eye、目)は、色でステータスを表現します。

- <span class='ov10'>赤</span>：危険ステータス

- <span class='wer'>オレンジ</span>：アラートステータス

- <span class='mstop'>グレー</span>：モニタリングの一時停止

さらに、サーバー上にカーソルを置くと、数値情報を含むポップアップメッセージが表示されます。サーバーをクリックすると、そのサーバーの要約ページに移動し、詳細を把握できます。

## Traffic Max Valueオプション

ネットワーク環境によってトラフィック量が変化する可能性があるため、トラフィックの最大値を調整できます。トラフィックの最大値を設定すると、値に応じてRx/Txグラフが変更します。
