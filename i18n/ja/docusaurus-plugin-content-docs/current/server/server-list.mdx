ホーム画面 > プロジェクト選択 > **_サーバー一覧_** > **_サーバー一覧_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューから **_サーバー一覧_** > **_サーバー一覧_** メニューを選択してください。 **_サーバー一覧_**は、収集サーバーの全体の状態を表示します。サーバーの管理機能およびサーバー間の比較機能を提供します。

<ImgLang img='server-list.png' desc='サーバーリスト'/>

## ![](/img/number-01.png)サーバーステータス

収集サーバーのステータスごとの状態を表示します。 ステータスは1分ごとに更新されます。

-   ![非アクティブアイコン](/img/ico-disable-agent.svg)**非アクティブ**：エージェントは、そのサーバーからデータを収集できません。

-   ![危険アイコン](/img/ic-warning-critical.svg)**危険**：収集サーバーで危険イベントが発生した状態です。

-   ![警告アイコン](/img/ic-warning-filled.svg)**警告**：収集サーバーで警告イベントが発生した状態です。

-   ![正常アイコン](/img/ic-check-circle-filled.svg)**正常**：エージェントが収集サーバーで正常に動作している状態です。

-   ![アイコン](/img/ic-s-pause.svg)**一時停止**：エージェントがデータ収集を一時的に停止した状態です。

## ![](/img/number-02.png)データ更新

サーバー一覧ページのデータは、60秒ごとに自動的に更新されます。

-   ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止できます。 
-   ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再開できます。
-   ![更新アイコン](/img/ic-refresh2.svg)**_更新_**：ユーザーは現在の時間を基準にデータを更新できます。

## ![](/img/number-03.png)サーバーステータス

エージェントが割り当てられたサーバー全体のステータスを示します。一覧からエージェントを選択すると、**_サーバー詳細_**ページに移動します。**_サーバー詳細_**メニューの詳細については、[次の文書](server-detail)を参照してください。

### 検索

![number-s](/img/number-03.png)領域の右上にある **_検索ウィンドウ_**を使用してサーバー名(ONAME)、サーバーID(OID)、タグから入力値を含むサーバー一覧をフィルタリングすることができます。

### カラム

-   ![カラムアイコン](/img/ic-column.svg) **_カラム選択_**

    ![number-s](/img/number-03.png)領域の上部の ![カラムアイコン](/img/ic-column.svg)**_カラム選択_** ボタンをクリックすると、次のようにテーブルに表示するカラムを選択できます。

    <ImgLang img='server-list-cl.png' desc='サーバーリストカラム'/>

    :::note

    選択した**_カラム_**はブラウザのクッキー値として保存されます。また、ページの更新後もステータスが維持されます。 クッキーの削除またはその他の理由で設定エラーが発生した場合、全体を選択された状態に初期化されます。

    :::

-   **カラム幅設定**

    カラムタイトルの端をドラッグして、カラム幅を設定することができます。

-   **カラム配置**

    カラムのタイトルをクリックすると、昇順または降順の並べ替えが可能です。

    :::note

    **カラム幅** および **カラムソート** 設定値は、ブラウザークッキーとして保存され、ページの更新後もステータスが維持されます。クッキーの削除またはその他の理由で設定にエラーが発生した場合は、未設定状態でリセットされます。

    :::

### フィルター

![number-s](/img/number-03.png)領域上部のメニューから ![フィルターアイコン](/img/ico-filter.svg)**_フィルター_** ボタンをクリックすると、タグを使用してサーバーをフィルタリングできます。 

タグは、エージェントをインストール時に自動的に付与されます。 ユーザーが入力値を追加できます。 タグを使用して収集サーバーを識別するか、サーバーのステータスをフィルタリングします。

<ImgLang img='server-list-filter.png' desc='サーバーリストフィルター'/>

-   ![number-s](/img/number-04.png)領域でタグを選択して、サーバーのステータスをフィルタリングできます。 

-   エージェントごとにタグを入力して、新しいタグ機能を有効にできます。![number-s](/img/number-05.png)領域の入力ウィンドウを確認してください。

### モニタリング管理

エージェントコマンドを使用してサーバーを管理する機能です。 ![サーバ管理アイコン](/img/ico-setting.svg)**_管理_** ボタンをクリックし、一覧から管理機能を適用するサーバーを選択してください。 右側に **_一時停止_**, **_再起動_**, **_解除する_** ボタンが有効化されています。

-   ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：エージェントのデータ収集を一時停止します。サーバー一覧で一時停止として表示されます。

-   ![再起動アイコン](/img/ic-caret-right.svg)**_再起動_**：エージェントを再起動します。

-   ![停止アイコン](/img/ic-delete-critical.svg)**_停止する_**：エージェントの動作を停止します。 サーバー一覧から除外され、サーバーのステータスを確認できません。

### エージェント移動

![移転アイコン](/img/ic-swap.svg)**_エージェント移転_** ボタンを選択して、エージェントを他のプロジェクトに移転できます。

<ImgLang img='server-list-agent-transfer.png' desc='サーバーリストエージェント移動'/>

1.  右側の画面で、移動する対象プロジェクトを選択します。

2.  左側の画面で、移動するエージェントを選択してください。

3.  ![right-arrow](/img/right-arrow.svg)**_右矢印_** ボタンをクリックして、対象プロジェクトに移動します。

4.  **_保存_** ボタンをクリックすると移動が完了します。

:::note

-   実行から10分以内に同じエージェントを再度移動する場合、クラッシュが発生する可能性があります。 
-   収集されたデータは移動されません。 履歴データを参照するために、既存のプロジェクトは削除しないでください。

:::

### サーバー間の比較

サーバー間の比較チャートを次のように提供します。 

<ImgLang img='server-list-compare.png' desc='サーバー間の比較 sc'/>

1.  ![チャート比較アイコン](/img/ic-multiline-chart2.svg)**_比較する_** ボタンを選択してください。

2.  ![number-s](/img/number-06.png)をサーバーリストの左側に**_比較_** カラムが追加されたことを確認してください。

3.  **_比較_**カラムで比較するサーバーを選択してください。

4.  比較機能を有効にすると![number-s](/img/ic-number-07.svg)領域で比較チャートを参照することができます。

:::note

-   比較サーバーは最大5つまで選択できます。

-   サーバーごとに一意の色が割り当てられます。もとの色は ![number-s](/img/ic-number-07.svg)**_比較チャート_**でサーバーを区別するために使用されます。

-   **_期間参照_** ボタンをクリックすると、履歴データを参照できます。

:::
