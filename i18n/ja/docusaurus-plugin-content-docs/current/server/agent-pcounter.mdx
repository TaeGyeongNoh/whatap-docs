任意のWindows性能カウンターを収集できます。WhaTapのサーバーモニタリング設定ファイルに性能カウンターセット名を追加します。 

- 性能カウンターセット名を検索できます。
  ```bash title='Windows Powershell'
  Get-Counter -ListSet *
  ```

- 設定ファイルに性能カウンターセットを追加できます。
  ```bash title='Windows Powershell'
  Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "performance.counter.$(Get-Date -uformat %s)=検索する性能カウンターセット名"
  ```

- 追加した性能カウンターは<Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_metricsSearch" className="uitext" /> メニューの <Cmdname sid="category" className="uitext" /> `performance_counter_{性能カウンターセット名}`で表示することができます。

:::note

変更を有効にするには、サーバーを再起動してください。

:::
