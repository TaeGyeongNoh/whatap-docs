WhaTapのモニタリングサービスでモニタリングデータの収集と保存期間を案内します。

## データ収集

| 照会範囲        | 基本照会                  | 選択照会         |
| ----------- | --------------------- | ------------ |
| 3時間以下       | **RAWデータ**(リアルタイムデータ) | 5分データ/1時間データ |
| 3時間超過、15日以下 | 5分データ                 | 1時間データ       |
| 15日超過       | 1時間データ                | -            |

:::note

**RAWデータ**はモニタリングエージェントから伝送されるリアルタイムデータで、データの伝送周期は3秒、5秒、10秒などの間隔で設定されます。

:::

## データ保存期間

WhaTapのすべての製品別のデータ保存期間は次のように提供します。

- RAWデータ(リアルタイムデータ) 保存期間：**1か月**

- 統計データ(5分/1時間) 保存期間：**1年**

- プロファイルデータ保存期間：**1か月**

:::note

- **Network**製品の場合、統計データ(5分/1時間) 保存期間は、**100日**です。

- ログモニタリング機能を有効にした場合、ログデータの保存期間は基本**1日**です。 ユーザーが希望する期間に変更できます。 詳細については、[次の文書](../log/log-setting#log-data-setting)を参照してください。

:::
