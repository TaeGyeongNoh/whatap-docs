PythonエージェントのHTTPC、API Call関連オプションを提供します。

- **profile_httpc_resource_enabled** <span class='type'>Boolean</span>

  既定値`false`

  トレースからHTTP Callステップが収集されるとき、そのステップで使用されるCPUとメモリの使用量を追跡します。

- **trace_httpc_normalize_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  外部HTTP呼び出し(HTTP Call) URLを解析および、正規化する機能を有効にします。 

- **trace_httpc_normalize_urls** <span class='type'>String</span>

  外部HTTPリクエストURLをパーシングして正規化します。呼び出しURLパターンをパーシングして、パスパラメータを削除します。 

  例えば、`/a/{v}/b`と宣言すると`/a/123/b` → `/a/{v}/b`に置き換えられます。複数の登録をする場合は、コンマ(_,_)を使用します。`/a/*/b`として使用して、特定の種類のパスパラメータを削除することができます。
