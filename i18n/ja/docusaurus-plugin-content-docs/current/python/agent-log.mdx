アプリケーションサーバーが実行されると、モニタリング情報を収集します。収集した情報は、ログファイルに保存します。

## ログの設定

- **log_rotation_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  エージェントのログファイルを毎日変更します。ログファイルのデフォルトパスは、_$WHATAP_HOME/logs/whatap-xx.log_です。

- **log_keep_days** <span class='type'>Day</span>

  既定値`7`

  ログファイルの保存期間を設定します。
