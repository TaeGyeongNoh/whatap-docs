### コンテナ要約分析とPod要約分析{#kube-resource-summary-analysis}

<Cmdname sid="TTL08230" className="uitext" />と<Cmdname sid="TTL08317" className="uitext" type="kube_pod"/>はKubernetes環境で各コンテナとPodの性能とリソースに対する分析報告書を提供します。 クラスターとネームスペースのコンテナおよびPodのリソース割り当ておよび使用状況をリアルタイムでモニタリングし、レポート内の提案を参考にして効果的に対応できます。

:::note 

現在、各要約分析機能は韓国語のみで提供しています。 

:::

#### コンテナ要約分析{#container-summary-analysis}

- 基本情報: コンテナの起動時間、再起動回数、属するPodおよびネームスペース情報を提供します。

- コンテナイメージ: 現在使用中のコンテナイメージと同じ画像を使用している他のコンテナの数およびリソース全体の使用量を確認できます。

- ノード情報: コンテナが実行されるノードのOS、IPアドレス、CPUコア数、メモリサイズなどのハードウェア情報を提供します。

- リソース使用状況: コンテナのCPUとメモリリソースの使用量、設定された限度(Limit) およびリクエスト(Request) 値と比較した使用率を提供します。
