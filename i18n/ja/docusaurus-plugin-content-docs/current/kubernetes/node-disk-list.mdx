<Cmdname sid="side_nodeDiskList" className="uitext" />を照会します。 WhaTap Kubernetesモニタリングの<Cmdname sid="side_nodeDiskList" className="uitext" />で各ノードにマウントされた個別ディスクの現在の使用量、余裕空間、性能指標をリアルタイムでモニタリングし、ディスクの状態を簡単に確認できます。 これにより、クラスターのディスク リソースを効率的に管理し、潜在的な問題を事前に把握して対応できます。

## 基本画面ガイド

現在クラスターに存在するノードとそのノードにマウントされたディスクの一覧を次のように確認できます。 照会時間を基準に、最近1分間収集されたディスク情報に基づいて一覧を表示します。

<ImgLang img='k8s-node-disk-list.png' desc='ノードディスクリスト' />

#### ノード名ソート

同じノードに属するディスクのリストが連続してリストされている場合は、そのノード名(***OnodeName***)がマージされて表示されます。 ただ、他のカラムを基準にソートされて連続性が壊れた状態であれば、マージも解除されます。 

#### 未表示ノードディスク

WhaTap Kubernetesエージェントが起動すると、各ノードにマウントされたディスクの一覧が表示されます。 次のような場合、一覧には表示されません。 

- ディスクが特定のパスにマウントされている場合に限って一覧に表示されます。 **マウントポイントが存在しないディスク**は、一覧に表示されません。

- WhaTap KubWhaTap エージェント**起動後に追加されたディスク**は、一覧に表示されません。 

{/* - **마운트 포인트**: 디스크가 특정 경로에 마운트되어 있어야 목록에 표시됩니다. マウントポイントのないディスクは、一覧に含まれません。

- **リアルタイムモニタリング**: マウントされたディスクの使用量、空き容量、I/O性能指標などをリアルタイムでモニタリングできます。 これにより、ディスクの状態を継続的に追跡し、必要に応じて迅速に対応できます。

- **制限事項**: エージェントの起動後に追加されたディスクは、リストに自動的に更新されません。 追加されたディスクをモニタリングするには、エージェントを再起動する必要があります。 */}
