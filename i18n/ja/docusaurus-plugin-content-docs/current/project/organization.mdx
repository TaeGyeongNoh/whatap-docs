組織は、グループをグループ化できる親グループです。ほとんどの場合、プロジェクトとグループで十分です。運用管理サービスを専門とする企業であれば、複数の会社がグループとして存在します。グループやプロジェクトの数が非常に多くなる可能性があります。この場合、グループの概念だけでは不十分です。グループを統合管理できるコンセプトが必要です。組織の概念を使用します。

専門的な運用管理サービス企業は、複数のサブグループを1つの組織にグループ化することで管理できます。各グループメンバーに権限を付与してみてください。権限を付与されたメンバーは、個別にグループを管理できます。

## 組織作成

1. 画面右上のプロフィールアイコンを選択すると、ドロップダウンメニューが表示されます。

2. **_組織生成_**を選択してください。

3. **_組織名_**と**_ロゴ_**、**_説明_**を入力して「次へ」ボタンを選択してください。

4. グループ一覧から組織に追加するグループを選択し、**_完了_**ボタンをクリックします。

**_組織が正常に作成されました。_**'というメッセージウィンドウが表示されましたら、組織の作成は成功しています。

:::note

- **_組織名_**は必須入力です。**_ロゴ_**と**_説明_**を入力することなく組織を作成することができます。
- 修正権限を持つグループのみ、組織に追加できます。
- 修正権限がない場合はそのグループを選択できません。
- グループを選択する必要なく組織を作成できます。
- 権限の詳細については、[次の文書](project-structure#project-auth)を参照してください。
- 修正権限のないグループに招待された場合、そのグループに対する修正権限はありません。
- 親グループから削除したり、他のグループに移動したりすることはできません。
- 組織**_ロゴ_**イメージの推奨サイズは、幅102ピクセル、高さ102ピクセルです。 
- 組織**_ロゴ_**画像は、3MB以下容量の_png_、_jpg_、_jpeg_ファイルの拡張子のみ可能です。

:::

## 組織に属しているグループやプロジェクトを確認する

ホーム画面の左上の**_非組織_**を選択してください。**_組織一覧_**が表示されたら目的の組織を選択してください。**_全体プロジェクト_**メニューから組織に属するグループを確認することができます。**_プロジェクトリスト_**では、組織に属する全体グループのプロジェクトリストを確認することができます。

## 組織管理{#manage}

<ImgLang img='project-organize-manage.png' desc='組織管理' />

1. ホーム画面の左上の非組織を選択してください。

2. **_組織一覧_**が表示されましたら、管理する組織名にカーソルを合わせてください。 

3. ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

**_組織管理_**メニューに移動します。

<ImgLang img='project-organize-manage-group.png' desc='組織管理' />

- ![指示線1](/img/number-01.png) 組織名領域

  ユーザーが所属している組織の一覧を確認するには、![指示線 1](/img/number-01.png)の組織名領域を選択してください。**_組織一覧_**が表示されると別の組織を選択するか、**_非組織一覧に戻る_**を選択してプロジェクトリストに戻ることができます。

- ![指示線2](/img/number-02.png)**_組織一覧_**

  - すべての組織がパネル一覧に表示されます。組織を変更する場合は、![指示線 a](/img/char-a.svg) **_移動する_**を選択します。
  - 組織名とロゴ、説明を編集するには、![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてください。
  - 組織を削除するには、![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。所属するグループとプロジェクトは、組織に所属していないグループに移動します。
  - 修正権限がない場合は、修正・削除ボタンが表示されません。

- ![指示線3](/img/number-03.png)**_グループ一覧_**

  現在選択されている組織に所属するすべてのグループが一覧として表示されます。![右方向アイコン](/img/right-arrow.svg)ボタンをクリックすると、選択したグループの管理画面に移動できます。

- ![指示線4](/img/number-04.png)組織修正

  現在選択している組織の基本情報を修正するには、[編集アイコン](/img/ico-edit.svg)ボタンを選択してください。**_組織修正_**ウィンドウが表示されると、組織名、ロゴ、説明を修正することができます。修正を完了した後、**_保存_**ボタンを選択してください。

## 組織メンバーを招待する

1. 組織メニューから**_メンバー一覧_**を選択します。

2. 画面右上の**_メンバー招待_**ボタンを選択してください。

3. **_組織_**項目でサブグループやプロジェクト単位を選択してメンバーを追加することもできます。

4. **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

5. **_電子メール_**項目で招待するユーザーの電子メールを入力し、**_追加_**ボタンを選択してください。他のユーザーを追加することができます。

6. 招待するユーザーをすべて追加して、**_完了_**ボタンを選択してください。

:::note

招待を完了する前にキャンセルする招待メンバーがいる場合は、**_招待メンバー一覧_**で![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

:::

## 組織メンバーを確認する{#member-list}

<ImgLang img='organization-member.png' desc='組織のメンバーを特定する' />

組織メニューから**_メンバー一覧_**を選択します。組織のツリーでグループまたはプロジェクトを選択できます。選択したグループ、プロジェクトごとに所属するメンバーを確認できます。選択したグループ、プロジェクトごとにメンバーを追加・削除したり権限を修正できます。

### 組織メンバーの権限を修正する

1. 組織メニューから**_メンバー一覧_**メニューを選択します。

2. **_組織メンバー_**一覧で権限を修正するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3. **_メンバー権限の修正_**ウィンドウが表示されると、**_メンバー権限_**と**_組織のグループに対して、ユーザー権限_**を設定してください。

   - **_メンバー権限_**：組織と組織に所属するグループが許可する権限を設定できます。
   - **_組織のグループ別のユーザー権限_**：選択したメンバーの権限を組織に属するグループ別に設定することができます。

4. すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

- 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーの権限を修正できます。選択したグループまたはプロジェクトによって適用できる権限の範囲が異なる場合があります。
- メンバー権限の詳細については、[次の文書](project-structure#project-auth)を参照してください。

:::

## 組織メンバーを削除する

1. **_組織管理_**メニューから**_メンバー一覧_**を選択してください。

2. **_組織メンバー_**一覧から削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。 

3. メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。 

:::note

- サブグループまたはプロジェクトでメンバーを削除する場合は、メッセージウィンドウのチェックボックスをクリックし、**削除**ボタンを選択してください。
- 組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に所属するメンバーを削除できます。

:::
