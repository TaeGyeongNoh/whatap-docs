プロジェクトをグループ化して管理できます。ユーザーは、複数のプロジェクトを所有できます。複数のプロジェクトを管理するためにグループという概念を提供します。2つ以上のプロジェクトがある場合、プロジェクトごとにユーザーを追加したり、権限を付与したりするのは面倒な場合があります。この場合、2つのプロジェクトを1つのグループにまとめ、グループとしてユーザーを管理できます。

:::note

- 1つのプロジェクトは、1つのグループのみに属します。
- プロジェクトは、グループに属していない可能性があります。

:::

プロジェクトユーザーはグループ ユーザーを継承し、グループに属するユーザーはグループ内のすべてのプロジェクトをモニタリングできます。特定プロジェクトに限って権限が付与されたユーザーなら、グループの有無に関係なく、そのプロジェクトのみをモニタリングできます。

## グループを追加する

1. サイドメニューで<Cmdname sid="BTN07332" className="uitext" /> > ![追加アイコン](/img/ico-add.svg) <Cmdname sid="BTN06697" className="uitext" />ボタンを選択します。

   <ImgLang img="group-add.png" desc="グループ追加" />

2. 作成するグループの名前を**_グループ名_**に入力してください。

3. グループに関する説明を**_グループの説明_**に入力してください。**_グループ説明_**は必須ではありません。

4. **_プロジェクトの追加_**一覧でグループに追加するプロジェクトを選択してください。プロジェクトを検索して選択することもできます。

5. プロジェクトの追加が完了したら、**_グループの追加_**ボタンをクリックしてください。

:::note

グループに追加できるプロジェクトの一覧は、修正権限があるプロジェクトまたはグループに属していないプロジェクトです。

:::

## グループにプロジェクトを追加する

1. ホーム画面の**_プロジェクトリスト_**のプロジェクト項目の右上に![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。 

   <ImgLang img='project-add-group.png' desc='プロジェクト設定' />

2. **_グループに追加_**を選択してください。

3. プロジェクトを追加するグループを選択してください。

4. **_グループに追加_**ボタンを選択してください。

## 選択したグループのプロジェクトリストを表示する

ホーム画面で**_全体プロジェクト_**メニューを選択すると、メニューの右側にグループとプロジェクト一覧パネルが表示されます。該当パネルで照会を希望する下位グループを選択してください。選択したグループに含まれるプロジェクトのみが一覧画面に表示されます。 

<ImgLang img='project-select-group.png' desc='プロジェクトグループを選択する' />

グループに属していないプロジェクトを確認するには、一覧パネルを下にスクロールしてください。**_グループに属さないプロジェクト_**一覧を確認することができます。もう一度全体のプロジェクト一覧を確認するには、メニューの下段の![全体プロジェクト一覧に移動するアイコン](/img/ic-home.svg) **_全体プロジェクト一覧に移動する_**アイコンを選択してください。 

## グループからプロジェクトを削除する

1. ホーム画面の**_プロジェクトリスト_**のプロジェクト項目の右上に![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。 

   <ImgLang img='remove-project-from-group.png' desc='グループから削除' />

2. **_グループから削除_**を選択してください。

3. **_グループからプロジェクトを削除_**ウィンドウが表示され、**_確認_**ボタンをクリックしてください。

## グループを削除する

1. ホーム画面で**_全体プロジェクト_**メニューの下位グループの中からいずれか一つを選択してください。

2. **_グループ管理_**を選択してください。

3. **_グループ情報_**セクションの右上の**_グループ削除_**を選択してください。

4. **_グループ削除_**メッセージウィンドウの内容を確認した後、**_削除_**ボタンを選択してください。

:::caution

- グループを削除すると、ユーザー一覧の情報も削除されます。 
- グループに含まれるプロジェクトは、グループ情報のみ削除します。 
- 削除した内容は復元できません。

:::

## グループメンバーを招待する{#add-member}

1. ホーム画面で**_全体プロジェクト_**メニューの下位グループの中からいずれか一つを選択してください。

2. **_グループメンバー_**メニューを選択してください。

3. 画面右上の**_メンバー招待_**ボタンを選択してください。

4. **_グループ_**項目で下位プロジェクト単位を選択してメンバーを追加することができます。

5. **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

6. **_電子メール_**項目で招待するユーザーの電子メールを入力し、**_追加_**ボタンを選択してください。他のユーザーを追加することができます。

7. 招待するユーザーをすべて追加して、**_完了_**ボタンを選択してください。

:::note

招待を完了する前にキャンセルする招待メンバーがいる場合は、**_招待メンバー一覧_**で![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

:::

## プロジェクト別のメンバーを確認する

<ImgLang img='group-memer-by-project.png' desc='プロジェクト別のメンバー確認' />

ホーム画面から**_全体プロジェクト_**メニューの下位グループの中からいずれかを選択してください。**_グループメンバー_**メニューを選択してください。グループの下位にプロジェクトを選択すると、プロジェクトに属するメンバーを確認できます。

## グループメンバーの権限を修正する

1. ホーム画面で**_全体プロジェクト_**メニューの下位グループの中からいずれか一つを選択してください。

2. **_グループメンバー_**メニューを選択してください。

3. **_グループメンバー_**一覧で権限を変更するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

4. **_メンバー権限の変更_**ウィンドウが表示されると、**_メンバー権限_**と**_グループのプロジェクト別のユーザー権限_**を設定します。

   - **_メンバー権限_**：グループとグループに属するプロジェクトで許可する権限を設定できます。
   - **_グループのプロジェクト別ユーザー権限_**：選択したメンバーの権限をグループに属するプロジェクトごとに設定できます。

5. すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

メンバー権限の詳細については、[次の文書](project-structure#project-auth)を確認してください。

:::

## グループメンバーを削除する

1. ホーム画面で**_全体プロジェクト_**メニューの下位グループの中からいずれか一つを選択してください。

2. **_グループメンバー_**メニューを選択してください。

3. **_グループメンバー_**一覧から削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

4. メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note

下位プロジェクトからメンバーを削除する場合は、メッセージウィンドウでチェックボックスをクリックし、**削除**ボタンを選択してください。

:::
