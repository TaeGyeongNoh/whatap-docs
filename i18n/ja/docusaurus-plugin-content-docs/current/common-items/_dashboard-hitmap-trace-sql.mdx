### SQL要約 {#anlaysis-trace-sql-summary}

<ImgLang img="analysis-tx-sql-summary.png" desc="SQL要約" />

SQL文についての情報を確認するには、<Cmdname sid="TTL06129" className="uitext" />タブを選択します。 不要に繰り返し実行される場合、低い件数に比べて実行時間が長くかかる場合を把握し、クエリ性能改善のための分析情報として活用できます。

- ![統計アイコン](/img/ic-static.svg) : <Cmdname sid="transaction_statistics" className="uitext" />情報を確認できます。 チャートのグラフを選択すると<Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" />メニューに移動します。 選択した時間帯のトランザクション一覧を確認できます。

- ![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="sql" className="uitext" />: SQLに関する詳細情報をTXTファイルにダウンロードできます。
