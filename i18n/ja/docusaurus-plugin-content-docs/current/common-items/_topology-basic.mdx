<Cmdname sid="side_topology" className="uitext" />メニューの基本的な使い方について説明します。

<ImgLang img="topology-control.png" desc="アプリケーショントポロジーの基本動作" />

トポロジー画面では、ユーザーの利便性を次のように基本的な機能を提供します。

- 自動更新：規定の30秒間隔で自動更新し、トポロジーの変更を自動的に更新します。 

  :::note

  更新間隔を変更するには、右上の![設定アイコン](/img/ico-setting.svg)を選択し、更新間隔を入力します。5秒から5分まで設定できます。ブラウザを更新して、最新情報にアップデートできます。

  :::

- ズームイン/ズームアウト：マウスホイールを操作してトポロジを拡大または縮小できます。ノード数が多い場合は、必要に応じてトポロジを表現できます。ズームイン/ズームアウトのレベルは、内部設定で保持されます。

- ノードドラッグ：ノードを左クリックしてドラッグし、目的の場所に固定できます。

- プロジェクトノードをクリック

  - <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_appTopology" className="uitext" />メニューでノードをクリックすると、当該プロジェクトのアプリケーショントポロジ(<Cmdname sid="side_instanceTopology" className="uitext" />)に移動します。

  - ホーム画面 &gt; <Cmdname sid="side_integratedTopology" className="uitext" />メニューでノードをクリックすると、そのプロジェクトのトポロジ情報を取得し、現在のトポロジに表示します。

- ハイライトフィルタリング：ノードおよび隣接ノードのリンクをフィルタリングしてハイライトできます。
