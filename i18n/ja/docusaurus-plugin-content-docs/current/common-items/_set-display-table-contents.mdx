<Title hashid='setting-table-contents' level={props.level}>テーブルコンテンツの表示設定</Title>

<InDoc product="browser">

ブラウザーエラー一覧に表示するコンテンツの内容を設定できます。 多くの情報が表示されて見づらい場合や必要な内容を追加したい場合に利用できます。

</InDoc>

1. テーブル右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。

2. ドロップダウンメニューから<Cmdname sid="table_content_setting" className="uitext" />を選択してください。

3. 次のオプションを設定します。

   - <Cmdname sid="whether_display_content" className="uitext" />：テーブル一覧に表示するコンテンツの表示有無を選択してください。

   - <Cmdname sid="tag_management" className="uitext" />：テーブル一覧に表示するタグを選択できます。 見えないように設定したり、順番を変更できます。

4. すべての設定を完了して![閉じるアイコン](/img/ico-close.svg)ボタンを選択します。

設定した内容がテーブルに適用されます。
