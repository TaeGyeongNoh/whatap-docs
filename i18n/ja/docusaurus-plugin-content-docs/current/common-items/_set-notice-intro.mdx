イベント条件を設定し、メール、SMS、メッセンジャー、App Pushなどのさまざまな経路で通知を受信できます。

- エージェントと収集サーバーの通信障害およびアプリケーションのリソース使用率、アクティブなトランザクション、エラー、応答遅延、応答分布異常に関する通知設定ができます。
- アプリケーションの通知で受け入れが難しい条件の通知設定は、タグカウントの通知機能によって行います。
- マシンラーニングベースのAI通知設定により、ヒットマップパターンを検知し、通知機能を提供します。AIモジュールがアプリケーション実行分布パターンを判断して通知を送信します。複雑な条件なしに問題の状況を迅速かつ正確に認識できます。
- <Cmdname sid="side_eventSetting" className="uitext" />メニューにより、しきい値ベースの通知条件および受信設定を行うことができます。
- <Cmdname sid="side_eventNoti" className="uitext" />メニューでは、個人通知の受信チャネルおよびタイムゾーンを設定できます。
- <Cmdname sid="side_eventHistory" className="uitext" />メニューでは、通知発生履歴を確認できます。
