## 照会基準の変更

<Cmdname sid="transaction_map" />のチャート領域、左上のボタンを選択してデータ照会基準を変更することができます。

- <Cmdname sid="elapsed" className="uitext" />：トランザクションが実行された経過時間に基づいてデータを照会します。

- <Cmdname sid="httpcTime" className="uitext" />：HTTP呼び出しに対するリスポンス時間に基づいてデータを照会します。

- <Cmdname sid="sqlTime" className="uitext" />：SQLフェッチ時間に基づいてデータを照会します。

チャート領域の右上の**_Error_**ボタンを選択すると、エラーが発生したトランザクションのみを照会できます。 この状態でチャートのトランザクションをドラッグすると、エラー状態のトランザクションリストが表示された<Cmdname sid="TTL06209" className="uitext" />ウィンドウを開くことができます。 もう一度全体のトランザクションを照会するには、**_Total_**ボタンを選択してください。

:::tip

**Y軸の調整**

- チャートの左上の![上方向アイコン](/img/up-arrow.svg)または![下方向アイコン](/img/down-arrow.svg)ボタンを選択すると、Y軸の最大値を簡単に変更できます。 また、キーボードの上または下方向のボタンを押して、同じ機能を利用できます。

  Y軸の区間によって![上方向アイコン](/img/up-arrow.svg)または![下方向アイコン](/img/down-arrow.svg)ボタン、キーボードボタンを利用する際の動作方法については、以下を参照してください。

  - 10秒から50秒の区間は10秒間隔に変更

  - 50秒から300秒は50秒間隔に変更

  - 300秒以上は100秒間隔に変更

- Y軸の最大値を直接入力して変更することができます。 ![編集アイコン](/img/ico-edit.svg)ボタンを選択し、希望する値を入力して<Cmdname sid="apply" className="uitext" />ボタンを選択してください。

  <ImgLang img="dashboard-txmap-input-y-axi.png" />

:::

:::note

<Cmdname sid="transaction_map" />のチャートに表示されるトランザクションの最大数(<span class="uitext">Limit</span>)は、2,000,000個です。

:::
