### トランザクション情報画面の案内{#transaction-info}

<ImgLang img='transaction-info-window.png' desc='トランザクション情報' />

- ![数字1](/img/number-01.png)領域では、ステップ情報をさまざまな形式で確認できます。 希望する形式のタブを選択します。

- ![数字2](/img/number-02.png)領域では、トランザクションに関する基本情報を確認できます。

- ![数字3](/img/number-03.png)![統計アイコン](/img/ic-static.svg)：該当URLまたは各実行区間の統計データ画面が表示されます。 統計データ画面のグラフチャートで、希望する時間をクリックすると<Cmdname sid="side_statistics" className="uitext" />または<Cmdname sid="side_txSearch" className="uitext" />メニューに移動します。 選択した時間に基づいて統計データを照会することができます。

  :::note

  - <Cmdname sid="side_statistics" className="uitext" />メニューの詳細については、[次の文書](analysis-apm-trs)を参照してください。

  - <Cmdname sid="side_txSearch" className="uitext" />メニューの詳細は[次の文書](tx-profile)を参照してください。

  :::

- ![数字4](/img/number-04.png)SQLステップを選択すると、SQL変数とHTTPクエリを照会することができます。 詳細については、[次の文書](#transaction-parameter)を参照してください。
