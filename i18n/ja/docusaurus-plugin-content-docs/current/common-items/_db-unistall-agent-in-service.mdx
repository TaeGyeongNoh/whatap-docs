### サービス画面から削除する

[WhaTapモニタリングサービス](https://service.whatap.io)からエージェントを削除するには、<Cmdname sid="side_dbAgents" className="uitext" />メニューに移動します。エージェントを削除する前に、エージェントをインストールしたサーバーでエージェントプロセスを終了する必要があります。 

1. <Cmdname sid="side_projectList" className="uitext" />からエージェントを削除するプロジェクトを選択します。

2. 左側のメニューから<Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_dbAgents" className="uitext" />を選択します。

3. <Cmdname sid="side_dbAgents" className="uitext" />で削除するエージェントで右側の![削除アイコン](/img/ico-trash.svg)ボタンをクリックします。

:::note

- 非活性化状態のエージェントのみ削除できます。

  <ImgLang img='db-disable-agent.png' desc='' />

- エージェントのバージョンによっては、削除機能をサポートしていない場合があります。最新バージョンのエージェントに更新してください。

:::
