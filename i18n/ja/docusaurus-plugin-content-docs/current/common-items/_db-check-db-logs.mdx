## ログの確認

<Xclude product="mssql">

- *dbx.log*

  エージェントの起動および収集に関するログはすべて、*`$WHATAP_HOME`/logs/dbx.log*パスで確認できます。*dbx.log*ファイルのサイズが25MBを超える場合は、*dbx.log.[n]*ファイル名でバックアップします。ファイルをクリーンアップする必要がある場合は、サイズの大きい(古い) ファイルから削除してください。

- *xos.log*

  XOSの起動に関するログは、_`$XOSインストールパス`/xos`ymmd`.log_ファイルに記録します。ログは日付で作成します。

</Xclude>
