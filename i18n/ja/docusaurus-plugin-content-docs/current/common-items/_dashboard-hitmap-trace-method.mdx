### メソッド要約 {#anlaysis-trace-method-summary}

<ImgLang img="analysis-tx-method-summary.png" desc="メソッド要約" />

メソッド情報のみを確認するには、<Cmdname sid="TTL06130" className="uitext" />タブを選択しますて。 エージェントに追跡が設定されたメソッド名と所要時間を表示します。 不要なロジックが繰り返し実行される場合、低い件数に比べて実行時間が長くかかる場合を把握し、メソッドロジック改善のための分析情報として活用できます。

- ![統計アイコン](/img/ic-static.svg) : <Cmdname sid="transaction_statistics" className="uitext" />情報を確認できます。 チャートのグラフを選択すると<Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" />メニューに移動します。 選択した時間帯のトランザクション一覧を確認できます。

- ![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="TTL06398" className="uitext" />: メソッドに関する詳細情報をTXTファイルにダウンロードできます。
