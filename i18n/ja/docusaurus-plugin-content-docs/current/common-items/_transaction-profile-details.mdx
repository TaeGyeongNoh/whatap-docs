## 詳細分析 {#details}

ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_hitmap" className="uitext" />

<ImgLang img='tx-hitmap.png' desc='ヒットマップ' />

1. ![数字1](/img/number-01.png)<Cmdname sid="time_selector" className="uitext" />で、ヒットマップを照会する期間を選択してください。

2. ![数字1](/img/number-01.png)<Cmdname sid="option_type" className="uitext" />、<Cmdname sid="application" className="uitext" />から照会対象を選択してください。

3. ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

4. ![数字2](/img/number-02.png)<Cmdname sid="hitmap_transaction" className="uitext" />から目的の領域を![数字3](/img/number-03.png)のようにドラッグしてください。

5. ![数字5](/img/number-05.png)<Cmdname sid="trace_list" className="uitext" />に一覧が表示されたら、分析したいトランザクション項目を選択してください。

トランザクションのパフォーマンス分析のためのクライアント情報などの属性、トランザクションの処理パフォーマンス、各区間別詳細遂行履歴などを確認できる<Cmdname sid="TTL06527" className="uitext" />ウィンドウが表示されます。

:::note

- <Cmdname sid="side_hitmap" className="uitext" />メニューの機能の詳細については、[次の文書](trs-view#details)を参照にしてください。

- アプリケーションダッシュボードのヒットマップウィジェットの詳細については、[次の文書](dashboard-hitmap-trace)を参照にしてください。

:::
