<Cmdname sid="side_flexboard" className="uitext" />はユーザー定義型の統合ダッシュボードです。アプリケーション、サーバー、データベース、コンテナなどのプロジェクトのデータを使用して、リアルタイムのダッシュボードが作成できます。

<ImgLang img='flex-board-mod.png' desc='Flexボードの修正モード' />

- 事前に設定されたテンプレートを提供し、初期設定が簡単にできます。これにより、ユーザーは必要に応じてダッシュボードを構成可能です。

- さまざまなタイプのデータウィジェットが追加できますし、ウィジェット毎の設定を修正して好みのデータ形式で表示できます。

- データフィルタリングにより、ユーザーが希望するモニタリング対象を要約できます。

- 時間範囲を設定して、重要な時間のデータを確認できます。

- サブチャートを活用して、さまざまな方法でデータを確認できます。

- ダッシュボードをお気に入りメニューとして登録できます。

- パーソナライズされたダッシュボードを他のアカウントにコピーして使用できます。

Flexボードは、次のアクセスパスによってさまざまな用途があります。

- ホーム画面 &gt; <Cmdname sid="side_integratedFlexboard" className="uitext" />

  - ウィジェットを作成すると、表示可能なすべてのプロジェクトを選択オプションとして提供します。 

  - ユーザーアカウントにダッシュボードが保存され、他のユーザーにコピーする機能を使用して共有できます。

  - 個人アカウントダッシュボードで、権限による影響はありませんが、読み取り専用として共有されたダッシュボードの場合は修正できません。 

- ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_flexboard" className="uitext" />

  - ウィジェットを作成すると対応するプロジェクトの情報が自動入力されます。

  - プロジェクトメンバーで作成したFlexボードは、自動共有されます。

  - プロジェクト<Cmdname sid="edit" className="b600" />権限、または<Cmdname sid="ITM06494" className="b600" />権限、**Site Admin**権限を持つユーザーが修正できます。

:::tip

Flexボードを修正できるユーザーは、次の主要な機能を利用できます。

- ダッシュボードをJSONファイル形式で書き出しまたは読み込みできます。

- ダッシュボード内のデータリクエストと応答内容を確認できます。

- ウィジェットの設定オプションをJSONファイル形式として、表示または修正することができます。

:::

:::note

- プロジェクト内の<Cmdname sid="side_flexboard" className="uitext" />メニューでは、ダッシュボードの変更権限のあるユーザーのみが<Cmdname sid="edit_mode" className="uitext" />および<Cmdname sid="admin" className="uitext" />モード、<Cmdname sid="filter" className="uitext" />機能にアクセスできます。アクセスできるメンバー権限は、次のとおりです。

  - プロジェクト<Cmdname sid="edit" className="b600" />権限 

  - <Cmdname sid="ITM06494" className="b600" />権限

  - **Site Admin**権限

- <Cmdname sid="view_mode" className="uitext" />および<Cmdname sid="filter" className="uitext" />機能の詳細については、[次の文書](flexboard-mode)を参照してください。

- メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参照してください。

:::
