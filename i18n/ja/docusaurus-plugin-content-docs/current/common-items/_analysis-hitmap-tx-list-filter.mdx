### トレース一覧をフィルタリングする

<Cmdname sid="trace_list" className="uitext" />セクションで照会された結果を次のオプションからフィルタリングできます。

<ImgLang img="analysis-hitmap-tx-list-filter.png" desc="TXトレース" />

- <Cmdname sid="view_all" className="uitext" />: 照会された結果ですべてのトランザクションを確認できます。

- <Cmdname sid="active_stack" className="uitext" />: 照会された結果で、アクティブスタックを含むトランザクションのみを確認できます。

- <Cmdname sid="multi_transaction" className="uitext" />: 照会された結果で、マルチトランザクションを含むトランザクションのみを確認できます。

- ***Search URL***: トランザクションURLと一致する文字列を検索できます。 希望する文字列を入力した後、キーボードの`Enter`キーを押します。
