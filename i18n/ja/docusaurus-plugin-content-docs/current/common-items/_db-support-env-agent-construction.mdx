WhaTapのデータベースモニタリングサービスはエージェントと収集サーバーで構成され、エージェントと収集サーバーの間ではセキュリティのためにデータを暗号化します。外部ライブラリを使用せずに作成され、インストールおよび更新時のシステム再起動は必要ありません。

-   WhaTapデータベースモニタリングサービスは、データベースサーバーにエージェントをインストールすることなく、別のエージェントサーバーでもモニタリングできます。ユーザーの環境に合わせて柔軟に適用できます。

    |                      別途のエージェントサーバーにインストール可能                     |                        DBサーバーにインストール可能                       |           DBリソースモニタリング(XOS) 可能          |

    </div>

-   XOSエージェントは、データベースサーバーのプロセス使用状況をモニタリングできる付加オプションエージェントです。データベースサーバーでプロセス使用状況をモニタリングする場合は、データベースサーバーに別のエージェントを起動してデータを収集できます。

-   エージェントとサーバーは、バイナリベースの通信プロトコルを使用します。また、データを選択的に暗号化することで、データのセキュリティを確保しながら、暗号化の負荷を低減しました。

:::note

-   モニタリング対象データベースサーバーとのネットワーク通信が可能なサーバーにWhaTapエージェントをインストールする必要があります。

-   WhaTapのDBXエージェントサーバには**Java 8バージョン**以降の**JDK**をインストールする必要があります。

:::
