### HTTPコール要約 {#anlaysis-trace-httpcall-summary}

<ImgLang img="analysis-tx-http-call.png" desc="HTTPコール要約" />

HTTP呼び出しのリクエスト件数、合計時間、平均時間などを確認するには、<Cmdname sid="TTL06131" className="uitext" />タブを選択します。 不要な外部への呼び出しが繰り返して実行される場合や、低い件数だが比較的に実行時間が長くかかる場合などを把握するなど、トランザクションの遅延要因が外部か内部かを分析する情報として活用できます。

- ![統計アイコン](/img/ic-static.svg) : <Cmdname sid="transaction_statistics" className="uitext" />情報を確認できます。 チャートのグラフを選択すると<Cmdname sid="side_analysis" className="uitext" /> &gt; <Cmdname sid="side_txSearch" className="uitext" />メニューに移動します。 選択した時間帯のトランザクション一覧を確認できます。

- ![ダウンロードアイコン](/img/ico-download.svg) <Cmdname sid="http" className="uitext" />: HTTP Callに関する詳細情報をTXTファイルにダウンロードできます。
