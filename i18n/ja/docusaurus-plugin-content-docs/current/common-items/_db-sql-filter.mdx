## 参照結果フィルタリング

### フィルター条件を追加する

照会された結果からSQLテキスト値またはエージェント名、DBユーザー名などを基準にフィルタリングし、希望する結果を照会できます。

<ImgLang img='db-sql-filter-adding.png' desc='条件を追加する' />

1. <Cmdname sid="filter" className="uitext" />オプションで<LinkImage img="ico-add.svg" className="ico" />ボタンを選択します。

2. <Cmdname sid="TTL06740" className="uitext" />項目で目的のフィルタリング基準を選択します。

   - 選択した項目の値が文字に該当する場合は<Cmdname sid="includes" className="uitext" />(青)、<Cmdname sid="excludes" className="uitext" />(赤) の条件を選択できます。

     ![条件色](/img/db-sql-filter-condition-color.png)

   - 選択した項目の値が数字に該当する場合は`==`(等しい), `>=`(より大きいか等しい), `<=`(より小さいか等しい) 条件を選択できます。

3. <Cmdname sid="condition" className="uitext" />項目で条件を選択します。

4. 条件に一致する文字列または数字を入力してください。

5. <Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::note

- フィルタリング条件を追加するには、<Cmdname sid="bsm_button_add" className="uitext" />ボタンを選択し、1 ~ 5の手順を繰り返します。 追加した条件は、AND(`&&`) 条件として適用されます。

- 条件を追加する時に一部の項目を削除するには、フィルター条件の右側の![削除アイコン](/img/ico-trash.svg)ボタンを選択します。 全体の条件を削除するには、![削除アイコン](/img/ico-trash.svg) <Cmdname sid="BTN06616" className="uitext" />ボタンを選択します。

- <Cmdname sid="filter" className="uitext" />オプションに適用された条件をすばやく削除するには、<LinkImage img="ico-remove.svg" className="ico" />ボタンを選択します。

- `sqlText`に「WhaTap」が含まれている場合は、除外するようにデフォルトで設定されています。

  <ImgLang img='db-sql-filter-default.png' desc='WhaTap' />

:::

### フィルター条件を修正する

<ImgLang img='db-sql-filter-modify.png' desc='フィルターの修正' />

<Cmdname sid="filter" className="uitext" />オプションに適用されている項目をクリックします。 <Cmdname sid="TTL06737" className="uitext" />画面が表示されたら、必要な項目を修正し<Cmdname sid="apply" className="uitext" />ボタンを選択します。
