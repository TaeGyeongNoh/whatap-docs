## SQL統計

ホーム画面 &gt; プロジェクト選択 &gt; <Cmdname sid="reportStat" className="uitext" /> > <Cmdname sid="statistics_sql" className="uitext" />

データベースごとの統計データを提供します。SQL別の時系列チャートで発生時間とパフォーマンスを確認することができます。 

<Cmdname sid="statistics_sql" className="uitext" />メニューの詳細については、次の動画を参照してください。

<Youtube ko="lt_R3SeTbOc" en="lt_R3SeTbOc" ja="lt_R3SeTbOc" />

:::note

5秒ごとに収集されるセッション情報とその時点で実行されているSQL Textに基づいて提供される統計ページです。このページは、収集方法によって誤差が生じる可能性があります。

:::
