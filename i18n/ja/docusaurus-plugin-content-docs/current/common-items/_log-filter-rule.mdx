フィルター入力時に使用できる文法の一覧です。**_ライブテール_**と**_ログ検索_**は同じ文法を使います。

| 文法            | 例                             | 説明                                      |
| ------------- | ----------------------------- | --------------------------------------- |
| include(+)    | `+ote-gateway`                | `ote-gateway`を含むものだけをフィルタリング            |
| ^             | `@txid:+-6611219762256426545` | `@txid`が`-6611219762256426545`のみフィルタリング |
| exclude(-)    | `-ote-gateway`                | `ote-gateway`を除いたもののみフィルタリング            |
| startWith(\*) | `ERROR*`                      | `ERROR`で始まるものだけをフィルタリング                 |
| endWith(\*)   | `*ERROR`                      | `ERROR`で終わるものだけをフィルタリング                 |

:::note

include(+) 文法を使用する場合、'`-`1234'のように、負の数を含む文字列を検索する場合は、'`+`'記号を追加して'`+-`1234'を検索してください。

:::
