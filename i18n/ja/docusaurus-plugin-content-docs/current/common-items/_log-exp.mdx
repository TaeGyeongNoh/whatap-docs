**_ログトレンド_**メニューでタイプごとに分類されたログの発生件数とその推移から特定のエラータイプの発生パターンを確認したり、時間ごとの詳細なログデータを確認できます。ハイライト機能で、必要なログをすばやく識別できます。カテゴリ別に収集されたログの推移を照会できます。<Loguimainterm />

<ImgLang img='log-trend.png' desc='ログトレンド'/>

## データ参照

- 下部までスクロールすると、次のデータを表示します。

- ![number-s](/img/number-01.png)で検索するログデータの**_時間_**と収集単位として**_カテゴリ_**を指定します。

- **_カテゴリ_**を変更すると、選択されたカテゴリ対応するログを参照します。![number-s](/img/number-02.png)バーチャトと![number-s](/img/number-03.png)ログテーブルで確認できます。

- ![number-s](/img/number-02.png)棒グラフのバーをクリックすると、バーの時間範囲に対応するログを検索します。 

- ![number-s](/img/number-03.png)ログテーブルの左上から照会されたログの総数を確認できます。 

- ![number-s](/img/number-03.png)ログテーブル上段の右の <LinkImage img='ic-fullscreen.svg' desc='全画面表示アイコン'/>**_全画面_**アイコンを選択すると、**_ログ_**と**_タイムスタンプ_**を全画面で確認することができます。

- ![number-s](/img/number-04.png) サイドメニューからタグでフィルタリングすることでログを確認できます。 検索キーは最大2つまで選択でき、検索値は複数選択できます。

## ログをチャートで検索

<ImgLang img='log-trend-bar.png' desc='ログトレンド棒チャート'/>

- 棒チャートで![aアイコン](/img/char-a.svg)目的の時間をクリックすると、その時間のログを確認できます。 

- 棒チャートの下のログテーブル上部左側の![bアイコン](/img/char-b.svg)時間選択オプションを使用して、次のように選択した時間帯をさらに細分化してログを検索することができます。

  - 1min：interval（チャート内の棒の間隔）

  - 時間選択オプション：選択された時間範囲を6つの間隔に分割した時間帯
