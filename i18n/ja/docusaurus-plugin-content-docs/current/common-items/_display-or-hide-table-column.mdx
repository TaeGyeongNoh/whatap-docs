<Title hashid='display-or-hide-column' level={props.level}>テーブルカラムの表示または非表示</Title>

<InDoc product="browser">

ブラウザーエラー一覧テーブルのカラム ヘッダー項目を表示または非表示できます。

</InDoc>

1. テーブル右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。

2. ドロップダウンメニューから<Cmdname sid="column_settings" className="uitext" />を選択します。

3. <Cmdname sid="column_settings" className="uitext" />画面が表示されたら <Cmdname sid="add_a_column" className="uitext" />セクションでお探しの項目を選択するか、選択を解除します。

4. すべての設定を完了して![閉じるアイコン](/img/ico-close.svg)ボタンを選択します。

設定した内容がテーブルに適用されます。
