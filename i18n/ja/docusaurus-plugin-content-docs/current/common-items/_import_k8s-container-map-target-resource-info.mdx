- ***Node***: [ノード詳細](node-details)、[オブジェクトマニフェスト](object-manifest) 情報を確認し、[ノードリスト](resource-node-list) メニューへ移動できます。

- ***Deployment***: [オブジェクトマニフェスト](object-manifest) 情報を確認し、[Deployment一覧](deployment-list) メニューへ移動できます。

- ***ReplicaSet***: [オブジェクトマニフェスト](object-manifest) 情報を確認できます。 

- ***Pod***: [オブジェクトマニフェスト](object-manifest)、[Pod要約分析](container-map-target-info#pod-summary-analysis) 情報を確認し、[Pod一覧](resource-pod-list) メニューへ移動できます。 

- ***ContainerId***: ***コンテナ情報***、[コンテナ要約分析](container-map-target-info#container-summary-analysis) 情報を確認し、[コンテナ一覧](resource-container-list) メニューへ移動できます。

- ***Image***: ***コンテナイメージ***情報を確認し、[コンテナイメージ](container-images) メニューへ移動できます。 

- ***ServiceName***: [オブジェクトマニフェスト](object-manifest) 情報を確認し、[Service一覧](service-list) メニューへ移動できます。
