## Flexボードの編集

**_Flexボード_**メニューの**_ダッシュボード一覧_**で![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。画面の右側にFlexボード管理ウィンドウが表示されます。

:::note

**Flexボード編集機能権限のガイド**

-   ホーム画面 > **_Flexボード_**

    個人アカウントダッシュボードで、権限による影響はありませんが、読み取り専用として共有されたダッシュボードの場合は修正できません。 

-   ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_Flexボード_**

    プロジェクト**修整**権限、または**プロジェクトFlexボード編集**権限、**Site Admin**権限を持つユーザーが修正できます。

-   メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参照してください。

:::

<ImgLang img='flexboard-manage.png' desc='Flex ボード管理' />

設定できる属性は次のとおりです。

-   **_名前の変更_**：ダッシュボード名を修正できます。

-   **_プロジェクト_**：ダッシュボードのウィジェットからデータを読み込むプロジェクトを選択できます。

    :::note

    このオプションは、ホーム画面 > **_Flexボード_**メニューのみ設定できます。

    :::

-   **_レイアウト_**

    -   ウィジェットを選択し、ドラッグして場所を移動できます。

    -   ウィジェットの右下をマウスで選択し、ドラッグしてウィジェットのサイズを調整できます。

    -   ウィジェットを削除するにはウィジェット右上の![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。

    -   **_オーバーラップ_**トグルボタンをアクティブにすると、ウィジェットの配置を重なった状態で配置できます。

    :::note

    ホーム画面 > **_Flexボード_**メニューに入った場合、各ウィジェットごとにデータを読み込みするプロジェクトを選択できます。ウィジェット右上の![追加アイコン](/img/ico-add.svg)ボタンをクリックし、目的のプロジェクトを選択してください。プロジェクトを検索して選択でき、複数選択することもできます。選択を完了した後、**_適用_**ボタンを選択してください。

    :::

-   **_ウィジェット.json_**：ダッシュボードからウィジェット設定をJSON形式で読み込むことができます。![コピーアイコン](/img/ico-copy.svg)ボタンをクリックすると、JSONの内容をコピーします。

-   **_オプション.json_**：ダッシュボードに設定されたオプションをJSON形式で読み込むことができます。 ![コピーアイコン](/img/ico-copy.svg)ボタンをクリックすると、JSONの内容をコピーします。

## Flexボードの削除

**_Flexボード_** > **_ダッシュボード一覧_**から削除する項目の右側にある![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。確認メッセージが表示されると、**_削除_**ボタンを選択してください。

:::note

-   ダッシュボード所有者がダッシュボードを削除する場合、ダッシュボードを共有しているすべてのユーザーアカウントもダッシュボードを削除します。
-   ダッシュボードを共有するユーザーがダッシュボードを削除した場合、そのユーザーの**_ダッシュボード一覧_**から削除されます。元のダッシュボードは影響を受けません。

:::
