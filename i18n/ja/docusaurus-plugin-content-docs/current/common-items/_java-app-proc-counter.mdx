## app_proc_counter {#appproccounter}

アプリケーションのプロセス指標を収集したデータのカテゴリです。

- 収集間隔：5秒
- 統計データ：5分、1時間

### Tags {#appproccounter-tags}

<div className="c3-30-60-10">

| タグ名            | 説明                         | 備考    |
| -------------- | -------------------------- | ----- |
| `alias`        | アプリケーションの別称                | ユニーク値 |
| `container`    | コンテナ名                      | ユニーク値 |
| `containerKey` | コンテナキーの値                   | ユニーク値 |
| `host_ip`      | ホストのIPアドレス                 | ユニーク値 |
| `okindName`    | `whatap.okind`に分類されたグループ名  | -     |
| `oname`        | エージェント名                    | ユニーク値 |
| `onodeName`    | `whatap.onode`に分類されたグループ名  | -     |
| `pid`          | 特定プロセスの固有識別子(ID)           | -     |
| `type`         | プラットフォームタイプ(アプリケーションの開発言語) | 言語名   |

</div>

| フィールド名                   | 単位      | 説明                                    | 備考 |

| `heap_tot`               | バイト     | 現在割り当てられているヒープメモリの総量                  | -  |

| `threadpool_activeCount` | count   | 現在実行中のスレッドプールのスレッド数                   | -  |

</div>
