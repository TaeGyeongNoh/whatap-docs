## app_proc_counter

アプリケーションプロセスのメトリクスです。

- 収集間隔：5秒
- 統計データ：5分、1時間

### Tags

| タグ名       | 説明           | 備考    |
| --------- | ------------ | ----- |
| host_ip   | Host IP      | ユニーク値 |
| oname     | エージェント名      | ユニーク値 |
| okindName | アプリケーションの種類名 | -     |
| onodeName | アプリケーションノード名 | -     |
| type      | アプリケーションタイプ  | 言語名   |

### Fields

| フィールド名   | 単位  | 説明                  | 備考 |
| -------- | --- | ------------------- | -- |
| cputime  | ミリ秒 | CPU使用時間             | -  |
| heap_max | バイト | ヒープメモリの最大サイズ(-Xmx値) | -  |
| heap_tot | バイト | ヒープ合計量              | -  |
| heap_use | バイト | ヒープ使用量              | -  |
