検索キーは、ログデータ内で希望する特定の値にアクセスするための識別子を意味します。 検索キーに該当する実際のデータが検索値です。 左側の![number-s](/img/number-02.png)領域のタグはカテゴリ別にパーシング(parsing) された検索キーです。 タグを選択してフィルターを入力します。 <span class='logwr'>オレンジ色</span>タグはカテゴリーで、<span class='lowfg'>青色</span>タグは検索キーです。

</InDoc>

<InDoc pages='log-search'>

例えば![number-s](/img/number-02.png)領域の<span class='logwr'>AppLog</span>と<span class='logwr'>AppStdOut</span>はカテゴリーであり、その下の<span class='lowfg'>oid</span>などのタグは、パーシング(parsing) された検索キーです。 検索キーは<Cmdname sid="side_log" className="uitext" /> &gt; <Cmdname sid="side_logSetting" className="uitext" />の<Cmdname sid="log_parser_setting" className="uitext" />タブでパーシングロジックを登録することで設定できます。 

</InDoc>
