- ![アイコン](/img/ic-setting.svg) <Cmdname sid="table_settings" className="uitext" />ボタンをクリックすると<Cmdname sid="column_settings" className="uitext" />と<Cmdname sid="ITM08349" className="uitext" />オプションメニューが表示されます。 

  ![ログテーブル設定](/img/log-table-contents-options.png)

1. <Cmdname sid="column_settings" className="uitext" />

    * <Cmdname sid="add_a_column" className="uitext" />：タグを選択して、テーブルにカラムを追加できます。

    * <Cmdname sid="column_order_setting" className="uitext" />：カラムを追加すると、カラム順序の設定にカラムが追加されます。目的のカラムをドラッグして、カラムの順序を変更します。

    :::note 

    <Cmdname sid="add_a_column" className="uitext" />時、***log***カラム選択を解除する場合、<Cmdname sid="ITM08349" className="uitext" />が確認できません。

    :::

2. <Cmdname sid="ITM08349" className="uitext" />

    <ImgLang img='log-table-contents.png' desc='テーブルのコンテンツ設定'/>

    - **ログ表示の有無**

      - チェックが解除されたアイテムは、テーブルに表示されません。デフォルトで**_content_**、<Cmdname sid="tag" className="uitext" />すべてチェックされており、両方の項目を表示します。 

      - 次のように<Cmdname sid="tag" className="uitext" />を解除する場合、テーブルからログの<Cmdname sid="tag" className="uitext" />は表示されません。
        <ImgLang img='log-table-contents-remove-tag.png' desc='ログテーブルコンテンツタグ非表示'/>

    - **タグ管理**

      - <Cmdname sid="tag_management" className="uitext" />一覧にタグを追加すると、追加した順にログのタグが一覧表示されます。タグの順序はドラッグして変更できます。

      - 追加したタグを無効にすると、無効にしたタグはログのタグに表示されません。 

:::note 

同一のプロジェクト内の<Cmdname sid="side_logTail" className="uitext" />、<Cmdname sid="side_logSearch" className="uitext" />、<Cmdname sid="side_logExplorer" className="uitext" />メニューは、テーブル設定を共有します。 

:::
