以前のバージョンとの互換性のため、データベースモニタリングV2以前のサービス画面を利用することができます。 

1. <Cmdname sid="side_management" className="uitext" /> &gt; <Cmdname sid="side_projectManagement" className="uitext" />メニューに移動します。

2. <Cmdname sid="TTL06144" className="uitext" />セクションで<Cmdname sid="TTL06145" className="uitext" />を***V1***に選択してください。

3. <Cmdname sid="save" className="uitext" />ボタンを選択します。

:::note

- 過去のデータを表示した時に画面に表示されないデータが見つかった場合は、<Cmdname sid="TTL06145" className="uitext" />を***V1***に変更してから、以前のバージョンの画面でデータを表示できます。

- 互換性の理由から、この機能は 2024.12.31 までしか利用できません。

:::
