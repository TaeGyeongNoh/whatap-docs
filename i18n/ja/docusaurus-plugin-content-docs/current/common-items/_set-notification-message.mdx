プロジェクトメンバーにメールまたはサードパーティープラグインを通じて伝える、アラート通知メッセージを修正できるユーザー定義機能を提供します。 画面右上の<Cmdname sid="goto_alertTemplate" className="uitext" />ボタンを選択します。 受信するメッセージで必要でないと思われる項目を隠すことも、ユーザーが希望する特別な内容を追加することもできます。

<ImgLang img='notification-set-message.png' desc='通知メッセージの設定' />

:::note

- この機能は、<Cmdname sid="metrics" className="b600" />, <Cmdname sid="composite_metrics" className="b600" />警告通知にのみ適用できます。

- この機能は、プロジェクトの選択 &gt; <Cmdname sid="side_lab" className="uitext" /> &gt; <Cmdname sid="side_alertTemplate" className="uitext" />メニューパスでもアクセスできます。

- この機能は<Cmdname sid="sms_alert_settings" className="b600" />権限を持つメンバーが設定できます。 詳しい内容は[次の文書](../project/project-structure#project-auth)を参照してください。

:::

## 基本画面ガイド

<ImgLang img='notification-set-message-main.png' desc='通知メッセージの設定' />

- アラートメッセージの種類は<Cmdname sid="OCCURRED" className="occur set-noti" />と<Cmdname sid="DSC08323" className="resolve set-noti" />で分類できます。 <Cmdname sid="OCCURRED" className="occur set-noti" />は、イベントが発生した時に配信される通知であり、<Cmdname sid="DSC08323" className="resolve set-noti" />は、イベントが解消された時に配信される通知です。

- 受信したメッセージの詳細項目を隠すには![数字 1](/img/number-01.png)トグルボタンを選択して非活性化にしてください。

- 受信するメッセージの内容を修正するには、![数字 2](/img/number-02.png) ![編集アイコン](/img/ico-edit.svg)<Cmdname sid="edit" className="uitext" />ボタンを選択します。

- ![数字 3](/img/number-03.png)領域は受信メッセージの詳細です。 ![数字 4](/img/number-04.png)領域は受信メッセージの詳細に対する内容です。 ![数字 5](/img/number-05.png) ![プレビューアイコン](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" />ボタンを選択すると、受信するメッセージの内容をプレビューできます。

## 受信メッセージ詳細項目を隠す

受信メッセージの内容で、必要でないと思われる項目を隠すことができます。 各詳細項目の一番右のトグルボタンを選択して、該当項目を受信メッセージで隠すことも、表示することも設定できます。 ![プレビューアイコン](/img/ico-noti.svg)<Cmdname sid="preview" className="uitext" />ボタンを選択すると、受信するメッセージの内容をプレビューできます。

- 詳細項目を活性化する例

  <ImgLang img='notification-set-message-preview.png' desc='ー活性化プレビュー' />

- 詳細項目を非活性化する例

  <ImgLang img='notification-set-message-preview-disabled.png' desc='非活性化プレビュー' />

変更事項を反映するには![保存アイコン](/img/ic-s-check.svg)<Cmdname sid="save" className="uitext" />ボタンを選択します。 他のカテゴリーの変更事項を一挙に反映するには、画面右上の<Cmdname sid="save_all" className="uitext" />ボタンを選択します。

:::note

- 隠した詳細項目を受信メッセージにまた追加するには、該当項目のトグルボタンを選択して活性化します。

- すべての詳細項目を受信メッセージに追加するには、![活性化アイコン](/img/ic-power.svg)<Cmdname sid="active_all" className="uitext" />ボタンを選択します。

:::

## 詳細項目内容を修正する

詳細項目の内容にユーザーが必要な内容を追加できます。 

1. 各詳細項目の一番右の![編集アイコン](/img/ico-edit.svg)<Cmdname sid="edit" className="uitext" />ボタンを選択します。

2. <Cmdname sid="edit_message" className="uitext" />画面が表示されたら、追加内容を入力します。

   <ImgLang img='notification-set-message-edit.png' desc='メッセージ編集' />

3. <Cmdname sid="confirm" className="uitext" />ボタンを選択します。

4. ![プレビューアイコン](/img/ico-noti.svg) <Cmdname sid="preview" className="uitext" />ボタンを選択すると、受信するメッセージの内容を予め確認してください。

   <ImgLang img='notification-set-message-edit-preview.png' desc='メッセージ編集' />

変更事項を反映するには![保存アイコン](/img/ic-s-check.svg)<Cmdname sid="save" className="uitext" />ボタンを選択します。 他のカテゴリーの変更事項を一挙に反映するには、画面右上の<Cmdname sid="save_all" className="uitext" />ボタンを選択します。

:::note

<Cmdname sid="edit_message" className="uitext" />画面でテキストブロックはアラート発生状況によって、変更され出力される変数です。 テキストブロックはドラッグして位置を変更できます。 テキストブロックを削除した場合、画面の下からテキストブロックボタンを選択して、再度追加できます。

<ImgLang img='notification-set-message-edit-block.png' desc='メッセージ編集' />

:::

:::caution

ユーザーによって内容を修正する前のメッセージは、ユーザーの言語設定に合わせて転送されますが、内容の修正後は言語設定に関わらず、ユーザーから定義されたフォーマットで通知を転送します。

:::

## 他のプロジェクトに設定をコピーする

ユーザーが設定した内容をほかのプロジェクトにコピーできます。 

1. 画面右上の<Cmdname sid="apply_to_other_project" className="uitext" />ボタンを選択します。

2. <Cmdname sid="apply_project" className="uitext" />画面が表示されたら、設定内容をコピーするプロジェクトを選択します。 すべてのプロジェクトを選択するには<Cmdname sid="select_all" className="uitext" />を選択します。

3. <Cmdname sid="apply" className="uitext" />ボタンを選択します。

:::info

現在のプロジェクトと同一の製品タイプのみ選択できます。

:::

## 初期化

ユーザーが設定したすべての変更事項を初期化するには、![初期化アイコン](/img/ico-reset.svg) <Cmdname sid="initialized" className="uitext" />ボタンを選択します。 すべてのカテゴリーの変更事項を初期化するには、画面右上の<Cmdname sid="alerttemplate_reset_all" className="uitext" />ボタンを選択します。 確認メッセージが表示されたら、<Cmdname sid="confirm" className="uitext" />ボタンを選択します。

:::caution

初期化された設定は復旧できません。

:::
