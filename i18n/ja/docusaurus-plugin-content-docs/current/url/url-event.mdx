ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_イベント設定_**

URLモニタリングは、イベント設定を通じてメール形式のアラート通知を提供します。モニタリング中のプロジェクト画面では、リアルタイムで発生するイベントについても通知を提供します。 登録されたURLのイベントを対象に、次のように個別に設定ができます。 

1.  イベントリスト画面で編集するURLの ![修正アイコン](/img/ico-edit.svg) **_修正_** アイコンを選択してください。

    <ImgLang img='url-event.png' desc='イベントリスト' />

2.  そのURLに対する**_イベント設定の修正_**ウィンドウを確認してください。 

    <ImgLang img='url-event-mod.png' desc='URLイベント設定' />

    **_ステータス_**、 **_経過時間_** などのイベントルールは、モニタリングメトリクスと条件式を組み合わせて設定します。 

    -   **ステータスイベント**

        ステータスイベントは、選択した **_ステータス_** コードのオプションによってイベントが発生します。

        <ImgLang img='alert_status.png' desc='ステータスイベント' />

    -   **経過時間イベント**

        経過時間イベントは、リアルタイムで収集中の**_経過時間_**値に、ユーザーが入力したしきい値と条件文に該当する場合、イベントが発生します。

        <ImgLang img='alert_elapsed.png' desc='経過時間イベント' />

:::note 

**通知メッセージとは？**

そのルールでイベントが発生した場合に表示するタイトルです。 個々のイベントルールで通知メッセージを設定できます。 設定を適用しない場合、基本的に表示される通知メッセージは **_ステータスコード_**、 **_経過時間_**、 **_URL：詳細理由_** です。

:::
