Focusは、WhaTapモニタリングで提供されていない任意のデータを時系列でアップロードするプログラムです。 

運用チームが自社開発したチェックスクリプトやデータベースクエリを定期的または一回のみ操作して収集した値をアップロードできます。アップロードされた値は、そのプロジェクトの分析・メトリクスチャートメニューで確認できます。また、Flexボードに追加することができます。

Focusを使用するには、WhaTapプロジェクトの中からServer、Applicatoon(Java、PHP、NodeJS、Python)、Database、Kubernetesプロジェクトのいずれかを選択した後に、プロジェクトアクセスキー文字列と収集サーバーIPを実行オプションと指定して使用します。
