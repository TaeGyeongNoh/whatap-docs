Javaウェブアプリケーションサーバーに接続されたユーザー数をエージェント設定を通じて集計する方法を案内します。リアルタイムのユーザー集計の活性化有無の設定、ユーザー数を集計するためのクッキー制限設定、IPアドレスまたはHTTPヘッダーの特定値に基づいたユーザーの集計方法、クライアントIP情報追跡のためのHTTPヘッダー設定などを含みます。

- **trace_user_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  リアルタイムのユーザー集計の有無を設定します。ユーザー追跡オプションが重複設定されている場合、動作の優先順位は次のとおりです。

  > 1. **trace_user_using_ip** / **wclient_using_ip**
  > 2. **trace_user_using_jsession**
  > 3. **user_header_ticket** / **wclient_header_ticket**

  :::note

  Javaエージェント 2.2.0バージョン以降の場合、`wclient_using_ip`、`wclient_header_ticket`オプションをご利用ください。

  :::
