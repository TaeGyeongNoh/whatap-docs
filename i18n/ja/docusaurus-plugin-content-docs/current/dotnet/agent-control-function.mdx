.NETアプリケーションサーバーの機能を制御します。

- **stat_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  統計情報の追跡機能を有効にします。 5分単位で収集されたトランザクション、SQL、HTTPCALL、UserAgent、Client IPなどの統計データなどが含まれます。 

- **license** <span class='type'>String</span>

  既定値`NONE`

  エージェントのインストール時にサーバーから付与されたアクセスキーを入力します。 アクセスキーは、エージェントが属するプロジェクトとセキュリティ通信のためのパスワードキーが含まれます。

- **active_stack_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  アクティブスタック追跡を有効にします。**_分析_** > **_スタック_**メニューから**_トップスタック_**および**_ユニークスタック_**、**_アクティブスタック_**が表示します。

- **realtime_user_thinktime_max** <span class='type'>Milliseconds</span>

  既定値`300000`

  リアルタイムにブラウザのユーザー数を測定する場合、同じユーザーとして認識される呼び出し間隔の最大値を設定します。

- **time_sync_interval_ms** <span class='type'>Milliseconds</span>

  既定値`300000`

  エージェントとサーバー間を同期する時間間隔を設定します。同期しない場合は、`0`に設定します。

- **text_reset** <span class='type'>Int</span>

  既定値`0`

  WhaTapエージェントは、送信されるテキストタイプのデータをhash処理するため、翌日まで再送信されません。以前の設定値と異なる値を入力する場合は、再度送信します。

  :::note

  トランザクションURL、SQL Stringなどがテキストタイプデータに対応します。

  :::

- **profile_method_enabled** <span class='type'>Boolean</span>

  既定値 `true`

  実行時間`profile_method_time` オプションの値を超えるユーザー機能の情報をトランザクションの詳細情報から確認することができます。

  :::note

  設定した値を有効にするには、IISを再起動してください。

  :::

- **profile_method_time** <span class='type'>Milliseconds</span>

  既定値 `1000`

  `profile_method_enabled`オプションで収集する関数の実行時間を設定できます。

  :::note

  設定した値を有効にするには、IISを再起動してください。

  :::

- **profile_method_stack_enabled** <span class='type'>Boolean</span>

  既定値`false`

  ユーザー関数のスタック情報をトランザクションの詳細から確認できます。

  :::note

  設定した値を有効にするには、IISを再起動してください。

  :::
