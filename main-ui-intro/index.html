홈 화면(UI) 안내

와탭 모니터링 서비스에 로그인하면 펼쳐지는 초기 화면의 요소들을 소개합니다. 초기 화면의 기본 메뉴는 프로젝트 목록입니다.

고정 메뉴 영역​

  • 메뉴: 왼쪽에는 사용자가 진입한 메뉴 경로를 표시합니다.

  • 공지사항: 와탭 모니터링 서비스의 공지사항을 확인할 수 있습니다.

  • 테마: 현재 페이지의 테마를 변경할 수 있습니다. 기본값은 Light 모드입니다.

    • Light: 기본 적용된 테마입니다.
    • Dark: 검은 배경의 테마를 적용합니다.
    • Presentation: 빔 프로젝트를 이용할 경우 최적화된 색상을 적용합니다.
    • Add Custom Theme: 사용자가 원하는 색상을 설정할 수 있습니다.

  • 와탭 Docs

    와탭 Docs 기술 문서 페이지로 이동합니다. 와탭 Docs에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 채팅 문의

    채팅을 통해 기술 및 비용/결제 등과 관련한 온라인 고객 지원을 받을 수 있습니다. 아이콘을 선택하면 채팅 창이 나타납니다.

    채팅 아이콘이 표시되지 않는다면 다음 문서를 참조하세요.

  • 프로필 아이콘: 가장 오른쪽에 위치한 아이콘을 선택하세요. 계정 정보 관리, 조직 생성, 조직 관리 그리고 사용자 언어를 선택할 수 있는 언어 기능을 제공합니다. 이용 중인 서비스 화면에서 로그아웃하려면 로그아웃 버튼을 선택하세요.

프로젝트 검색 영역​

  • 프로젝트 명 / PCODE(텍스트 입력란): 프로젝트 이름 또는 프로젝트 코드를 입력해 원하는 프로젝트를 검색할 수 있습니다.

    프로젝트 코드(PCODE)는 해당 프로젝트의 관리 > 프로젝트 관리 메뉴의 프로젝트 코드 항목에서 확인할 수 있습니다.

  • 즐겨찾기: 즐겨찾기 설정한 프로젝트만 목록에 나타납니다. 프로젝트를 즐겨찾기 설정을 하려면 프로젝트 목록에 나타난 프로젝트 카드뷰 왼쪽 상단에 을 선택하세요. 즐겨찾기 설정을 해제하려면 다시 을 선택하세요.

  • 확장 카드 뷰: 프로젝트 목록을 카드 형태로 확인할 수 있습니다. 프로젝트의 기본 정보를 함께 표시합니다.

  • 축소 카드 뷰: 프로젝트 목록을 확장 카드 뷰 보다 축소된 카드의 형태로 목록을 확인할 수 있습니다.

  • 리스트 뷰: 프로젝트 목록을 상품 카달로그별로 분류해 표 형식으로 확인할 수 있습니다.

  • 15 / page: 페이지당 목록에 나타나는 프로젝트 수를 설정할 수 있습니다. 한 페이지에 최대 100개까지 나타나도록 설정할 수 있습니다. 다른 페이지로 이동하려면 페이지 번호를 선택하세요.

조직 및 그룹, 프로젝트 영역​

  • 비조직: 조직을 선택하거나 조직 관리 메뉴로 이동할 수 있습니다. 조직 관리에 대한 자세한 내용은 다음 문서를 참조하세요.

  • + 그룹: 그룹을 새로 추가할 수 있습니다. 그룹 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • + 프로젝트: 프로젝트를 추가할 수 있습니다. 프로젝트 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 검색: 그룹 또는 프로젝트, 에이전트 단위로 검색할 수 있습니다. 프로젝트 검색에 대한 자세한 내용은 다음 문서를 참조하세요.

  • 프로젝트 타입 필터: 필터링을 설정해 원하는 상품 카테고리별로 프로젝트 목록을 확인할 수 있습니다. 설정된 값에 따라 전체 프로젝트 아래에 목록을 표시합니다.

  • 또는 버튼을 선택하면 프로젝트 목록 영역을 숨기거나 표시할 수 있습니다.

  • 전체 프로젝트: 전체 프로젝트 목록을 확인할 수 있습니다. 특정 프로젝트를 선택해 진입한 상태에서 이 메뉴를 선택하면 초기 프로젝트 목록 화면으로 이동할 수 있습니다.

  • 영역 아래에는 프로젝트 목록을 확인할 수 있습니다. 그룹은 모양으로 표시합니다. 그룹 외 프로젝트는 상품 종류별 아이콘으로 표시합니다.

통합 메뉴 영역​

  • 메뉴: 마우스를 오버한 상태에서 드래그하면 메뉴 영역을 위 또는 아래로 크기를 조절할 수 있습니다.
  • 사이트맵: 사용자가 진입한 메뉴 경로에서 선택할 수 있는 모든 메뉴 목록을 확인하고 이동할 수 있습니다.

    진입한 메뉴 경로 또는 프로젝트에 따라 사이트맵 창에서 선택할 수 있는 메뉴들은 다르게 나타날 수 있습니다.

  • 프로젝트 목록: 전체 프로젝트 목록을 확인할 수 있습니다.
  • Flex 보드: 사용자 정의형 통합 대시보드입니다. 애플리케이션, 서버, 데이터베이스, 컨테이너 등 프로젝트에서 수집한 메트릭스로 실시간 대시보드를 만들 수 있습니다.
    상품별 문서
    • 애플리케이션 모니터링 > Flex 보드
    • 서버 모니터링 > Flex 보드
    • 데이터베이스 모니터링 > Flex 보드
    • 쿠버네티스 모니터링 > Flex 보드
    • Amazon ECS > Flex 보드
    • Amazon CloudWatch > Flex 보드
    • Azure Monitor > Flex 보드
    • Naver Cloud Monitoring > Flex 보드
    • Oracle Cloud Monitor > Flex 보드
  • 통합 보고서: 여러 프로젝트에 대한 보고서를 간편하게 생성할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.
  • 통합 APP. 그룹 토폴로지: 토폴로지 기능을 지원하는 여러 프로젝트들을 일괄 선택해 단일 토폴로지로 확인할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.
  • 실시간 알림 목록: 사용자가 권한을 보유한 모든 프로젝트들의 알림을 실시간으로 확인하고, 알림 현황을 한눈에 파악할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.
  • 관리: 전체 그룹 및 멤버를 관리할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

프로젝트 검색하기​

  1. 화면 왼쪽 위에서 버튼을 선택하면 프로젝트 검색 창이 나타납니다.

  2. 검색할 프로젝트 또는 그룹 이름을 입력하면 일치하는 프로젝트 목록이 검색 창 아래에 나타납니다.

  3. 원하는 프로젝트를 선택하세요.

해당 프로젝트 화면으로 이동합니다.

검색 창에서 에이전트 기준으로 프로젝트를 검색하려면 그룹/프로젝트를 클릭한 다음 드랍다운 목록에서 Agent를 선택하세요.

프로젝트 목록​

사용자가 멤버로 등록된 프로젝트 목록을 확인할 수 있습니다.

하단 메뉴 영역​

  • 이용내역: 결제 정보, 이용 요금 등과 관련한 메뉴를 이용할 수 있습니다. 자세한 내용은 다음 문서를 참조하세요.

  • : 사이드 메뉴를 감추거나 표시할 수 있습니다. 사이드 메뉴가 감춰진 상태에서 버튼을 선택하면 사이드 메뉴를 표시합니다.

  • : 와탭 모니터링 서비스 화면을 전체 화면 크기로 이용할 수 있습니다. 전체 화면을 종료하려면 ESC 키를 선택하세요.

상품 소개 메뉴​

사이드 메뉴의 가장 왼쪽의 아이콘 버튼을 선택하면 상품 소개 페이지로 이동할 수 있습니다. 상품별 카테고리로 분류되어 있으며, 소개 페이지로 이동한 다음 원하는 상품에서 시작하기 버튼을 선택하세요. 프로젝트 생성 메뉴로 이동합니다.

상품 분류설명
Java, Node.js, PHP, Python, .NET, Go 상품의 애플리케이션 모니터링을 위한 프로젝트를 생성할 수 있습니다.
Server 모니터링을 위한 프로젝트를 생성할 수 있습니다.
Kubernetes 모니터링을 위한 프로젝트를 생성할 수 있습니다.
MySQL, SQL Server, Oracle, PostgreSQL, Redis, MongoDB, CUBRID, Tibero, Altibase 상품의 데이터베이스 모니터링을 위한 프로젝트를 생성할 수 있습니다.
URL 모니터링을 위한 프로젝트를 생성할 수 있습니다.
Amazon CloudWatch, Amazon ECS, Azure Monitor, Oracle Cloud Monitor, Naver Cloud Monitoring 상품의 클라우드 모니터링을 위한 프로젝트를 생성할 수 있습니다.
Log 모니터링에 대한 소개와 설정 방법을 안내합니다.
Browser 모니터링을 위한 프로젝트를 생성할 수 있습니다.
