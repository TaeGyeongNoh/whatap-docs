통합 대시보드

홈 화면 > 사이트 맵 > 통합 대시보드

사용자가 속한 모든 프로젝트의 데이터를 조회할 수 있는 대시보드를 생성할 수 있습니다. 서로 다른 상품군의 현황을 대시보드 위에 위젯으로 구성해 한눈에 파악할 수 있습니다.

통합 대시보드 메뉴를 처음 이용한다면 Flex 보드 메뉴를 사용할 것을 권장합니다. Flex 보드에 대한 자세한 내용은 각 상품별 문서를 참조하세요.

상품별 Flex 보드 문서
  • 애플리케이션 모니터링 > Flex 보드
  • 서버 모니터링 > Flex 보드
  • 데이터베이스 모니터링 > Flex 보드
  • 쿠버네티스 모니터링 > Flex 보드
  • Amazon ECS > Flex 보드
  • Amazon CloudWatch > Flex 보드
  • Azure Monitor > Flex 보드
  • Naver Cloud Monitoring > Flex 보드
  • Oracle Cloud Monitor > Flex 보드

통합 대시보드 생성하기​

템플릿을 이용해 대시보드 생성하기​

와탭에서 제공하는 템플릿을 통해 대시보드를 쉽고 빠르게 제작할 수 있습니다.

  1. 화면 위에 대시보드 생성하기를 선택하세요.

  2. 대시보드 생성하기 메뉴 아래에 템플릿 목록이 나타납니다.

  3. 원하는 템플릿을 선택하면 대시보드 템플릿 창이 나타납니다.

  4. 대시보드 이름과 설명을 입력한 다음 데이터를 불러올 프로젝트를 선택하세요. 선택할 수 있는 프로젝트의 개수는 템플릿에 따라 다를 수 있습니다.

  5. 확인 버튼을 선택하세요.

통합 대시보드 목록에서 생성한 대시보드를 확인할 수 있습니다. 목록에서 새로 생성한 대시보드를 선택하면 대시보드 화면으로 이동합니다.

빈 템플릿으로 대시보드 생성하기​

  1. 화면 위에 대시보드 생성하기를 선택하세요.

  2. 대시보드 생성하기 메뉴 아래에 템플릿 목록이 나타납니다.

  3. 가장 왼쪽에 빈 대시보드 만들기를 선택하세요.

  4. 대시보드 생성하기 창이 나타나면 이름과 설명을 입력한 다음 저장 버튼을 선택하세요.

  5. 통합 대시보드 목록에서 생성한 대시보드를 확인할 수 있습니다. 목록에서 새로 생성한 대시보드를 선택하면 대시보드 화면으로 이동합니다.

  6. 화면 위에 + 위젯 추가 버튼을 선택해 원하는 위젯을 추가하세요.

위젯 추가하기​

대시보드 화면에서 위젯을 추가해 서로 다른 프로젝트의 지표를 확인할 수 있습니다. 예를 들어, 사용자가 A 프로젝트의 히트맵 차트 위젯과 B 프로젝트의 히트맵 차트 위젯을 대시보드에 추가해 동시 확인할 수 있습니다.

  1. 통합 대시보드 목록에서 위젯을 추가할 대시보드를 선택해 대시보드 화면으로 이동하세요.

  2. 화면 위에 + 위젯 추가 버튼을 선택하세요.

  3. 위젯 생성 창이 나타나면 원하는 위젯 이름과 위젯 차트의 종류, 프로젝트를 차례로 선택하세요.

    • 선택한 위젯 차트의 종류에 따라 선택할 수 있는 상품 카테고리 및 프로젝트가 다를 수 있습니다.

    • 하나의 상품 카테고리에서는 여러 개의 프로젝트를 선택할 수 있습니다. 다른 상품 카테고리와 중복해 선택할 수 없습니다.

    • Custom chart는 지표 및 오브젝트 병합, 기간 필터 등의 옵션을 설정할 수 있습니다.

  4. 다음 버튼을 선택하세요.

  5. 각 프로젝트에 속한 에이전트를 에이전트 또는 그룹 ID, 노드 ID 기준으로 선택할 수 있습니다. 프로젝트 항목 가장 왼쪽에 ▶︎ 버튼을 선택하세요.

    • 위젯으로 지표를 확인할 에이전트를 클릭한 다음 적용 버튼을 선택하세요.

    • 모든 에이전트의 지표를 확인하려면 전체를 클릭한 다음 적용 버튼을 선택하세요.

  6. 완료 버튼을 선택하세요.

통합 대시보드 화면에서 추가한 위젯을 확인하세요.

  • 위젯의 오른쪽 아래 부분을 마우스 오버해 드래그하면 크기를 조절할 수 있습니다.

  • 위젯의 제목 부분을 마우스 오버해 드래그하면 위치를 이동할 수 있습니다.

  • 위젯에 대한 정보를 확인하려면 위젯의 오른쪽 위에 버튼으로 마우스를 오버하세요.

위젯 수정하기​

  1. 위젯의 오른쪽 위에 버튼을 선택하세요.

  2. 위젯 생성 창이 나타나면 위젯 차트 종류와 프로젝트를 선택해 변경할 수 있습니다.

  3. 다음 버튼을 선택하세요.

  4. 프로젝트의 속한 별도의 에이전트를 선택하려면 프로젝트 항목 가장 왼쪽에 ▶︎ 버튼을 선택한 다음 원하는 에이전트를 선택하세요.

  5. 완료 버튼을 선택하세요.

통합 대시보드 화면에서 수정한 위젯을 확인하세요.

위젯 삭제하기​

통합 대시보드 화면에서 삭제하려는 위젯의 오른쪽 위에 버튼을 선택하세요. 확인 메시지가 나타나면 확인 버튼을 선택하세요.

통합 대시보드 이름 및 설명 수정하기​

  1. 통합 대시보드 목록에서 이름 또는 설명을 수정하려는 대시보드 항목의 버튼을 선택하세요.

  2. 대시보드 수정 창이 나타나면 이름과 설명을 수정하세요.

  3. 수정을 완료한 다음 저장 버튼을 선택하세요.

통합 대시보드 목록에서 수정한 내용을 확인할 수 있습니다.

통합 대시보드 공유하기​

통합 대시보드에서 생성한 대시보드를 다른 계정의 사용자와 공유하고 재사용할 수 있습니다.

복사하기​

사용자가 생성한 대시보드를 복사하거나 다른 사용자와 공유할 수 있습니다.

통합 대시보드 목록에서 복사하려는 대시보드 항목의 버튼을 선택하세요. 나에게 복사 또는 다른 사람에게 복사 버튼을 선택하세요.

  • 나에게 복사를 선택하면 대시보드 목록에 '(Copied)' 접두어가 추가된 항목을 확인할 수 있습니다.

  • 다른 사람에게 복사를 선택하면 다른 사용자의 이메일을 입력한 다음 보내기 버튼을 선택하세요. 읽기 전용 옵션을 선택하면 공유 받은 사용자가 대시보드를 수정할 수 없습니다.

    대시보드를 공유한 이후 편집한 대시보드의 변경 사항은 공유되지 않습니다.

내보내기, 가져오기​

json 파일로 저장한 통합 대시보드 설정을 다른 사용자 계정에서 가져와 복사할 수 있습니다. 대시보드 관리자 권한의 기능입니다.

  1. 통합 대시보드 목록에서 다른 사람에게 공유하려는 대시보드 항목의 버튼을 선택하세요.

  2. Export Dashboard 창이 나타나면 내보내기 버튼을 선택하세요.

  3. json 파일을 다운로드합니다.

  4. 다른 사용자에게 다운로드한 파일을 전달하세요.

  5. 통합 대시보드 화면 위에 가져오기 버튼을 선택하세요.

  6. Export Dashboard 창이 나타나면 저장 버튼을 선택하세요.

대시보드 목록에 '(Imported)' 접두어가 추가된 항목을 확인할 수 있습니다.

통합 대시보드 삭제하기​

통합 대시보드 목록에서 삭제하려는 대시보드 항목의 버튼을 선택하세요. 팝업 메시지에서 삭제 버튼을 선택하세요.
