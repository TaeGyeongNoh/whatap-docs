Container images

Home > Select Project > Container > Container Images

The Container Images menu provides available images in the containers, number of containers running with those images, and the total usage amounts for CPU and memory in the list format.

Filter​

If Filter key is selected in the Filter window, the filtering for Registry, Name, Tag, and Repo Digest is provided.

Download CSV​

By selecting Download CSV in the upper right corner, you can download the original data as a file in CSV format. The file name is generated by adding the project code and timestamp at the time of data inquiry after kubernetes_container_images_.

e.g. kubernetes_container_images_{pcode}_{timestamp}.csv

Container details view​

If you select the View containers in detail icon on the right of the Name column, the Container list menu appears where the image has been searched.
