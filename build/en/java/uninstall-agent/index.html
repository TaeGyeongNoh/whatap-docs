Agent removal and rollback

Agent deletion​

On the application server, delete the JVM's -javaagent option and then restart the server. Afterwards, delete the $WHATAP_HOME environment variable.

Agent rollback​

  1. Select a project for rollback in WhaTap Monitoring Service.

  2. On the left of the screen, select Management > Agent Update.

  3. Select Other versions.

  4. Select a version for rollback and then click Selected Version or Download to all agents with the selected version.

  5. During agent installation, in the modified script, change the WHATAP_JAR environment variable to the version for rollback.
