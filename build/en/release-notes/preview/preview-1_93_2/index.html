Hotfix : 2023-01-10​

Common​

Flex Board​

  • Fixed When creating or editing a dashboard, the dashboard title has been limited to 100 characters.

  • Fixed When using the Filter function after selecting a dashboard, deleted the Close button that appears when modifying the saved filters.

  • Fixed Modified to open the detailed information window for only the products that support the active transaction function when the equalizer chart is clicked.

  • Fixed Improved the loading layout of the detailed information window that appears when the active transaction widget is clicked.

  • Fixed Fixed the problem where the animation of the time selection UI option is not updated.

Metrics Chart​

  • Fixed Fixed the bug where all agents are selected when changing the time in the widget.

  • Fixed In Application Dashboard, fixed the bug where double selection of agents is made when going to Metrics Chart.

Alert​

  • Fixed In Event Setting > Composite Metrics > Add alert policy, modified not to be saved without entering the title and message.

  • Fixed In Event Setting > Composite Metrics > Add alert policy > Creating as a Chart > Show Event > Text mode, fixed the bug where loading continues when changing to the widget type or table.

  • Fixed In the Add alert policy window under Event Setting, fixed the bug where the URL path is changed to the application type (apm), if Project Notification Setting Menu or its shortcut is clicked.

  • Fixed In Notification Setting > Event Notification Settings Per-User, fixed the bug where invalid time is saved if of the receiving time selector option is clicked and deleted the button.

  • Fixed If Event History is accessed for the first time, fixed the bug where the data is not applied to the table list according to the set filters.

Log​

Fixed Fixed the issue where in Live Tail, the Tail data calls at an interval of 3 seconds are not delayed.

User Behavior​

Fixed In Sitemap > Management > User Behavior, fixed the issue where text in the table overflows outside the table area.

Agent/Server Configuration​

  • Change In a project in the Kubernetes environment, the agent list is divided into master, node, and application units.

  • Feature Added the agent alias editing function.

  • Fixed Fixed the bug where the search filter does not work in some projects.

  • Fixed Fixed the problem where the agent list is collapsed because the selected agent cannot be clicked while the agent list is expanded.

  • Deprecate Removed the Applying Multiple function from the server project or the project in the Kubernetes environment.

Application​

Topology​

Fixed In Sitemap > STAT/REPORT > Topology > Application Statistics Topology, fixed the issue where an error occurs when an agent is selected.

Analysis​

Fixed In CUBE, fixed the bug where the agent selection filter is not applied to charts.

Fixed In Remote File Viewer, renamed the menu on the screen to match the menu name on the left of the screen.

Container​

Container Map​

  • Fixed Fixed the bug where tooltip text does not appear when you hover your mouse over the icon.

  • Fixed Fixed the bug where the value in the input field is different from the selected value after the Exclude or Include option is selected while using the filter function.

AWS AZURE​

Project Creation​

  • Change Added the installation guide text upon creation of an Azure project.

  • Change Changed the Validity period(hours) to be entered as mandatory when creating an Amazon CloudWatch project.

URL​

  • Fixed Deleted the real-time query () button in the window that appears after clicking the chart in URL List.
