Preview 2.8.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.8.0.a3​

2024년 08월 08일

공통​

Fixed 일부 텍스트에 적용되는 테마가 누락된 문제 수정

Preview 2.8.0.a2​

2024년 08월 08일

공통​

  • Feature 다음 메뉴 경로에서 멤버 목록을 CSV 형식의 파일로 다운로드할 수 있는 기능 추가

    • 홈 화면 > 프로젝트 선택 > 관리 > 프로젝트 멤버
    • 홈 화면 > 관리 > 통합 멤버 관리
    • 홈 화면 > 조직 관리 > 멤버 목록

  • Changed 통합 그룹 관리 버튼 위치 및 UI 변경

  • Fixed 통합 Flex 보드 또는 Flex 보드 메뉴의 리치 텍스트 위젯에서 수정 화면을 열고 닫는 과정에서 저장하지 않은 내용이 유지되는 문제 수정

  • Fixed 통합 Flex 보드 또는 Flex 보드 메뉴에서 리치 텍스트 위젯의 수정 화면에서 텍스트와 툴바 버튼의 툴팁이 겹쳐 보이지 않도록 툴팁 요소에 테마 설정

  • Fixed 웹 브라우저의 화면 크기에 따라 사이드 메뉴에서 조직 목록의 스크롤 오류 수정

Application​

  • Changed 대시보드 > 애플리케이션 대시보드 메뉴에서 액티브 트랜잭션 이퀄라이저 위젯 이름을 액티브 트랜잭션으로 변경, 위젯 이름을 줄바꿈 없이 한 줄로 표시하도록 수정

  • Fixed 트랜잭션의 SQL 문에서 치환된 변수 # 을 $로 치환해서 표시하지 않도록 수정

Server​

Fixed 서버 설정 > 실행 환경 변수 또는 환경변수 메뉴에서 에이전트 목록 확장 버튼을 선택하면 화면 상단의 메뉴명이 에이전트 로그 뷰어로 변경되는 문제 수정

Database​

Oracle V2, Oracle Pro​

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 Main Waits 위젯의 데이터를 active_session 카테고리에서 조회하도록 변경

  • Fixed 다음 메뉴 경로에서 holder type, waiter type 값이 표시되지 않는 오류 수정

    • 대시보드 > 인스턴스 모니터링 메뉴의 락 트리 테이블

    • 분석 > 락 트리

Oracle V2​

Fixed 다음 메뉴 경로에서 같은 sid의 holder와 waiter가 락(Lock)으로 표기되는 오류 수정

  • 대시보드 > 인스턴스 모니터링 메뉴의 락 트리 테이블

  • 분석 > 락 트리

Browser​

  • Changed 대시보드 > 브라우저 에러 > 에러 메세지 테이블 섹션의 UI 개선

  • Changed 대시보드 > 브라우저 에러 > 에러 메세지 테이블 > 에러 상세 > 페이지별 에러 건수 섹션의 UI 개선

Network​

Changed 사용자 언어가 영어일 때, 관리 > 에이전트 설치 메뉴의 설치 전 유의사항 섹션에서 Version을 Kernel Version으로 수정

Log​

  • Changed 로그 > 로그 설정 > 로그모니터링 시작하기에서 카테고리별 데이터 보존 기간 항목 중 #이 붙은 카테고리는 표시되지 않도록 처리

  • Fixed 다음 메뉴 경로에서 컬럼 설정을 진행하면 로그 원문이 노출되지 않는 문제 수정

    • 로그 > 라이브 테일
    • 로그 > 로그 트렌드
    • 로그 > 로그 검색
    • 트레이스 분석 > 트랜잭션 로그

Preview 2.8.0.a1​

2024년 08월 01일

공통​

  • Feature 통합 Flex 보드 또는 Flex 보드 메뉴에서 메트릭스 타입 필터 내 조건 간 OR 선택지 추가

  • Feature 통합 Flex 보드 또는 Flex 보드 메뉴에서 리치(Rich) 텍스트 위젯의 글꼴 색상 선택 옵션 추가

  • Changed 테마 색상 일부 변경

  • Changed 관리 > 에이전트 설치 메뉴에서 액세스 키 관련 주의 문구 제거

  • Fixed 모바일 기기의 브라우저로 와탭 서비스 화면 접속 시 하단 영역이 가려지는 문제 수정

  • Fixed 통합 Flex 보드 또는 Flex 보드 메뉴에서 모든 메트릭스로 위젯을 추가하면 지표에 대한 설명이 위젯 제목으로 표시되는 문제 수정

  • Fixed 통합 Flex 보드 메뉴에서 읽기 전용으로 공유 받은 대시보드의 그룹 위젯에 포함된 위젯들이 데이터를 호출하지 못하는 문제 수정

  • Fixed 통합 Flex 보드 메뉴의 그룹 위젯에서 선택한 필터가 없고 대시보드 필터를 적용할 때, 그룹의 하위 위젯에 대시보드 필터가 적용되지 않는 문제 수정

  • Fixed 경고 알림 > 이벤트 설정 V2 메뉴의 이벤트 추가 또는 수정 화면에서 Warning 또는 Critical 지표 중 하나만 활성화된 경우, 동일한 설정값을 입력하면 저장되지 않는 문제 수정

  • Fixed 경고 알림 > 이벤트 설정 V2 메뉴의 복합 메트릭스 이벤트 추가 또는 수정 화면에서 MXQL 데이터가 조회되지 않는 문제 수정

    이벤트 데이터 조회 섹션의 필터 옵션으로 문자열 타입 값을 갖는 태그 또는 필드를 설정 후 텍스트 모드로 전환하면 MXQL 데이터가 조회되지 않는 문제를 수정했습니다.

Application​

  • Changed 트레이스 분석 팝업창의 필터링 선택 상자를 그룹 형식의 버튼으로 변경

  • Changed 사용자 설정 언어가 영어일 때, 대시보드 > 애플리케이션 대시보드 메뉴에서 Apdex 위젯 안내 모달의 가독성 향상을 위해 크기 수정

Server​

Fixed 경고 알림 > 이벤트 설정 메뉴에서 이벤트를 추가하거나 수정할 때 이벤트 발행 조건 내 폰트를 기본 폰트로 수정

Kubernetes​

  • Changed 워크로드 > 컨테이너 애플리케이션 목록 메뉴에서 ApplicationName, ApplicationId 컬럼을 애플리케이션 정보 컬럼 하위로 이동

  • Fixed 워크로드 > 컨테이너 애플리케이션 목록 메뉴에서 매핑된 컨테이너 정보가 없는 경우 경고 메시지가 표시되지 않는 버그 수정

  • Fixed 대시보드 > 컨테이너 맵 메뉴의 컨테이너 요약 분석에서 이미지 리소스 정보 오류 수정

Database​

공통​

  • Feature 세션 킬 기능을 성공하거나 실패하면 사용자 행위로 기록하는 기능 추가

  • Changed 관리 > 프로젝트 멤버 메뉴에서 와탭 지원팀 초대하기 기능 사용 시 dbx@whatap.io 계정이 함께 초대되지 않도록 수정

PostgreSQL​

Feature 대시보드 > 멀티 인스턴스 모니터링 메뉴에 현재 진행 중인 vacuum 프로세스 수를 확인할 수 있는 Vacuum Sessions 위젯 추가

MySQL​

Fixed 분석 > 데드락 메뉴에서 조회한 데이터가 없을 때 표시되는 메시지가 절반만 표시되는 문제 수정

Oracle V2, Oracle Pro​

  • Feature SQL 상세 팝업창의 Runtime Plan, Explain Plan 테이블에 FILTER_PREDICATES 컬럼 추가

  • Fixed 통계/보고서 > SQL 통계 메뉴에서 SQL 상세 팝업창으로 진입했을 때 OBJECT_NAME 컬럼의 링크가 활성화되지 않는 문제 수정

  • Fixed 통계/보고서 > SQL 통계 메뉴에서 상세 차트의 elapse time 값이 비정상적으로 높은 문제 수정

  • Fixed 대시보드 > 인스턴스 목록 메뉴에 Cluster Name을 표기하도록 수정

Redis V1, MongoDB V1​

Fixed 대시보드 > 인스턴스 목록 메뉴에 Cluster Name을 표기하지 않도록 수정

Preview 2.8.0.a0​

2024년 07월 25일

공통​

  • Changed 세로 이퀄라이저 바 차트에서 개별 그래프의 라벨 표시 제거

Kubernetes​

Browser​
