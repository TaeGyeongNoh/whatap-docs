Service 1.90.0

2022-10-24​

コンテナマップ​

Fixed ContainerカテゴリとPodカテゴリのヒットマップチャート表示される情報が異なる現象を修正

共通​

Fixed アカウント情報 - 既存のパスワードが9文字以下の場合、新しいパスワードを変更できない問題を修正

2022-10-20​

ログ​

  • Fixed 無料試用期間が画面に表示される現象を修正(NaN → 0)
  • Fixed ログモニタリングの非活性化状態のプロジェクトを開く際に「ログモニタリングの活性化案内」メッセージに内容がない不具合を修正
  • Fixed ライブTail メニューで構文解析されたキーを検索するAPIを呼び出さない現象を修正

URL​

Fixed URL一覧 - 名前変更ボタンの誤字(cancle → cancel) を修正

2022-10-18​

チャート​

  • Feature Out of Memory メトリクスチャートの移動ロジックを追加 - 分析フローを生成のため、行をクリックするとメトリクスチャートに移動するよう修正
  • Fixed ヒットマップのダークテーマを修正し、画面処理を改善するよう修正

リソース​

  • Change リソース > マスターメタ情報 > 「データなし」メッセージを マスターメタデータなしに修正

コンテナマップ​

  • Change 概要ビューウィンドウが開いた状態でステータス別のテキスト数が修正されます。
  • Change ログ検索タブで(ログカテゴリが存在する場合) ログ対象のカテゴリを修正
  • Fixed カテゴリがPodの場合にコンテナが複数あると、一部のコンテナ情報のみが表示されるバグを修正

アプリケーションの状況と性能の推移​

  • Fixed エージェント情報が誤って表示されるエラーを修正

ログ​

  • Feature ライブTail, 検索機、エクスプローラーにテーブルカラムの順序変更機能を追加

  • Change ログ設定のデータ保持期間オプションを修正

    • トライアルプロジェクト：最大3日までデータを保持
    • 有料プラン変更プロジェクト：最大40日のデータを保持
    • front.conf ファイルにui_log_retention_limitオプションを追加すると、その値までデータを保持するように修正

Kubernetes​

  • Change クラスタープロジェクトでヘッダーメニューをネームスペース移動時に、既存メニューを維持した状態で移動できるように修正
  • Change 性能サマリーのページウィジェットデータを修正

共通​

  • Change 照会および分析権限をオフできないように非活性化処理を修正
  • Fixed通知受信設定 - サードパーティーのSlackにredirect urlが正常に接続されないバグを修正
  • Fixed 統合レポート メニューから初期入力時、時間選択の値が正常に初期化されないバグを修正
