イベント履歴

ホーム画面 > プロジェクト選択 > アラート通知 > イベント履歴

警告通知の履歴を確認できます。過去1年間の履歴を表示できます。各項目を設定し、ボタンをクリックします。

ノート

選択したプロジェクトによって、画面イメージが異なる場合があります。

  • 時間選択

    • 右側の緑色のボタンを選択して表示時間を選択することができます。

    • または ボタンを選択すると、選択した表示時間だけ間隔を移動します。

    • 詳細な時間を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、適用ボタンを選択します。

  • フィルター: 題名またはメッセージ内容に基づいてイベント履歴をフィルタリングできます。

  • エージェント：プロジェクトに含まれるエージェントを選択できます。

  • CSV：参照したイベント履歴結果をcsvファイルとして保存することができます。最大CSV行数を入力し ダウンロード ボタンを選択してください。

  • カラム選択：取得されたイベント履歴の結果の列項目を追加することができます。

  • アラート設定: アラート通知 > アラート設定メニューに移動します。

  • 題名 / メッセージ: アラート設定メニューで追加したイベントのイベント名とメッセージ項目の内容です。

  • アラート発生時刻: イベントが発生した時刻です。

    • イベントが解消されずに進行中である場合、進行中タグが表示されます。

    • 整備中である場合、イベントが発生するとメンテナンス中に発生タグが表示されます。

    ノート

    メンテナンス計画メニューの詳細については、次の文書を参考にしてください。

  • イベント解消時刻: 設定したイベントが解決された時刻です。もし該当カラムが見えなかったらカラム選択ボタンをクリックしイベント解消時刻を選択してください。

  • アプリケーション: アラート設定メニューでイベントを追加してアラート通知対象のフィルタリング項目を設定すると表示されます。
