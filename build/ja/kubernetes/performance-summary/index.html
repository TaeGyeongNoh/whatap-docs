性能サマリー

ホーム画面 > プロジェクトを選択 > ダッシュボード > 性能サマリー

最近の時間別チャートを通じてコンテナ/Pod数量、ノード数量などKubernetesの主要指標にどのような変化があったのかを確認することができます。次の画像は、クラスタープロジェクト(Kubernetes) の例です。

  • ショートカットオプション

    ウィジェットの右上に位置する詳細アイコンを選択すると、次のようにショートカットオプションを確認できます。

  • ラインチャート設定オプション

    ラインチャートウィジェット別右上に位置する 詳細を見るアイコンを選択すると設定オプションを確認することができます。ラインチャートの場合、設定オプションを通じてidとなる値を変更することができます。この設定は保存されません。

    例、Node CPU (%)ウィジェットの場合Nodeごとのデータ照会が可能です。メトリクスを変更してContainerNameごとに照会できます。

Kubernetesクラスタ統合パフォーマンス​

クラスタープロジェクト(Kubernetes)の場合、 Kubernetesクラスター統合パフォーマンス領域で、WhaTapノード エージェントのインストールに関係なく、クラスター全体の情報を表示します。ネームスペースプロジェクトで分離された情報も統合して表示します。

Kubernetesネームスペースパフォーマンス​

ネームスペースプロジェクト(Namespace)の場合、 Kubernetesネームスペースパフォーマンス領域で、WhaTapノードエージェントをインストールに関係なく、ネームスペース情報を表示します。ネームスペースに関連付けられたコンテナ、Pod、ノードの情報を表示します。

プロジェクトパフォーマンス​

プロジェクトパフォーマンス領域は、プロジェクト内のWhaTapノード エージェントおよびアプリケーションエージェントがインストールされている対象の情報を表示します。ネームスペースプロジェクトに分離された情報は、ネームスペースプロジェクトで確認することができます。

ノート

エージェントのインストールが正常に完了していない場合、データが表示されない可能性があります。

ノード情報​

ウィジェットを提供するNode情報は次のとおりです。

  • 直前30分チャート

    • Node CPU (%)、Node Memory (%)など
ノート

  • ノード数量ウィジェットの右上にその他のアイコン>ショートカットアイコンを順番に選択するとクラスター>ノード一覧メニューに移動します。

  • Node CPU (%)ウィジェットとNode Memory (%)ウィジェット右上の 詳細を見るアイコン > ショートカットアイコンを順番に選択すると分析 > メトリクスチャートメニューに移動します。該当メニューを使用すると、より詳細な条件でチャートを確認できます。

コンテナ情報​

ウィジェットによって提供するコンテナ情報は次のとおりです。

  • 直前10分チャート

    • Container Status、Container CPU (%)、Container Memory (%)など

アプリケーション情報​

ウィジェットによって提供するアプリケーション情報は次のとおりです。

  • 直前10分チャート

    • Active TX、TPS、Response Timeなど
ノート

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在する場合に表示します。
