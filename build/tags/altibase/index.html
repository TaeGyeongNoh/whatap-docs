Flex 보드 관리하기

Flex 보드에서 대시보드를 편집하고 관리하는 방법을 안내합니다. 대시보드의 이름 변경, 프로젝트 선택, 레이아웃 조정 등을 포함하며, 위젯의 추가, 이동, 크기 조절, 삭제 방법도 확인할 수 있습니다. 또한, 위젯의 데이터 소스를 선택하는 방법과 대시보드의 json 형식으로의 내보내기 기능도 제공합니다.
