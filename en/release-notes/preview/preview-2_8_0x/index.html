Preview 2.8.0.a

Note

In preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview 2.8.0.a3​

August 8, 2024

Common​

Fixed Fixed the issue where themes applied to some text elements are missing.

Preview 2.8.0.a2​

August 8, 2024

Common​

  • Feature Added the feature to download the member list as a CSV file from the following menu paths:

    • Home > Select Project > Management > Project Member
    • Home > Management > Integrated Member Management
    • Home > Organization Management > Member List

  • Changed Changed the location and UI of the Integrated Group Management button.

  • Fixed In the Rich text widget in Integrated Flex Board or Flex Board, fixed the issue where the unsaved content is retained when opening and closing the Edit screen.

  • Fixed On the Edit screen of the Rich text widget in Integrated Flex Board or Flex Board, set the theme for the tooltip element so that the text and the tooltip of the toolbar button do not overlap.

  • Fixed In the side menu, fixed the scroll error of theOrganization list depending on the web browser screen size.

Application​

  • Changed In Dashboard > Application Dashboard, changed the widget name of Active Transaction Equalizer to Active Transaction, and modified to display the widget name on one line without line breaking.

  • Fixed Modified the substituted variable # in the SQL statement for transactions not to be displayed as $.

Server​

Fixed In Server CONF. > Runtime ENV. Variable or EVN. Variable, fixed the issue where the menu name of the top of the screen is changed to Agent Log Viewer, if you select the agent list expansion button.

Database​

Oracle V2, Oracle Pro​

  • Changed In Dashboard > Monitoring Multiple Instances, modified to view the data in the Main Waits widget in the active_session category.

  • Fixed Fixed the error where the holder type and waiter type values ​​are not displayed in the following menu paths:

    • Lock tree table in Dashboard > Monitoring a Database Instance

    • Analysis > Lock Tree

Oracle V2​

Fixed Fixed the error where the holder and waiter of the same sid are displayed as Lock in the following menu paths:

  • Lock tree table in Dashboard > Monitoring a Database Instance

  • Analysis > Lock Tree

Browser​

  • Changed Improved the UI of the Dashboard > Browser Error > Error message Table section.

  • Changed Improved the UI of the Dashboard > Browser Error > Error message Table > Error Detail > Errors per page section.

Network​

Changed If the user language is English, modified the Version to Kernel Version in the Precautions before installation section of Management > Agent Installation.

Log​

  • Changed In Log > Log Configuration > Getting started with log monitoring, modified not to display the items with # among Data retention per category.

  • Fixed Fixed the issue where the original log text is not displayed when proceeding with Column Settings in the following menu paths:

    • Log > Live Tail
    • Log > Log Trend
    • Log > Log Search
    • Trace analysis > Transaction log

Preview 2.8.0.a1​

August 1, 2024

Common​

  • Feature InIntegrated Flex Board 또는 Flex Board, added the OR option between the conditions within the metrics type filters.

  • Feature In Integrated Flex Board 또는 Flex Board, added the font color selection option for rich text widgets.

  • Changed Changed some colors for themes.

  • Changed In Management > Agent Installation, removed the caution regarding the access key.

  • Fixed Fixed the issue where the bottom area is hidden when accessing the WhaTap service screen through the browser on a mobile device.

  • Fixed Fixed the issue where the description of the metric is displayed as the widget title when widgets are added using All metrics in Integrated Flex Board or Flex Board.

  • Fixed Fixed the issue where the widgets included in a group widget on the dashboard shared as read-only in Integrated Flex Board cannot call data.

  • Fixed Fixed the issue where the dashboard filters are not applied in the sub widgetof the group when applying the filters if there is no filter selected in the group widget in Integrated Flex Board.

  • Fixed Fixed the issure where the setting is not saved when the same set value is entered if only one of Warning and Critical metrics is activated in the event addition or modification screen of Alert > Event Configuration V2.

  • Fixed Fixed the issue where the MXQL data is not viewed on the composite metrics addition or modification screen of Alert > Event Configuration V2.

    We fixed the issue where the MXQL data is not viewed when changed to the text mode after setting the tags or fields with the string values by using the Filter option in the Query event data section.

Application​

  • Changed Changed the filtering selection box in the Trace analysis pop-up window to the group format button.

  • Changed If the user language is set to English, changed the font size to improve the readability of the Apdex widget guide modal in Dashboard > Application Dashboard.

Server​

Fixed When adding or modifying events in Alert > Event Configuration, changed the Event rule condition's my font to the default font.

Kubernetes​

  • Changed In Workload > Container Application List, moved the ApplicationName, ApplicationId column to under the Application information column.

  • Fixed In Workload > Container Application List, fixed the bug where warning messages are not displayed when there is no mapped container.

  • Fixed In Container summary analysis of Dashboard > Container Map, fixed the image resource information error.

Database​

Common​

  • Feature Added the feature to log user actions when the Session kill feature succeeds or fails.

  • Changed Modified not to invite the dbx@whatap.io account together when using the Invite the WhaTap support team feature in Management > Project Member.

PostgreSQL​

Feature Added the Vacuum Sessions widget that can check the number of vacuum processes running in Dashboard > Monitoring Multiple Instances.

MySQL​

Fixed In Analysis > Dead Lock, fixed the issue where the message displayed when there is no data retrieved, is half displayed.

Oracle V2, Oracle Pro​

  • Feature Added the FILTER_PREDICATES column in the Runtime Plan and Explain Plan tables of the SQL details pop-up window.

  • Fixed Fixed the issue where the OBJECT_NAME column link is not enabled when you enter the SQL details pop-up window from Stat/Report > SQL Statistics.

  • Fixed In Stat/Report > SQL Statistics, fixed the issue where the elapse time value in the detailed chart is abnormally high.

  • Fixed Modified to display the cluster name in Dashboard > Instance List.

Redis V1, MongoDB V1​

Fixed Modified not to display the cluster name in Dashboard > Instance List.

Common​

  • Changed Removed the label display for individual graphs from the vertical equalizer bar chart.

  • Fixed In Analysis > Metrics Anomaly Detection, fixed to prevent multiple items from being selected for a category when applying the Filter option.

  • Fixed In Integrated Flex Board 또는 Flex Board, fixed the issue where the Edit button appears on the widgets whose titles cannot be modified.

  • Fixed In the Widget templates list of Integrated Flex Board or Flex Board, fixed the issue where the expand/collapse icon does not change when there is no widget list to display under the product category.

  • Fixed Fixed the Japanese notation of the product introduction displayed when creating a new project.

Kubernetes​

Cloud​

Fixed Fixed the issue where the installation is not made due to the indentation by the script in the Management > Agent Installation menu for the Amazon ECS platform.

Browser​

Fixed In User session analysis > User session flow, fixed the problem with the sorting for elapsed times.
