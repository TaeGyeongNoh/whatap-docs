Release Notes - Preview>Preview 2.0.0.a

Preview 2.0.0.a

https://preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.0.0.a0​

2023년 11월 29일

공통​

My usage​

  • Changed 결제 정보를 등록하지 않더라도 프로모션 배너를 노출하고, 결제 정보를 등록하지 않은 상태에서 프로모션 코드를 입력할 수 없도록 변경
  • Feature Invoice Preview History 메뉴에서 사용량 수치를 클릭하면 사용량 상세 정보를 표시하는 기능 추가
  • Fixed Quota 메뉴에서 다국어가 적용되지 않은 용어 수정

Dashboard​

  • Fixed Integrated dashboard 메뉴에서 위젯의 툴팁이 사라지지 않는 현상 수정

Analysis​

  • Fixed Metrics chart 메뉴의 차트 위젯 상세 보기에서 Delete entire widget 버튼이 동작하지 않는 문제 수정
  • Fixed Metric anomaly detection 메뉴에서 Delete entire widget 버튼이 동작하지 않는 문제 수정

Stat/Report​

Fixed Report 메뉴에서 Guide 버튼 클릭 시, 설정된 언어에 맞춰 가이드(Docs) 링크로 이동하도록 수정

User Interface​

  • Fixed 서비스 화면에서 다크 테마가 적용되지 않은 요소를 모두 적용 완료했습니다. 다크 테마는 서비스 화면의 오른쪽 위에 버튼을 선택한 다음 Dark를 클릭하세요.
  • Changed 채팅 문의 화면에서 1:1 online training 텍스트를 Apply for WhaTap user training으로 변경

Application​

  • Fixed 구버전의 Application dashboard 메뉴에서 위젯 정보 툴팁의 콘텐츠가 출력되지 않는 문제 수정
  • Feature Trace analysis 창의 테이블에서 Error class 컬럼 추가
  • Changed Application dashboard 메뉴의 HITMAP 위젯에 HITMAP transaction 메뉴로 이동하는 버튼 추가
  • Feature Application dashboard 메뉴의 각 위젯에 툴팁 및 관련 정보, 다국어 지원 추가

Server​

  • Fixed Log > Log setting 메뉴에서 에이전트 설정 및 실행 스크립트 내용을 정상 작동하도록 수정
  • Changed Log > Log setting 메뉴에서 에이전트 설정 및 실행 스크립트의 encoding 항목을 설정된 언어에 따라 다르게 표시하도록 수정
  • Changed Alert > Event configuration 메뉴에서 메트릭스 이벤트 추가 및 수정 시 이벤트 발생 조건 및 대상 설정 관련 주의 문구 추가
  • Fixed Server list > Server Detail 메뉴의 상세 보기 화면에서 이름이 길어 생략(...) 처리된 경우 마우스 오버 시 툴팁이 표시되도록 수정

Database​

공통​

Feature Stat/Report > SQL statistics 메뉴에서 쿼리(query)에 마우스 오버 시 쿼리 미리보기 기능 추가

PosgreSQL V2​

  • Feature Analysis > Wait analysis 메뉴에서 에이전트 설정 방법 안내 모달 추가
  • Feature Analysis > Wait analysis 메뉴에서 기능 소개 모달 추가
  • Fixed Analysis > Wait analysis 메뉴에서 데이터가 출력되지 않는 문제 수정
  • Feature Stat/Report > PG SQL statistics 메뉴에서 쿼리(query)에 마우스 오버 시 쿼리 미리보기 기능 추가

MySQL V2​

Feature Stat/Report > MYSQL SQL STATISTICS 메뉴에서 쿼리(query)에 마우스 오버 시 쿼리 미리보기 기능 추가

Kubernetes​

  • Feature Resource 메뉴 하위로 다음의 메뉴 추가
    • Pod List: Pod 목록 및 상세 정보 제공
    • Application List: 컨테이너화된 애플리케이션 목록 및 상세 정보 제공
    • Container Images: 사용중인 이미지 목록 제공
  • Feature Management 메뉴 하위로 Correlated Project Management 메뉴 추가: 연계 프로젝트 등록, 조회, 삭제 기능
  • Feature 연계 등록된 프로젝트가 있을 경우 Trace analysis > Table view에서 SQL 대상 서버 정보 및 HTTPURLConnection 정보에 대해 Correlations 뷰 제공
  • Deprecated Metrics chart 메뉴에서 메트릭스 차트 '컨테이너' 카테고리 삭제
  • Changed Dashboard > Container Map 메뉴의 요약 보기 화면에서 현재 로그(Stdout) 조회 시 컨테이너 id를 컨테이너 이름으로 표시하도록 변경
  • Fixed Dashboard > Container Map 메뉴에서 요약 보기 창의 타이틀 버그 수정
  • Fixed Resource > Performance summary 메뉴에서 사용자의 기기에 따라 위젯 안의 콘텐츠가 제대로 표시되지 않는 문제 수정
  • Fixed Alert > Event configuration 메뉴에서 메트릭스 이벤트 추가 창의 Go to the guide 버튼에 설정된 링크 수정
  • Changed Resource > Node list 메뉴에서 컬럼 분류 및 위치 변경
    • CapacityCpu 컬럼을 Node Info에서 CPU 하위로 이동
    • CapacityMemory 컬럼을 Node Info에서 Memory 하위로 이동

Log​

  • Feature Log search, Live Tail 메뉴에서 로그 검색 필터 기능 개편
    • 검색값 입력 시 지원하는 검색 문법 안내 가이드 기능 추가
    • 추가된 태그 수정 시 키보드 입력으로 수정할 수 있는 기능 추가
    • 태그를 키보드 입력으로 생성 및 수정 가능하고, 연산자는 클릭하여 수정할 수 있는 기능 추가

Cloud​

Fixed Management > Agent installation 메뉴에서 Pricing 탭의 Price policy 버튼의 링크를 설정된 언어에 맞게 이동하도록 수정

  • Amazon CloudWatch, Azure Monitor, Oracle Cloud Monitor 대상

Network​

Fixed Analysis > TCP Session, UDP Session 메뉴에서 필터 검색이 실패되는 문제 수정
