Preview 1.97.X

In https://preview.whatap.io, the following new features and enhancements can be experienced in advance.

Preview-1.97.14​

March 27, 2023

Common​

Fixed Fixed the problem where CSV is not downloaded after MXQL data query.

Preview-1.97.12​

March 24, 2023

Common​

  • Fixed In Alert > Event Setting, added the description of the Event pause item upon addition of log events.
  • Change Rolled back the agent search performance improvements when using the search function on the left sidebar of the screen.

Server​

Fixed Fixed the bug where in case of copying an alert from the Alert menu, some projects that can be selected in Copy Policy are missing.

Preview-1.97.11​

March 24, 2023

Database​

Fixed In Alert > File Log Alert, fixed the problem where previous logs remain when searching for a time zone where no log occurred.

Preview-1.97.10​

March 23, 2023

Common​

My Usage​

In Project Subscription, added and modified the following functions.

  • Change Improved UIs for user convenience.
  • Feature Added the function to select multiple projects and convert them to subscription.
  • Change In the Monthly Estimation column, modified to see the detailed information for each product.

Management​

Change Through Manage Account INFO. > Withdrawal, added the confirmation steps upon membership withdrawal such as updating notices and the related information (unpaid amount, granted roles).

Container​

Fixed In Dashboard > Container Map, fixed the problem where the options are not reset when Reset is clicked.

AWS, Azure, NCP, OCI​

  • Change Modified the functions and UIs to set the collection cycles for each service in the project.
  • Feature Added the filter to display only the active services.
  • Feature Added the search bar for service names.

Preview-1.97.9​

March 22, 2023

Common​

  • Fixed Improved the agent search performance when using the search function on the left sidebar of the screen.
  • Change In Log > Log Trend, improved to release the previously selected filters if there is no result for the filter when the search time has been changed.
  • Deprecate The Integrated Dashboard and Integrated Metrics Board menus have been removed from the left side menu after May 1, 2023, and the notice to move it to Sitemap has been displayed.

Preview-1.97.8​

March 20, 2023

Container​

Fixed In Dashboard > Container Map, fixed the problem where partial matched containers are not displayed when Include is selected as the Filter option.

Preview-1.97.7​

March 17, 2023

Common​

MXQL Data​

Fixed The Download CSV function does not work, so the download button will be removed and redistributed.

Analytics​

  • Fixed Fixed the problem where the stack information does not appear normally.

Preview-1.97.6​

March 14, 2023

Fixed Modified the MXQL query generation logic to fix the bugs that occur when calling the MXQL data while special characters are used in the field names for MXQL queries.

Preview-1.97.5​

March 14, 2023

Fixed In Management > Maintenance Plan, fixed the problem where a whiteout occurs when the Monthly tab is clicked.

Preview-1.97.4​

March 14, 2023

Common​

Feature In Management > Project Management, added the alert language selection option.

Server​

  • Feature In Server List > Server Detail, added Count Descending as an aligning option in the process table.
  • Feature In Alert > Notification Setting, added the escalation option in the 3rd Party Plugin configuration.
    • Translation of Escalation has been applied in the Korean website.
    • Modified for validation after changing options.
    • Modified to change the color of the button if you do not save the options after modification.

Preview-1.97.2​

March 13, 2023

Fixed In Alert > Notification Setting, fixed the bug where the number of registered mobile devices is not displayed.

Preview-1.97.1​

March 9, 2023

  • Fixed In Log Setting of a server project, modified the agent setting options.
  • Fixed In Channel Talk > Invite WhaTap Support, fixed the bug where the invitation function does not work.

    The WhaTap Support invitation button is exposed to only the users with the Member Management role.

  • Fixed In Dashboard > Resource Board of a server project, fixed the problem where the Y-axis is displayed long if the size of the Top N equalizer chart screen is large.

Preview-1.97.0​

March 8, 2023

Common​

User Interface (UI)​

To increase the user satisfaction, we improved the user interface design as follows.

  • Change In My Usage > Preview Invoice, improved the UIs.
    • Provision of invoice previews grouped by project and product
    • It is possible to check the related products of the grouped data, total billing amount, and total invoice amount.
  • Change Changed the `menu path` at the top of the screen to `menu name`.
    • Before change: Top Path > Menu Name
    • After change: Menu Name
  • Change Added a WhaTap technical document link on the upper right of the screen and removed the technical document link in Channel Talk.
  • Fixed Repositioned the browser time discrepancy notification icon next to the service logo.
  • Fixed Fixed not to go to the project generation page when a group with no project is selected.

Flex Board​

  • Change Applied the loading indicator when loading the dashboard and list on the screen.
  • Feature Added a legend display in the settings of the chart of Series widget type.
  • Feature Added a stacked chart type in the series widget types.
  • Feature Added the `chart color by` option in the settings of the chart of Series widget type.
    • Previous: Display of chart colors based on the tag
    • Addition: Option to color charts by field
  • Feature Added the following widget templates in DB suite products.
    • DB commons
      • CPU % (xos) widget
      • Memory % (xos) widget
    • MySQL
      • Slow Query
    • PostgreSQL
      • Slow Query
      • PostgreSQL Wait Event Cout widget
      • PostgreSQL Client Ip Count widget
      • PostgreSQL Statistics table
      • PostgreSQL Active Session Count
      • PostgreSQL Active Session table

Log​

  • Change Changed the menu name for quick understanding of the Log menu's feature.
    • 라이브 Tail → Live Tail
    • Explorer → Log Trend
    • Search → Log Search
  • Change Changed the function to sort the searched contents by Timestamp in Log Search (Recent order/Past order).
  • Feature Modified to select up to 2 search keys in Log Trend (the search value is not limited).
  • Change In Log Setting of a server project, modified the agent setting options.

Alert​

  • Feature In Event Setting, added the Agent duplicate events option.

    This alert occurs while collecting metrics redundantly when there are multiple agents with the same ID set among the agents being monitored. Check the related options in the agent settings to make sure there are no duplicate agent names. It is recommended to use this function when an alert occurs due to duplicate agents in deployment such as blue and green. If multiple agents with the same name are launched at the same time, you are not notified for the selected time.

  • Feature In Notification Setting, added the Receive notifications with no receive tag set option.

    Not to receive alerts for which the Event Receiving Tag has not been set, select the option.

Management​

  • Feature Added the Log Read role.
  • Fixed Fixed the bug where the Member Management role cannot be turned off in the member invitation window.

has been changed to product catalog​

  • Feature Added the group search function in Project Group for addition of a project.
  • Change Modified to select a project group when adding a project with an organization selected.

Server​

  • Feature In Alert > Event History, added the option to show the last change time.
  • Change The Server Setting > Remote File Viewer menu name has been changed to Agent Log Viewer.

Application​

Application Dashboard​

  • Fixed Fixed the problem where data call is stopped after Yard stop in the hitmap widget.
  • Fixed Fixed the problem where disabling all agents in the active transaction widget does not update any chart.

STAT/REPORT​

Change In Statistics, changed the lookup range from the maximum of 62 days to the maximum of 31 days.

Container​

Change Improved the UI and content in Management > Agent Installation.

Database​

Database commons​

  • Feature In STAT/REPORT > SQL Statistics, added the following table functions.
    • Can resize the table columns.
    • Modified to save the setting so that the adjusted column size is not released even if you leave the page.
    • Selecting each column in the table header can sort the list based on the selection criteria.
  • Fixed In Management > Agent Installation, modified the location of the guide.

PostgreSQL, MySQL​

Feature In STAT/REPORT > Database Size, a new UI has been applied.

Tibero​

Change In STAT/REPORT > Tablespace Size, changed the color of the progress bar based on the tablespace usage.

AWS​

Fixed Fixed the bug where selecting Event History for an AWS project goes to an unrelated snapshot page.

RUM​

Feature Added a menu: Analysis > Error Trace.

You can collect and check errors that occur at the web application browser level.
