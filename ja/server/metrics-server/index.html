サーバーメトリクス

サーバーモニタリングメトリクスについて次のように案内します。

agent_status_summary​

エージェントの状態に関するメトリクスを10秒間隔で収集したカテゴリです。

Fields​

フィールド名単位説明備考
inActTimeミリ秒(ms)エージェントが無効になったままの時間-
isActive-現在のエージェントの活性化状態true / false
isRebooted-サーバーの再起動可否true / false
lastActTimeミリ秒(ms)最後にエージェントが活性化された状態の時刻0: 無効化された場合
oid-プロジェクトに含まれる各エージェントの固有の識別子-
startTimeミリ秒(ms)エージェントが開始された時点でのタイムスタンプ-
status-現在のエージェントの状態-
uptime秒(sec)サーバーが稼働している時間-
ノート

statusフィールドの値は、次のように区切ります。

  • ok: サーバーが有効になっている正常な状態です。

  • inactive: サーバーが無効になっている状態です。

  • warning: サーバーに警告(warning) イベントが進行中の状態です。

  • fatal: サーバーに危険(critical) イベントが進行中の状態です。

  • paused: エージェントがデータの収集を一時停止した状態です。
