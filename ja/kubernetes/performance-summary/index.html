性能サマリー

ホーム画面 > プロジェクトを選択 > リソース > パフォーマンスの概要

最近の時間別チャートによって、コンテナ/Pod、Node数などのKubernetes主要な指標がどのように変更されたかを確認できます。

ウィジェットの右上に位置する詳細アイコンを選択すると、次のようにショートカットオプションを確認できます。

ノード数量ウィジェットの右上にその他のアイコン>ショートカットアイコンを順番に選択するとリソース>ノード一覧メニューに移動します。

ウィジェットを提供するNode情報は次のとおりです。

  • 直前30分チャート
  • Node CPU (%)、Node Memory (%)など

ノードCPUウィジェットとメモリウィジェットの右上にもっと見るアイコン>ショートカットアイコンを順番に選択すると分析>メトリクスチャートメニューに移動します。このメニューを使用して、より詳細な条件でチャートを確認できます。

ウィジェットによって提供するコンテナ情報は次のとおりです。

  • 直前10分チャート
  • Container Status、Container CPU (%)、Container Memory (%)など

ウィジェットによって提供するアプリケーション情報は次のとおりです。

  • 直前10分チャート
  • Active TX、TPS、Response Timeなど

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在する場合に表示します。
