ホーム画面(UI) の案内

WhaTapモニタリングサービスにログインすると表示される初期画面の要素が表示されます。 初期画面の基本メニューはプロジェクトリストです。

固定メニュー領域​

  • メニュー：左側には、ユーザーが入力したメニューパスが表示します。

  • お知らせ: WhaTapモニタリングサービスのお知らせを確認できます。

  • テーマ：現在のページのテーマを変更できます。 デフォルト値はLightモードです。

    • Light：基本的に適用されたテーマです。
    • Dark：黒い背景のテーマを適用します。
    • Presentation：ビームプロジェクトの使用時に最適化された色を適用します。
    • Add Custom Theme：ユーザーが自由に色の設定ができます。

  • WhaTap Docs

    WhaTap Docsの技術文書ページに移動します。 WhaTap Docsの詳細については、次の文書を参照してください。

  • 顧客サポート

    オンライン顧客サポートを受けることができます。 お問い合わせボタンを選択して技術および費用/決済などに関するお問い合わせができます。 すでに相談した内容はチャットで確認することができます。 オンラインでお問い合わせをするにはを選択してください。

    お問い合わせ内容の言語設定、お知らせ、広告受信などを設定するには、を選択してください。

  • プロファイル アイコン: 右端のアイコンを選択します。 アカウント情報管理、組織生成、組織管理を使用して、ユーザー言語を選択できる言語機能を提供します。 利用中のサービス画面からログアウトするにはログアウトボタンを選択してください。

プロジェクト検索領域​

  • プロジェクト/コード(テキストフィールド): プロジェクト名またはプロジェクトコードを入力して、目的のプロジェクトを検索します。

    プロジェクトコード(PCODE)は、プロジェクトの管理 > プロジェクト管理メニューのプロジェクトコード 項目で確認できます。

  • お気に入り：お気に入りに設定したプロジェクトが一覧として表示されます。 プロジェクトをお気に入りに設定するには、プロジェクトリストに表示されたプロジェクトカードビューの左上にあるを選択してください。 お気に入り設定を解除するには、もう一度 を選択してください。

  • ワイドカード：プロジェクトリストをカード形式で確認できます。 プロジェクトの基本情報をまとめて表示します。

  • コンパクトカード：プロジェクトリストをワイドカードより縮小されたカード形式で一覧を確認できます。

  • リスト：プロジェクトリストを製品カタログ別に分類し、表形式で確認できます。

  • 15 / page：ページあたりの一覧に表示されるプロジェクト数を設定できます。 1ページに最大100個まで表示されるように設定できます。 別のページに移動する場合は、ページ番号を選択してください。

組織およびグループ、プロジェクト領域​

  • 非組織：組織を選択するか、組織管理メニューに移動できます。 組織管理の詳細については、次の文書を参照してください。

  • + Group：グループを新しく追加できます。 グループ追加の詳細については、次の文書を参照してください。

  • + Project：プロジェクトを追加できます。 プロジェクト追加の詳細については、次の文書を参照してください。

  • 検索：グループまたはプロジェクト、エージェント単位で検索できます。 プロジェクト検索の詳細については、次の文書を参照してください。

  • プロジェクトタイプフィルター：フィルターを設定して、製品カテゴリ別にプロジェクトリストを確認できます。 設定された値に従って、全プロジェクトの下に一覧を表示します。

  • またはボタンを選択すると、プロジェクトリストの領域を非表示または表示できます。

  • 全プロジェクト：全体プロジェクトの一覧を確認できます。 特定のプロジェクトを選択して、このメニューを選択すると、初期のプロジェクトリストの画面に移動できます。

  • 領域の下には、プロジェクトリストが確認できます。 グループはの形で表示します。 グループ外のプロジェクトは、製品の種類別アイコンとして表示します。

統合メニュー領域​

  • メニュー: マウスをドラッグすると、メニュー領域を上下に動かしてサイズを変更することができます。
  • サイトマップ：ユーザーが入力したメニューパスで選択できるすべてのメニュー一覧を表示して移動します。

    入力したメニューのパスまたはプロジェクトによって、サイトマップウィンドウで選択できるメニューは異なる場合があります。

  • プロジェクトリスト：全体プロジェクトの一覧を確認できます。
  • Flexボード：カスタマイズ可能な統合ダッシュボード。 アプリケーション、サーバー、データベース、コンテナなどのプロジェクトから収集したメトリクスでリアルタイムのダッシュボードを作成できます。
    製品別文書
    • アプリケーションモニタリング > Flexボード
    • サーバーモニタリング > Flexボード
    • データベースモニタリング > Flexボード
    • Kubernetesモニタリング > Flexボード
    • Amazon ECS > Flexボード
    • Amazon CloudWatch > Flexボード
    • Azure Monitor > Flexボード
    • Naver Cloud Monitoring > Flexボード
    • Oracle Cloud Monitor > Flexボード
  • 統合レポート：複数のプロジェクトのレポートを簡単に作成できます。 詳細については、次の文書を参照してください。
  • 統合トポロジー：トポロジー機能に対応する複数のプロジェクトを一括選択し、単一トポロジーとして確認することができます。 詳細については、次の文書を参照してください。
  • リアルタイムアラートリスト：ユーザー権限のあるすべてのプロジェクトの通知をリアルタイムで確認し、通知状況を一目で確認することができます。 詳細については、次の文書を参照してください。
  • 管理：全体グループとメンバーを管理できます。 詳細については、次の文書を参照してください。

プロジェクト検索​

  1. 画面の左上にあるボタンをクリックすると、プロジェクト検索ウィンドウが表示されます。

  2. 検索するプロジェクトまたはグループ名を入力すると、一致するプロジェクトの一覧が検索ウィンドウの下に表示されます。

  3. 目的のプロジェクトを選択します。

適切なプロジェクト画面に移動します。

検索ウィンドウでエージェントを基準にプロジェクトを検索するには、グループ/プロジェクトをクリックし、ドロップダウン一覧からAgentを選択します。

プロジェクトリスト​

ユーザーがメンバーとして登録されているプロジェクトの一覧を確認できます。

下部のメニュー領域​

  • 利用内訳：決済情報、利用料金などに関するメニューをご利用いただけます。 詳細については、次の文書を参照してください。

  • : サイドメニューを非表示または表示できます。 サイドメニューが隠れた状態でボタンをクリックするとサイドメニューを表示します。

  • : WhaTapモニタリングサービス画面を全画面サイズでご利用いただけます。 全画面を終了するにはESCキーを選択してください。

サービス紹介メニュー​

サイドメニューの左端のアイコンボタンを選択すると、サービス紹介ページに移動できます。 サービス別カテゴリに分類されており、紹介ページに移動し、目的のサービスから始めるボタンを選択してください。 プロジェクト作成メニューに移動します。
