Preview 1.95.2

https://preview.whatap.ioで、次の新機能と変更した内容を事前に体験することができます。

Hotfix: 2023-02-16​

Application​

分析​

Fixed ヒートマップ メニューで、 TXトレース テーブルの 終了時間 のカラムが 開始時間 カラムより前に表示されるバグを修正

Hotfix: 2023-02-09​

Application​

分析​

Fixed マルチトランザクション追跡 > ツリービュー メニューからプロファイルの連携情報がツリー状の構造で表現されない不具合を修正

Official Release: 2023-02-08​

共通​

Flexボード​

Feature ホーム画面 > Flexボード メニューのダッシュボード一覧からダッシュボード共有() 機能を追加

分析​

  • Fixed サイトマップ > 分析 > メトリクス異常検出 メニューからカテゴリ、フィールドを選択時の UIバグによる UIロジックを変更
  • Fixed キューブ メニューからで検索が終わっていない場合 開始日, 対象メトリクス, のダッシュボードタブを変更できないように修正

トランザクション詳細Application​

  • Change SQL 分析タブの名称を SQL要約に変更
  • Change メソッド分析 タブの名称を メソッド要約に変更
  • Feature HTTP呼び出し概要タブの追加：HTTP呼び出しの呼び出し件数、合計時間、平均時間に関する情報を提供します。

管理​

  • 管理 > プロジェクトメンバー メニューから、次の機能を追加します。

    • Feature プロジェクトメンバーの管理性の向上のために、メンバーの所属(グループ/プロジェクト) タイプを表示
    • Feature メンバー削除() ボタンを無効にする場合、ツールチップ(削除できない理由)を追加

  • 使用履歴、決済管理 メニューで次の機能を修正します。

    • Change 請求書プレビュー / 使用量 メニューからログ課金の変更に伴う、ログ表示単位および式単位の変更
      • ログ表示単位の変更：LOGs ⇒ LOG UNITs
      • ログ表示の単位変更：合計のログ行数の日時平均表示に変更
      • 請求書プレビュー メニューから 課金集計量 カラムのログ使用状況の表示を変更
      • 使用量 メニューから使用状況の詳細のメータリング概要カラムを変更

Application​

アクティブトランザクション​

Change スレッドを停止するためにシークレットキーを入力するウィンドウに入力内容が表示がされないように修正

Server​

Fixed サーバーリスト メニューからサーバーステータスのアイコンをクリック時に、表示されるモーダル内容が選択したサーバーと一致しないバグを修正

Container​

コンテナマップ​

Feature Podカテゴリビューのグループ化の条件にDaemonSetとStatefulSet を追加
