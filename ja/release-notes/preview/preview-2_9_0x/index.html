Preview 2.9.0.a

ノート

preview.whatap.ioで、次の新機能と変更した内容を事前に体験できます。

Preview 2.9.0.a3​

2024年09月12日

共通​

  • Feature アカウント管理 > アカウント情報メニューでユーザーの登録された電話番号の削除機能を追加

  • Feature リアルタイムアラートメニューで解除されたイベントを見る機能のOn/Off設定を追加

  • Changed リアルタイムアラートメニューで次の事項を変更

    • イベント一括解消ボタンのアイコンを変更

    • リアルタイム通知設定オプションのUI改善

    • ブラウザ通知をしない機能のOn/Offトグルボタンの位置をリアルタイム通知設定の中に変更

  • Changed アラート通知 > イベント設定 Newメニューのイベント追加画面で一時停止オプションの説明中イベント状態が解消されたら追加に通知する用語を解消された通知に変更

  • Fixed 分析 > メトリクス照会メニューでタグ選択を解除した時に、可読性に欠ける問題をスタイル修正で改善

  • Fixed 分析 > メトリクス照会で、テーブル一覧にマウスをオーバーする時のテキスト表記スタイルの改善

Application​

共通​

  • FeatureApplicationプラットフォームで連携したDatabaseプラットフォームのインスタンス情報を確認できるDBプロジェクト連携機能を追加

    • 分析 > ヒットマップメニューでDB追跡機能をDBプロジェクト連携に変更

    • トレース分析画面のテーブルビュータブでSQLステップにDBプロジェクト連携機能を追加

    ノート

    対応するDatabase製品はPostgreSQL、Oracle、Oracle Pro、MySQL、SQL Server、CUBRID、Altibase、Redis、MongoDBです。

  • Changed トレース分析画面のレコード要約タブでテーブルのデザインとデータの分類方法を変更

  • ChangedDatabaseプラットホームでApplicationプラットフォームに接続する連携分析機能のテーブルデザインを変更

    ノート

    この変更事項はPostgreSQL、MySQL製品に該当します。

  • Fixed インスタンス性能管理 > スレッドリスト/ダンプメニューから画面の上にある状況要約でフルスレッド項目に多言語対応を適用

  • Fixed トレース分析画面でSQL詳細モーダル下部の置換変数、バインド変数のトグルボタンが一緒に表示される時、デザイン要素が正しく表示されない問題を修正

Java​

FeatureWhaTapモニタリングサービスにMySQL、PostgreSQL製品プロジェクトとリアルタイムでトランザクションを連携・分析できる 連携分析機能を追加

Server​

  • Newインフラサーバー運用に必要な情報をエージェントが自動収集および保存、ユーザー定義フィールドを通じて追加した情報を照会し管理できるサーバー インベントリメニューを追加

    ノート

    • この機能は、サーバーリスト > サーバー インベントリメニューで確認できます。

    • この機能はServer Linux/Windowsエージェントv2.6.6以降、AIXエージェントv1.3.4以降、Solarisエージェントv1.3.4以降から対応します。

  • Feature HP-UX OS環境をサポートするエージェントを提供

  • Changed リアルタイムアラートで通知項目を選択すると、イベント履歴メニューに移動するように変更

Kubernetes​

  • Feature 管理 > エージェントインストールメニューにエージェントバージョンを選択できるオプションを追加

  • Changed ワークロード > Pending Pod状況メニューでノードリソースがPending原因と推定される場合に提供するメトリクスを拡大するように修正

  • Changed 管理 > エージェントインストールメニューで設置方式の基本選択値をYAMLに変更

  • Fixed アラート通知 > イベント設定メニューでメトリックスイベントを追加する時に提供する割当て不能ノードのアラート通知テンプレートの誤字を修正

Database​

共通​

  • Feature ダッシューボード > マルチインスタンスメニューでユーザー定義ウィジェットに次の機能を追加

    • パイチャートまたはテキストチャートにマージなしオプションを追加

    • マルチインスタンスメニューに入ると、ユーザー定義ウィジェットに関する案内メッセージウィンドウを表示

  • FeatureSQLスクリプトを実行して結果を確認できるスクリプト·マネージャーメニューを追加

    ノート

    この機能は、実験室 > スクリプト·マネージャーメニューで確認できます。

  • Changed 連携分析ボタンの位置をセッション詳細画面からアクティブセッションテーブルに移動

    ノート

    この変更事項はPostgreSQLV2、MySQLV2、OracleV2、Oracle Pro製品に該当します。

  • Changed 連携分析ポップアップ画面のデザインを変更

    • 連携プロジェクトの画面がデフォルト表示されるように修正

    • 連携プロジェクトのプラットフォームアイコンを表示するように修正

  • Changedプロジェクト作成画面でCUBRID、Altibase項目に対応バージョンを追加

CUBRID​

New CUBRID V2サービスを開始

ノート

CUBRIDモニタリングV2の新機能についての詳細は、次の文書を参考にしてください。

PostgreSQL V2, MySQL V2​

FeatureApplicationプラットフォームとリアルタイムで連携分析できる機能を追加

MySQL​

Fixedユーザー言語が英語の場合、管理 > エージェントインストールメニューでインストール内容の誤字を修正

Oracle、Oracle Pro​

Changed 分析 > SQL分析メニューで画面上段のチャート表示形式を変更

Oracle Pro​

Fixed 管理 > エージェントインストールメニューでエージェントのインストール内容を修正

  • Oracleアカウントではない他のOSアカウントでインストールする方法の説明を追記

  • ZIPファイルのダウンロード機能を削除

Browser​

FeatureページのURLパスごとのロード性能情報に関する統計を提供するページロード統計メニューを追加

ノート

この機能は、統計 > ページロード統計メニューで確認できます。

Log​

  • Feature ログ > ログTailおよびログ検索メニューでフィルターの入力画面にカーソルが位置している時、ECSキーを押すとポップオーバーが閉じられるように改善

  • Fixed ログ > ログTailおよびログ検索メニューでログの検索タグフィルター入力時に入力画面に残っている入力値を削除するように修正

  • Fixed ログ > ログ検索メニューでログフィルター情報を表示するツールチップのコンテンツが、正しく表示されない問題を修正

  • Fixed ログ > ログ設定メニューに進入する際、選択した組織のidがクエリパラメータで漏れる問題を修正

Preview 2.9.0.a2​

2024年09月09日

共通​

Featureログ関連メニューでテーブルカラムの幅を調整できる機能を追加

Kubernetes​

  • Newノード(Node)についての詳細情報が確認できるノードマップメニューを追加

    ノート

    この機能はダッシューボード > ノードマップメニューパスで確認でき、次の詳細を提供します。

    • ノード別リソース情報

    • 個別ノードの状態(CPU、Memory状態)

    • ノードに関する環境情報

    • ノード内部で実行中のWhaTapエージェントの状態

    • ノード関連のメトリクスカテゴリの正常収集の可否

  • New時間の流れによるノードの現状を確認できるノードタイムラインメニューを追加

    ノート

    この機能はクラスタープロジェクト専用であり、クラスター > ノードタイムラインメニューパスで確認できます。

  • Feature クラスター > ノードリストメニューの比較および詳細表示の機能を改善

    ディスク、ネットワークカラムグループを削除し、ノードを選択して比較および詳細ビューで関連情報を提供します。

  • Feature 管理 > Kubernetesエージェントリストメニューの詳細でメトリクスデータを確認できるメトリックスタブを追加

  • Feature クラスター> ノードディスクリストメニューで比較カラムを追加、比較カラム項目を選択したディスクの使用量推移が確認できる詳細画面を提供

  • Changedサイドメニューで分析 > メトリクスチャート Oldメニューを表示するように変更

  • Deprecated クラスター > ノードリストメニューから以前のバージョンで見る機能を削除

Log​

  • Changed ログ検索メニューの検索フィルター条件にcategoryだけ入れた場合のログ件数を全体{n}件の表示に表記

  • Fixed ログ設定メニューのログパーサ設定で'文字:' パターンのprefixが登録できない問題を修正

Preview 2.9.0.a1​

2024年09月05日

共通​

  • Feature 統合FlexボードまたはFlexボードメニューのビューモードでグループウィジェットレイアウトの閉じる機能を追加

  • Changed リアルタイムアラートメニューでのイベント発生時刻を月/日/時/分/秒まで詳細表示するように変更

  • Fixedリアルタイム通知メッセージの内容が長い場合、レイアウトが押し出される問題を修正

Application​

Fixed トポロジー >s APPトポロジーメニューでアプリケーションを表示する要素のアイコンが正常に表示されない問題を修正

Database​

Changed ダッシューボード > マルチインスタンスメニューで、Vacuum Sessionsウィジェットのボタンで開くVacuum履歴画面でtimeカラムの時間を照会された時間として表示するように変更

Preview 2.9.0.a0​

2024年08月29日

共通​

  • Feature 分析 > メトリクスチャート NewメニューのメトリックパネルにWhaTap Docsで接続するリンクボタンを追加

  • Changed 分析 > メトリクスチャート Newメニューで次の事項を修正

    • 対象選択オプションの内容が末尾省略された時、UIを改善

      • 更新のテキストを削除

      • すべてを開く/折りたたむのテキストの左側にアイコンを追加

    • ウィジェットがない時、一括変更およびダウンロードボタンを無効化

    • 対象選択オプションで検索結果に対する全体選択と解除機能を追加

    • 複数選択時に Shift キーを使用できます。フレーズをツールチップで表示するように修正

    • すべて選択ボタンツールチップをマウスオーバーすると表示するように修正

    • 時間選択オプションにマウスオーバーした時、表示されるツールチップを削除

  • Fixed 統合FlexボードまたはFlexボードメニューでウィジェットボタンのドロップダウンが不要に生成される問題を修正

  • Fixed グループへ移動画面で長いグループ名がモーダル画面から離れて表示される問題を修正

  • Fixed グループ削除画面でグループアイコンが'X'で表示される問題を修正

Application​

  • Feature インスタンス性能管理 > スレッドリスト/ダンプメニューで選択したエージェントがインストールされているアプリケーションのスレッド状況(RUNNABLE、WAITING、合計)を表示する機能を追加

  • Fixed トレース分析画面のTXログタブでログデータがない場合、案内メッセージを出力するように変更

    ノート

    ログ関連のエージェントオプションが設定されていない、またはログデータがない場合はメッセージを確認できます。

  • Fixed NetstatトポロジーメニューでGo商品アイコンが正常に表示されない問題を修正

  • Fixed トレース分析画面でSQL文の表示ボックスの色を修正

  • Fixed トレース分析画面でSQLの詳細を照会する時、APIが2回呼び出される問題を修正

    ノート

    API呼び出し回数の変更はユーザーに影響を与えません。

  • Fixed ダッシューボード > アプリケーションメニューでアクティブTXウィジェットのチャート領域を選択した時に表示される アクティブTX画面のデザイン要素を修正

    • トップ10タブでデータがない時に表示されるNo rows foundメッセージがテーブルヘッダーと重ならないように修正

    • 余白が生じた際、背景色が埋められない問題を修正

  • Fixed ダッシューボードメニューでヒットマップウィジェットのY軸の調整ボタンが断続的に見えない問題を修正

  • Fixed 分析 > スタックメニューのボタンでスナップショットを画像として保存する際にテーマが適用されない問題を修正

Database​

共通​

  • Fixed ダッシューボード > インスタンスリストメニューでインスタンス名を選択してインスタンスモニタリングメニューに移動する際に組織情報が漏れる問題を修正

  • Fixed 統計/レポート > SQL統計メニューで、SQLテキストプレビューのサイズを修正

  • Fixed SQL詳細画面でReset Bindボタンを選択するとバインド変数が初期化されない問題を修正

  • Fixed 管理 > エージェントインストール メニューで次の修正事項を反映

    • ZIPファイルのダウンロードコマンドの削除

    • TARファイルのダウンロードコマンドの修正

    • PostgreSQL、Oracle Pro、Altibase、MySQL商品以外の商品でZIP ダウンロードボタンが動作しない問題を修正

Oracle Pro​

Changed 分析 > ロックツリーメニューでクラスターを選択し、Global Lock情報を照会できる最大時間範囲を3時間に変更

Log​

Changed ログの下位メニューでカラム設定のUIを交替

  • 1つになったテーブル設定ボタンをカラム設定およびログ表示の詳細設定ボタンで分離

  • 全画面モードの時、カラム設定およびログ表示の詳細設定ボタンを取り消し
