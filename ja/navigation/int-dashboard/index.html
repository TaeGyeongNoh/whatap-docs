統合ダッシュボード

ホーム画面 > サイトマップ > 統合ダッシュボード

ユーザーが属するすべてのプロジェクトのデータを表示できるダッシュボードを作成できます。異なる製品群のステータスをダッシュボードの上にウィジェットとして構成し、一目で把握することができます。

統合ダッシュボードメニューを初めて利用する場合、Flexボードメニューの使用をお勧めします。Flexボードに関する詳細については、各サービス別の文書を参照してください。

商品別Flexボード文書
  • アプリケーションモニタリング > Flexボード
  • サーバーモニタリング > Flexボード
  • データベースモニタリング > Flexボード
  • Kubernetesモニタリング > Flexボード
  • Amazon ECS > Flexボード
  • Amazon CloudWatch > Flexボード
  • Azure Monitor > Flexボード
  • Naver Cloud Monitoring > Flexボード
  • Oracle Cloud Monitor > Flexボード

統合ダッシュボードを作成する​

テンプレートを使用してダッシュボードを作成する​

WhaTapが提供するテンプレートを使用してダッシュボードを簡単かつ迅速に作成できます。

  1. 画面の上のダッシュボードを作成するを選択してください。

  2. ダッシュボードを作成するメニューの下にテンプレート一覧が表示されます。

  3. テンプレートを選択すると、ダッシュボードテンプレートウィンドウが表示されます。

  4. ダッシュボード名と説明を入力し、データを読み込むプロジェクトを選択してください。選択できるプロジェクトの数はテンプレートによって異なる場合があります。

  5. 確認ボタンを選択してください。

統合ダッシュボード一覧で作成したダッシュボードを確認できます。一覧で新しく作成したダッシュボードを選択すると、ダッシュボード画面に移動します。

空のテンプレートでダッシュボードを作成する​

  1. 画面の上のダッシュボードを作成するを選択してください。

  2. ダッシュボードを作成するメニューの下にテンプレート一覧が表示されます。

  3. 左端の空のダッシュボード作成を選択してください。

  4. ダッシュボードを作成するウィンドウが表示すると、名前と説明を入力して保存ボタンを選択してください。

  5. 統合ダッシュボード一覧で作成したダッシュボードを確認できます。一覧で新しく作成したダッシュボードを選択すると、ダッシュボード画面に移動します。

  6. 画面上部の+ウィジェットの追加ボタンを選択して、ウィジェットを追加してください。

ウィジェットを追加する​

ダッシュボード画面にウィジェットを追加して、さまざまなプロジェクトの指標を確認できます。例えば、ユーザーがAプロジェクトのヒートマップチャートウィジェットとBプロジェクトのヒートマップチャートウィジェットをダッシュボードに追加して同時に確認することができます。

  1. 統合ダッシュボード一覧からウィジェットを追加するダッシュボードを選択し、ダッシュボード画面に移動してください。

  2. 画面の上の+ウィジェットの追加ボタンを選択してください。

  3. ウィジェット作成ウィンドウが表示されると、目的のウィジェット名とウィジェットチャートの種類、プロジェクトの順に選択してください。

    • 選択したウィジェットチャートの種類によって、選択できる製品カテゴリおよびプロジェクトが異なる場合があります。

    • 製品カテゴリでは、複数のプロジェクトを選択できます。他の製品カテゴリと組み合わせることはできません。

    • Custom chartは、メトリクスおよびオブジェクトマージ、期間フィルターなどのオプションを設定できます。

  4. 次へボタンを選択してください。

  5. 各プロジェクトに属するエージェントは、エージェントまたはグループID、Node ID を選択できます。プロジェクト項目の一番左の▶ボタンを選択してください。

    • ウィジェットで指標を表示するエージェントをクリックし、適用ボタンを選択してください。

    • すべてのエージェントの指標を表示するには、全体をクリックし、適用ボタンを選択してください。

  6. 完了ボタンを選択してください。

統合ダッシュボード画面で追加したウィジェットを確認してください。

  • ウィジェットの右下部分をカーソルに合わせてドラッグすると、サイズを調整できます。

  • ウィジェットのタイトル部分にカーソルを合わせてドラッグすると、位置を移動できます。

  • ウィジェットに関する情報を確認するには、ウィジェットの右上のボタンをクリックしてダブルクリック してください。

ウィジェットを修正する​

  1. ウィジェットの右上のボタンを選択してください。

  2. ウィジェットの作成ウィンドウが表示されたら、ウィジェットチャートの種類とプロジェクトを選択して変更できます。

  3. 次へボタンを選択してください。

  4. プロジェクトを使用して、別のエージェントを選択するには、プロジェクト項目の一番左の▶ボタンを選択してエージェントを選択してください。

  5. 完了ボタンを選択してください。

統合ダッシュボード画面で修正したウィジェットを確認してください。

ウィジェットを削除する​

統合ダッシュボード画面から削除するウィジェットの右上のボタンを選択してください。メッセージが表示されたら、確認ボタンを選択してください。

統合ダッシュボード名と説明を変更する​

  1. 統合ダッシュボード一覧から名前または説明を修正するダッシュボード項目のボタンを選択してください。

  2. ダッシュボード修正ウィンドウが表示すると、名前と説明を修正してください。

  3. 修正を完了して保存ボタンを選択してください。

統合ダッシュボード一覧で修正した内容を確認できます。

統合ダッシュボードを共有する​

統合ダッシュボードで作成したダッシュボードを他のアカウントのユーザーと共有して再利用できます。

コピーする​

ユーザーが作成したダッシュボードをコピーしたり、他のユーザーと共有したりできます。

統合ダッシュボード一覧でコピーしようとするダッシュボード項目のボタンを選択してください。 自分にコピーまたは他のユーザーにコピーボタンを選択してください。

  • 自分にコピーを選択すると、ダッシュボード一覧に「(Copied)」接頭語が追加された項目を確認できます。

  • 他のユーザーにコピーの場合、他のユーザーのメールアドレスを入力し、送信ボタンを選択してください。読み取り専用オプションを選択した場合、共有するユーザーはダッシュボードを修正できません。

    ダッシュボードの共有後、編集したダッシュボードの変更は共有されません。

書き出し、読み込み​

JSONファイルとして保存した統合ダッシュボード設定を、他のユーザーアカウントから読み込みしてコピーします。ダッシュボード管理者権限の機能です。

  1. 統合ダッシュボード一覧で他のユーザーと共有したいダッシュボード項目のボタンを選択してください。

  2. Export Dashboardウィンドウが表示されたら、書き出しボタンを選択してください。

  3. JSONファイルをダウンロードします。

  4. 他のユーザーにダウンロードしたファイルを転送してください。

  5. 統合ダッシュボード画面上の読み込みボタンを選択してください。

  6. Export Dashboardウィンドウが表示されると、保存ボタンを選択してください。

ダッシュボード一覧に「(Imported)」接頭語が追加された項目を確認できます。

統合ダッシュボードを削除する​

統合ダッシュボード一覧で削除しようとするダッシュボード項目のボタンを選択してください。ポップアップメッセージで削除ボタンを選択してください。
