グループ管理

プロジェクトをグループ化して管理できます。ユーザーは、複数のプロジェクトを所有できます。複数のプロジェクトを管理するためにグループという概念を提供します。2つ以上のプロジェクトがある場合、プロジェクトごとにユーザーを追加したり、権限を付与したりするのは面倒な場合があります。この場合、2つのプロジェクトを1つのグループにまとめ、グループとしてユーザーを管理できます。

ノート
  • 1つのプロジェクトは、1つのグループのみに属します。
  • プロジェクトは、グループに属していない可能性があります。

プロジェクトユーザーはグループ ユーザーを継承し、グループに属するユーザーはグループ内のすべてのプロジェクトをモニタリングできます。特定プロジェクトに限って権限が付与されたユーザーなら、グループの有無に関係なく、そのプロジェクトのみをモニタリングできます。

グループを追加する​

  1. サイドメニューでプロジェクト全体 > Groupボタンを選択します。

  2. グループ追加画面で作成するグループの名前をグループ名に入力します。

  3. グループに関する説明をグループ説明に入力してください。グループ説明は必須ではありません。

  4. プロジェクト追加一覧でグループに追加するプロジェクトを選択してください。プロジェクトを検索して選択することもできます。

  5. プロジェクトの追加が完了したら、グループ追加ボタンをクリックしてください。

ノート

グループに追加できるプロジェクトの一覧は、修正権限があるプロジェクトまたはグループに属していないプロジェクトです。

グループにプロジェクトを追加する​

  1. ホーム画面のプロジェクトリストのプロジェクト項目の右上にボタンを選択してください。

  2. グループに追加を選択します。

  3. プロジェクトを追加するグループを選択してください。

  4. グループに追加ボタンを選択します。

選択したグループのプロジェクトリストを表示する​

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルで照会を希望するグループの右側にボタンを選択します。

    選択したグループに含まれるプロジェクトのみリスト画面に表示されます。この状態でグループメンバーまたはグループ管理のメニューを利用できます。

  3. もう一度全体のプロジェクトリストを確認するには、サイドメニュー下段のプロジェクト一覧へ移動ボタンを選択します。

ノート

グループに属していないプロジェクトを確認するには、リストパネルを下にスクロールします。

グループからプロジェクトを除外する​

ホーム画面から除外する​

  1. ホーム画面のプロジェクトリストのプロジェクト項目の右上にボタンを選択してください。

  2. グループから削除を選択します。

  3. グループからプロジェクト削除ウィンドウが表示され、OKボタンをクリックしてください。

グループ管理から除外する​

グループに属しているプロジェクトを別のプロジェクトに移動または除外できます。

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルから削除するグループの右側にボタンを選択します。

  3. サイドメニューでグループ管理を選択します。

  4. グループ管理セクションから除外するプロジェクトを選択します。

  5. プロジェクト(N)の移動ボタンが表示されたら、ボタンを選択します。

  6. プロジェクトの移動画面が表示されたらグループ選択項目で移動するグループを選択またはその他のプロジェクトを選択します。

  7. 確認ボタンを選択します。

ノート

グループ選択項目でその他のプロジェクトを選択すると、そのプロジェクトは所属するグループから除外されます。

グループを削除する​

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルから削除するグループの右側にボタンを選択します。

  3. サイドメニューでグループ管理を選択します。

  4. グループ情報セクションでグループ削除を選択してください。

  5. グループ削除メッセージウィンドウの内容を確認した後、削除ボタンを選択してください。

注意
  • グループを削除すると、ユーザー一覧の情報も削除されます。
  • グループに含まれるプロジェクトは、グループ情報のみ削除します。
  • 削除した内容は復元できません。

グループメンバーを招待する​

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルからメンバーを招待するグループの右側にボタンを選択します。

  3. サイドメニューでグループメンバーを選択します。

  4. 画面右上のメンバー招待ボタンを選択します。

  5. グループ項目で下位プロジェクト単位を選択してメンバーを追加することができます。

  6. ユーザー権限項目で招待するユーザーの権限を設定した後、適用ボタンを選択してください。

  7. メール入力欄に招待するユーザーのメールアドレスを入力した後、追加ボタンを選択します。他のユーザーをさらに追加できます。

  8. 招待するユーザーをすべて追加して、完了ボタンを選択してください。

ノート

  • メンバー招待機能は、メンバー管理権限のあるユーザーのみが利用できます。 メンバー権限の詳細については、次の文書を参照してください メンバー権限の詳細については、次の文書を参照してください。

  • WhaTapモニタリングサービスに登録していないアカウントには、新規アカウントタグを表示し、会員登録メールを送信します。

  • 招待を完了する前にキャンセルする招待メンバーがいる場合は、招待メンバーリストでボタンを選択してください。

  • 形式が正しくないメールアドレスを入力する場合は、招待されません。

  • すでに招待されてるメンバーは追加で招待することはできません。上位の組織やグループのメンバーは自動的に追加されます。

プロジェクト別のメンバーを確認する​

  1. サイドメニューでプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルでメンバーリストを確認するグループの右側にボタンを選択します。

  3. サイドメニューでグループメンバーを選択します。

  4. グループに所属するプロジェクトを選択すると、プロジェクトに所属するメンバーを確認できます。

グループメンバーの権限を修正する​

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルで権限を修正するメンバーを含むグループの右側にボタンを選択します。

  3. サイドメニューでグループメンバーを選択します。

  4. グループメンバー一覧で権限を変更するメンバーの右側のボタンを選択してください。

  5. メンバー権限を修正する画面が表示されたら、メンバー権限とグループのプロジェクト毎ユーザー権限を設定します。

    • メンバー権限：グループとグループに属するプロジェクトで許可する権限を設定できます。
    • グループのプロジェクト毎ユーザー権限：選択したメンバーの権限をグループに属するプロジェクトごとに設定できます。

  6. すべての設定が完了したら、適用ボタンを選択してください。

ノート

メンバー権限の修正機能はメンバー管理権限のあるユーザーのみ利用できます。メンバー権限の詳細は次の文書を参照してください。

グループメンバーを削除する​

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルから削除するメンバーを含むグループの右側にボタンを選択します。

  3. サイドメニューでグループメンバーを選択します。

  4. グループメンバー一覧から削除するメンバーの右側のボタンを選択してください。

  5. メッセージウィンドウが表示されたら、削除ボタンを選択してください。

ノート

  • 下位プロジェクトからメンバーを削除する場合は、メッセージウィンドウでチェックボックスをクリックし、削除ボタンを選択してください。

  • メンバー管理権限がない場合、他のメンバーを削除できません。メンバー権限の詳詳細次の文書を参照してください。

メンバーリストをダウンロードする​

メンバーリストをCSV形式のファイルとしてダウンロードする場合、次のステップを実行します。

  1. サイドメニューのプロジェクト全体メニューを選択してグループとプロジェクトリストのパネルを開きます。

  2. パネルから削除するメンバーを含むグループの右側にボタンを選択します。

  3. サイドメニューでグループメンバーを選択します。

  4. 画面右上のボタンを選択します。

メンバーリストをCSVファイル形式でユーザーのローカルディスクに保存します。

ノート

メンバーリストのダウンロード機能はメンバー管理権限のあるユーザーのみ利用できます。メンバー権限の詳細は次の文書を参照してください。
