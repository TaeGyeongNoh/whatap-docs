Preview 2.7.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.7.0.a4​

2024년 07월 11일

공통​

Deprecated 통합 대시보드 / 통합 메트릭스 보드

Preview 2.7.0.a3​

2024년 07월 04일

공통​

  • Feature 사이드 메뉴에서 전체 프로젝트를 선택해 프로젝트 목록을 펼치면, 프로젝트 검색 입력란에 자동 포커스되는 기능 추가

  • Fixed 관리 > 프로젝트 멤버 메뉴에서 테마 변경 시 이름/소속 태그 색상이 반영되지 않는 문제 수정

Appication​

  • Feature 다음 메뉴 경로에서 조회 내용을 CSV 파일 다운로드 기능 추가

    • 인스턴스 성능 관리 > 라이브러리 버전
    • 인스턴스 성능 관리 > 실행 환경 변수

  • Feature 관리 > 에이전트 설정 메뉴에서 경로에서 선택한 에이전트의 에이전트 설정(whatap.conf) 파일 다운로드 기능 추가

  • Changed 인스턴스 성능 관리 > 힙 덤프 메뉴에서 일본어 안내 메시지 변경

  • Changed 인스턴스 성능 관리의 하위 메뉴에서 에이전트를 선택한 상태로 다른 하위 메뉴로 이동하면 에이전트 선택 사항을 유지하도록 변경

  • Changed 사이트맵 > 토폴로지 > Netstat 토폴로지 메뉴의 예시 이미지를 영문으로 변경

  • 경고 알림 > 이벤트 설정 메뉴에서 메트릭스 이벤트 템플릿을 다음과 같이 변경했습니다.

    • Deprecated 중복되는 WARNING_OLD_GENARATION_GC 템플릿 삭제

    • Deprecated 불필요한 WARNING_TPS_DELTA 이벤트 삭제

    • Changed TOO_MANY_ACTX 이벤트 조건 변경

      • 기존: endsWith(okindName, 'ote-virtual-0') && startsWith(onodeName, 'node-0') && type == 'java'

      • 변경: active_tx_count > 100

  • Fixed 분석 > 메트릭스 차트 메뉴에서 다음 위젯 표기 수정

    • Heap 크기(heap_tot) → Heap 총 사용량(heap_tot)

    • Heap 최대 사용량(heap_max) → Heap 최대 크기(heap_max)

Server​

Fixed 서버 목록 > 서버 상세 메뉴의 메모리 사용률 상세에서 3시간 이상 조회하면 Memory Swap Used 차트가 표시되지 않는 문제 수정

Database​

공통​

  • Feature 특정 날짜를 기준으로 설정된 시간대의 SQL의 추이를 비교하고 등락을 확인할 수 있는 분석 > Top SQL 비교 메뉴 추가

    노트

    이 기능은 다음 상품에서만 이용할 수 있습니다.

    • PostgreSQL
    • MySQL
    • Oracle V2
    • Oracle Pro

  • Changed 대시보드 > 인스턴스 목록 메뉴의 XOS2 지표 대체 안내 메시지가 더 이상 나타나지 않도록 변경

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 위젯 이름 변경(Active Sessions Count → Active Session)

Oracle V2​

  • New Oracle V2 서비스 오픈, 새로운 기능에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Feature 대시보드 > 인스턴스 목록 메뉴에서 에이전트 업데이트 및 버전 변경 안내 팝업 추가

  • Feature 관리 > 프로젝트 관리 메뉴에서 버전 선택을 할 수 있는 기능 추가

Oracle Pro​

  • Changed 상품명 변경: Oracle DMA → Oracle Pro

  • Feature SQL 상세 창에서 bind 변수를 수정하고 조회할 수 있는 Explain Plan 탭 추가

  • Fixed 통계/보고서 > SQL 통계 메뉴의 특정 쿼리에 대한 테이블 데이터와 시리즈 데이터의 값이 맞지 않는 문제 수정

  • Fixed 통계/보고서 > SQL 통계 메뉴의 시리즈 데이터가 제대로 표시되지 않는 문제 수정

  • Fixed 통계/보고서 > SQL 통계 메뉴의 탭 명칭 변경

    • USERNAME → SCHEMANAME

    • MODULE, OSUSER 추가

  • Fixed 대시보드 > 인스턴스 모니터링 메뉴에서 액티브 세션 테이블의 쿼리를 선택한 다음, SQL 통계 메뉴로 이동하면 sql_id로 필터를 기본 설정하도록 수정

Oracle V2 / Oracle Pro​

  • Feature SQL 상세 창에서 선택한 쿼리에 대한 통계 요약 정보를 제공하는 Runtime Plan 탭 추가

  • Feature 통계/보고서 > 테이블스페이스 사이즈 메뉴의 테이블스페이스 사이즈 상세 섹션 목록에서 항목을 선택하면 세그먼트 사이즈(TOP 50)를 표시하는 기능 추가

Preview 2.7.0.a2​

2024년 06월 27일

공통​

  • Changed 연계 프로젝트 관리 메뉴의 일본어 명칭 수정

  • Changed 사이드 메뉴의 조직 목록에 조직 아이콘 노출하도록 변경

  • Fixed 경고 알림 > 이벤트 설정 메뉴에서 이상치 탐지 상승/하락 패턴의 잘못 설정된 민감/둔감의 score 수정

  • Fixed Flex 보드 메뉴에서 액티브 스테이터스 차트가 보이지 않는 문제 수정

  • Fixed CSV 파일을 다운로드하면 한글이 깨지는 문제 수정

Application​

  • Feature 인스턴스 성능 관리 > 스레드 목록/덤프 메뉴의 테이블에서 CPU 시간 증가량을 표시하는 컬럼 추가

    노트

    화면 오른쪽 위에 버튼을 선택하면 증가량 컬럼에서 변화된 수치를 확인할 수 있습니다.

  • Fixed 다음 메뉴 경로에서 컬럼 설정 기능을 통해 전체 컬럼을 선택하면 화이트아웃 현상이 발생하는 문제 수정

    • 통계 > 트랜잭션
    • 통계 > 유저 에이전트

  • Fixed 구버전의 애플리케이션 대시보드 메뉴에서 안내 메시지의 로케일 깨짐 문제 수정

  • Fixed 대시보드 > 트랜잭션 맵 메뉴의 Top 30 목록에서 키보드 이동 시 발생하는 딜레이 문제 수정

  • Fixed 다음 메뉴 경로에서 데이터가 표시되지 않는 문제 수정

    • 트레이스 분석 > 멀티 트랜잭션 > 테이블
    • 분석 > 멀티 트랜잭션 추적 > 테이블

Database​

공통​

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 일부 위젯의 이름 변경

    • Active sessions → Active Session Count

    • Long Active Sessions → Long Active Session Count

  • Changed 다음 메뉴 경로에서 액티브 세션 섹션의 항목을 선택하면 나타나는 세션 상세 정보 창을 통해 분석 > 세션 히스토리 메뉴로 이동하면 해당 세션이 수행되는 시간의 전후 30분을 기본 조회하도록 변경

    • 대시보드 > 인스턴스 모니터링
    • 대시보드 > 멀티 인스턴스 모니터링

  • Fixed 경고 알림 > 이벤트 설정 메뉴의 데이터베이스 탭에서 인스턴스 항목의 기본값이 전체로 설정되는 문제 수정

Oracle Pro​

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 일부 위젯의 이름 변경

    • Bandwidth → [RAC] Interconnect Bandwidth
    • Load Balance → [RAC] Load Balance
    • Load Balance 위젯의 physical reads 필드 이름을 session logical reads 필드로 변경

  • Changed 통계/보고서 > 테이블스페이스 사이즈 메뉴에서 차트의 범례를 표시하는 방식 변경

    • Multidb이면 중간 구분자를 언더바(_) 에서 하이픈(-)으로 변경

    • Multidb가 아니면 중간 구분자를 표시하지 않도록 변경

  • Changed 경고 알림 > 이벤트 설정 메뉴 진입 시 이벤트 설정 New 메뉴로 이동하도록 변경

  • Fixed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 Interconnect Bandwidth 위젯과 Load Balance 위젯의 인스턴스 순서를 인스턴스 셀렉터의 순서와 동일하게 보여지도록 수정

  • Fixed 분석 > 세션 히스토리 메뉴에서 차트 데이터와 테이블 데이터가 일치하지 않는 문제 수정
