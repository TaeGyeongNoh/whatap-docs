Preview 2.10.0.a

노트

preview.whatap.io에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다.

Preview 2.10.0.a3​

2024년 10월 11일

Kubernetes​

  • Changed 클러스터 > 노드 디스크 목록 메뉴에서 OnodeName 컬럼의 을 선택하면 노드 상세 메뉴로 이동하도록 수정

  • Changed 관리 > 에이전트 설치 메뉴에서 에이전트 버전(Stable, Preview)에 대한 설명 보강

Preview 2.10.0.a2​

2024년 10월 10일

공통​

  • Changed 페이지네이션 컴포넌트 스타일 개선

  • Changed 이용 내역 > 결제 정보 메뉴에서 와탭 스터터팩 관련 프로모션 이미지 변경

  • Changed 언어 설정 메뉴의 아이콘 변경

Application​

  • Changed 통계 메뉴에서 CSV 다운로드 라인 수가 최대 조회 건수와 일치하도록 변경

    • 수정 전: CSV: 100,000 / 조회: 10,000

    • 수정 후: CSV: 10,000 / 조회: 10,000

  • Fixed 통계 메뉴에서 데이터 조회 중 페이지네이션 비활성화되도록 수정

  • Fixed 통계 하위 메뉴에서 에이전트 별칭이 있는 경우 검색되지 않는 문제 수정

  • Fixed 통계 하위 메뉴에서 에이전트 검색 필드 오른쪽에 위치한 선택 버튼 기능이 적용되지 않는 문제 수정

Server​

Fixed 서버 목록 > 서버 구성 정보 메뉴의 서버 구성 정보 에이전트 안내에서 Windows 옵션 추가 명령어에 누락된 역슬래시(\) 문자 추가

Kubernetes​

  • Feature 다음 메뉴에서 노드 요약 분석 기능 추가

    • 대시보드 > 컨테이너 맵

    • 대시보드 > 노드 맵

노트

이 기능은 쿠버네티스 에이전트 마스터 1.4.1 버전 이상에서 지원합니다.

  • Changed 대시보드 > 컨테이너 맵 메뉴의 컨테이너 요약 분석에 컨테이너 상태 및 애플리케이션 정보 보충

  • Fixed 대시보드 > 노드 맵 메뉴의 에이전트 지원 버전을 v1.3.7에서 v1.4.1로 변경

  • Fixed 대시보드 > 컨테이너 맵 메뉴로 처음 진입할 때 간헐적으로 라벨이 제대로 표시되지 않는 문제 수정

Database​

공통​

  • Feature 경고 알림 > 이벤트 설정 Old 메뉴에서 메트릭스 이벤트 템플릿 추가

    노트

    이 추가 기능은 Altibase, CUBRID, MySQL, SQL Server, Oracle, Oracle pro, Redis, MongoDB 상품에 적용됩니다.

  • Changed 대시보드 > 멀티 인스턴스 모니터링 메뉴에서 사용자 정의 위젯의 지표를 4개 초과하여 추가할 때 나타나는 메시지 변경

  • Fixed 대시보드 > 인스턴스 모니터링 메뉴에서 세션 킬 기능을 사용할 때 응답값이 FAIL - Invalid paramkey라면 '잘못된 파라미터 키 입니다.'라는 메시지를 표시하도록 수정

  • Fixed 복사된 링크를 통해 통계/보고서 > SQL 통계 메뉴로 진입하면 발생하는 화이트 아웃 현상 수정

MongoDB, MySQL, PostgreSQL​

Changed 통계/보고서 > 데이터베이스 사이즈 메뉴의 UI 변경(바 차트에서 누적 시리즈 차트로 변경)

Altibase​

Feature 통계/보고서 > 테이블스페이스 사이즈 메뉴에 세그먼트 사이즈를 확인할 수 있는 테이블 추가

Oracle, Oracle Pro, CUBRID​

Feature 통계/보고서 > 테이블스페이스 사이즈 메뉴에서 테이블스페이스 사이즈 상세 테이블에 TOTAL row 추가

Network​

Fixed 사이드 메뉴가 축소 모드일 때 실험실 메뉴의 아이콘이 정상 표시되지 않는 문제 수정

Log​

  • Feature 로그 검색 및 라이브 테일 메뉴에서 원하는 검색 조건을 즐겨찾기로 등록할 수 있는 기능 추가

  • Fixed 로그 설정 메뉴의 로그 1차 파서 설정에서 패턴 입력 후 시뮬레이션 기능을 실행하면, 패턴 입력란에서 ctrl+z가 동작하지 않는 문제 수정

URL​

Feature URL 추가하기 화면에서 확인 지역별 IP 안내 추가

Preview 2.10.0.a1​

2024년 09월 26일

공통​

  • Changed 경고 알림 > 이벤트 수신 설정 메뉴의 서드 파티 플러그인 추가에서 Webhook JSON 입력 필드 내용 최신화

    • uuid, okind, onode 키 추가

    • Server 상품의 경우 serverAlertPolicyName 키 추가

    노트

    서드 파티 플러그인 추가에 대한 자세한 내용은 다음 문서를 참조하세요.

  • Changed 통합 Flex 보드 또는 Flex 보드 메뉴에서 대시보드가 없거나 목록을 불러오는 중, 표시하는 안내 메시지의 세로 정렬을 가운데로 변경

  • Fixed 실시간 알림 목록에서 이벤트 항목의 비교 시간이 갱신되지 않는 문제 수정

  • Fixed 시간 선택자에서 시간 단위 제한이 있는 경우 일본어의 메시지 표기 오류 수정

    選択できる最大時間は3時です。 → 選択できる最大時間は3時間です。

Kubernetes​

  • Changed 보고서 메뉴에서 보고서 양식을 월간 애플리케이션 보고서, 월간 성능비교 보고서를 선택했을 때 여러 개의 에이전트, 에이전트 그룹, 에이전트 종류를 선택할 수 있도록 수정

  • Fixed 대시보드 > 노드 맵 메뉴에서 스타일 버그 수정: 텍스트 테마 오류, 옵션 영역 깨짐 현상

  • Fixed 대시보드 > 노드 맵 메뉴에서 과거 시간을 조회하면 노드 데이터가 정상 표시되지 않는 문제 수정

  • Fixed 대시보드 > 노드 맵 메뉴에서 임계치 설정 중 불필요한 Normal 설정 제거

  • Fixed 대시보드 > 노드 맵 메뉴에서에서 노드 전체를 선택 후 로그 메뉴로 이동하면 로그 데이터가 조회되지 않는 문제 수정

Database​

공통​

  • Fixed 사이드 메뉴에서 로그 파일 뷰어, 파일 로그 알림 메뉴를 제거하고 사이트맵에서 확인할 수 있도록 수정

  • Fixed SQL 상세 창에서 SQL 문에 콜론(:)이 여러 개 포함된 경우, 이를 바인드 변수로 인식하는 문제 수정

Oracle V2, Oracle Pro​

  • Fixed 분석 > 락 트리 메뉴에서 event# 컬럼 이름을 event로 수정

  • Fixed SQL 상세 창에서 Runtime Plan의 평균 데이터가 합산되어 계산되는 문제 수정

PostgreSQL, MySQL​

Fixed 대시보드의 액티브 세션 테이블에서 쿼리를 선택해 Plan 조회 시 Object Detail 창에서 테이블 데이터가 없을 때 컬럼 이름만 표시하도록 수정

Log​

  • Fixed 로그 트렌드 메뉴에서 키워드와 매칭되는 태그 값에 대해 후보 목록을 필터링하도록 수정

  • Fixed 로그 트렌드 메뉴의 필터 입력창에 값을 입력한 상태에서 목록이 갱신되면 필터링이 해제되는 문제 수정

Preview 2.10.0.a0​

2024년 09월 13일

Server​

  • Fixed HP-UX 운영체제 환경의 에이전트 설치 내용 중 오타 및 링크 오류 수정

  • Fixed 서버 목록 > 서버 구성 정보 메뉴에서 사용자 정의 필드로 입력한 값이 간헐적으로 저장되지 않는 문제 수정

  • Deprecated 서버 목록 > 서버 구성 정보 메뉴의 컬럼 설정 기능 중 정렬 옵션 제거
